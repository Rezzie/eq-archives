Simple. The router you are using is a SWITCH, not a HUB. The difference, the switch will NOT broadcast packets to all the ports and so the linux machine will NOT see any of the EQ packets destined for the Winblows machine.

Buy a hub, and make sure that it is truly a hub and not a cheap switch in disguise, and hook both the EQ and linux machines to it, then use the uplink port to connect it to your router.

To test if the device is truly a hub:

As root on the linux machine:

tcpdump -i eth0 | grep <ip of windows machine>

then on the windows machine surf the net. If in the linux terminal window you ONLY see BROADCAST AND ARP packets, then you have a SWITCH and need to get a different model, else you should see ALL the packets that the windows machine is sending/receiving while surfing.
