Sparr
Zombie 		posted 12-03-2001 11:05 PM            
Odin's Eye works as a passive sniffer. It doesnt have to be the gateway for your PC, just has to be capable of "seeing" the packets to/from the PC. This can be done by setting it up ShowEQ style with the linux box as the gateway for the PC, or you can just hook the Linux and DAOC boxes up to the same hub. Be sure its not a switch though (most cable/dsl routers are switches).
The original ShowEQ setup is like this:
code:

Internet--LinuxBox--DAOCBox

My setup is like this:

code:

CableCompany--CableModem--Router--Hub--DAOCBox
                            |      |`--DAOCBox
                            |      `---LinuxBox
                         OtherPCs

be sure to enclose your ascii pictures inside a set of [Code] [/Code} so it looks right. the } is supposed to be a ] but I had to change it so the board wouldnt eat it.
