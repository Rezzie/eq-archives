!cc gmdeaf:
When active, !cc gmdeaf will prevent anyone not on your buddy-list from sending you a message via /tell. Messages from anyone flagged with GM access will still be recieved. When gmdeaf is enabled, all advisors must remember to add the petitioner they are contacting to their buddy-list with !cc addbuddy <player>.

!cc addbuddy <player>:
Manually adds Player name to your buddy list.

!cc rembuddy <name>:
Manually removes Player name from your buddy list.

!cc rembuddy ?:
Completely wipes everyone from the "?" section of the Friends view. It will not remove anyone that has been added as a buddy.

/ch v - Switches quickly to vicinity channel.
/ch g - Switches quickly to the GM channel.

/command assist [id] This command teleports you to the person with id used in command.

/command summon [id], the ID or Identification number you can find in different ways. this will summon the person with that ID to your location.

/command setname firstname [newname] Sets a new firstname, this command needs you to target the player who's name needs a change.

/command setname nick [newname] Sets a new nick, this command needs you to target the player who's name needs a change.

/command setname lastname [newname] Sets a new lastname, this command needs you to target the player who's name needs a change.

/command teleport x y z, where x and y are the coordinates and z is the PF. You can also leave out the PF if you will teleport yourself in the same PF.

/teleportdynel x y z. You use it by selecting a char near you and then using the command to teleport him.
