Zeppo is correct.

Mandrake is built from RH and is more user friendly.

I would strongly advise Linux n00bs install Mandrake 8.1. Switch to RH or the distribution of your choice once you have developed a level of comfort with Mandrake.

One piece of advice - when you install Mandrake, install *everything* by using the expert install and make sure you select server install (rather than workstation). Don't try to pick packages - install them all. This will take more space on your HD, but you'll find your SEQ install (as well as the pre-reqs) goes much smoother.

Good luck!
