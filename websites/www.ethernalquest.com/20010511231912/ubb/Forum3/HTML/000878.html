>Taking your examples I came up with using level:[25-60] ...

Now that wont work, hehe. Regexp is based on 'single' character matching. It does not convert things like '25' and '60' to actual numbers.

Example: the regexp 'abcd[1-2]' will match 'abcd1', 'abcd2' and 'abcd15' when tested with the following set of strings:

abcd1
abcd2
abcd7
abcd9
abcd15

I'm not at home right now so I can't double check (might have missed out someting).

I dont know what kinda regexps ShowEQ can handle but this one works just fine =)
