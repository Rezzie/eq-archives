Well, I think there probably is a file somewhere in the source, but I don't know anything about it or if you can even configure that. However I can say that if you're not at least an intermediate level programmer, you're probably not going to be able to do anything about it ;)

The Name:will* thing is in showeq.conf under Categories or something isn't it? Well I don't really know anything about the spawnlist categories, so all I can really say is mess around with it and do what you can, but you could do something similar with filtering. The idea is you open up your filters.conf file and put whatever you want to filter out under the proper category using regular expressions. Like if you wanted to filter everything level 35 you'd just put 35 under the [Level] category like this:
[Level]
35
and everything level 35 would be filtered. Doing a range is a bit more complicated if you don't know regex so if you wanted to filter 35 to 39 you could do something like this:
[Level]
^3[5-9]$
and if you wanted to filter other levels you can put more than one expression. This thread has a pretty good explanation, I think: http://www.ethernalquest.com/ubb/Forum3/HTML/001452.html

It can be confusing at first but you'll figure it out ;)
