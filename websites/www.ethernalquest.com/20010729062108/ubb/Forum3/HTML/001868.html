I setup a IPTABLES firewall a few months ago and I hated every minute of it. None of the commands are the same and hte logic is backwards too. The only reason I would use IPTABLES of IPCHAINS is if you want to have DHCP running on a network but still maintain a restriced connection to the internet for users. With IPTABLES you can create MASQ rules based on their MAC address as opposed to thier IP address.

I know all that doesn't help much, but what I would suggest is reinstalling IPCHAINS and then using a registry tweaker on your windows machine and up the packet size to the max if you are using a cable modem or someother type of LAN connection.
