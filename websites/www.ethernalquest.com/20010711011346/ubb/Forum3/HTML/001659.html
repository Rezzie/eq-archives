Zombie 		posted 05-17-2001 05:33 AM            
if you already have the seq source checked out, just do

cd linux
cvs login
cvs -z3 update -A -P
make distclean

inside the linux dir is a CVS dir that retains info like the CVSROOT, so you do not need to specify it if you perform operations from within the checked out module dir.

and always do make distclean and ./configure after updating cvs, it will save you posting here about why you get so many errors when you attempt a make.
