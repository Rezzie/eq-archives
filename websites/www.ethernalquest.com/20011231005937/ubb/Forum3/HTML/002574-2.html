Zombie 		posted 12-14-2001 12:22 PM            
Sun, you're just having some trouble with syntax usage on your version of tar. Go ahead and do what it says, type 'tar --help' and read what it says. I guess you're looking for -Acdtrux all together, when what you're probably seeing is an explanation of -A and -c, and -d, and so on....

The -Acdtrux it's talking about are the switches (options) for the command. In other words, tar is the command and you can add things to it like -A or -c or -Ac to tell it specifically what to do.

The options that Necro used were z, x, v, and f. The z tells it to un-gzip the file (see the .gz on the end of that particular file?), the x stands for eXpand (untar), the v means please use Verbose mode (tell me what's going on), and the f stands for File (ok what file are we going to use here?).

I'm not familiar with the version of tar you're using, but it looks like it has the x, so you're ok there. If you have gunzip you can try gunzip on the *.gz file to get a .tar file out of it, and then you don't need the z. The v isn't actually necessary in the first place. I don't know about the f. Read what it says when you type tar --help and see what switch is used to specify a file.
