1) Get the latest CVS Version

2) go to the conf directory inside where you untarred the source code at. copy showeq.conf.dist to the showeq share directory (cp showeq.conf.dist /usr/local/share/showeq/showeq.conf)

3) Depends on what window manager you are using. Even better would be to create a script that you run from the command line so that you can still look at faction numbers. To do this open start up your favorite editor (pico, vi, emacs, nedit, etc...) and create a file that basically does the command line stuff. In my case it looks like:

/usr/local/bin/showeq -i eth0 10.0.0.7 -n -F -A -G -K -S

Save the file as whatever you want. Then go and change the files permissions to make it executable with chmod. If you don't know chmod, then read the man pages (man chmod).
