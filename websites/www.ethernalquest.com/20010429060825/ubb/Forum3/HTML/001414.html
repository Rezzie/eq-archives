gunzip 3c90x-1.0.0i.tar.gz
tar xf 3c90x-1.0.0i.tar
cd 3c90x-1.0.0
chmod +x install3c90x
./install3c90x

That should set up the card. If it still doesn't work you may need to read the driver documentation (readme.txt).

Zombie 		posted 03-17-2001 03:43 AM         
Ah ha! I think I finally figured this out! RedHat 7.0 errors out during the compile when doing ./compile_UP so 3c90x.o is never created. So you have to manually type in the compile command. Here's the revised instructions for doing this:

1. Open the file /etc/modules.conf and type in the following line:
alias eth0 3c90x

2. cd /root/3c90x-1.0.0
Replace /root with whatever directory you downloaded the tarfile to.

3. gcc -c 3c90x.c -O2 -Wall -Wstrict-prototypes -fomit-frame-pointer -fno-strength-reduce -pipe -m486 -malign-loops=2 -malign-jumps=2 -malign-functions=2 -DCPU=486 -DMODULE -D__KERNEL__ -I/usr/src/linux/include -DMODVERSIONS

4. Delete /etc/conf.modules if it exists.

5. insmod 3c90x.o
