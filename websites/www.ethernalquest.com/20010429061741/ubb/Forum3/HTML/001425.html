Zombie 		posted 03-17-2001 12:19 AM         
md5sum is a command used to verify file integrity. To see if your libEQ.a is the original, run this command in the console:
md5sum /usr/lib/libEQ.a

If that returns 1f9be1dc9c56619a34d916c52bd551c2 then you're ok. If not, redownload it from http://libeq.trifocus.net Before you try recompiling with libEQ, do a make distclean and re-run configure and make. BTW, ldconfig does nothing for static libraries (like libEQ).
