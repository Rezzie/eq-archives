Go to your window 9X machine and select START, then RUN and type in WINIPCFG.

This program will give you the IP that is currently assigned to you EQ window 9X machine.

Make sure you have the gateway & proxy settings the same on your linux box and your window 9X box. A good test is to surf the web from your linux box.

On your linux box, bring up a terminal windows & type ping YY.YY.YY.YY and see if you ping your EQ window 9x machine.
Y = the IP from WINIPCFG

Then and start SEQ by typing:

./showeq -i ethX YY.YY.YY.YY -n -F -A -G -K -S

X = 0 or 1 (nic that will sniff your EQ packets)
Y = the IP from WINIPCFG
