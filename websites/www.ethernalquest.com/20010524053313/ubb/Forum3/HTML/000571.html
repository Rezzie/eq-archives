yea, filters do work. I killed the spawn and spawns file and onlz work with filters file in the same dierectory than the showeq executable is located. One simlple trick shows u how the filter syntax is and if the filter file is accepted: Toggle Filter of a selected MOB inside the showeq program. u will find this option in the files menu if I remeber right. After toggling filter save filters. After that rebuild spawn list. If the before selected MOB now is disappeared u got the first point running. Now see in your filters file if u find the MOB now in the filtered section. If u find all works fine and u know the exact syntax. From beginning of this little test u need the ORIGINAL filters file from the distribution set located in the directory of showeq. If this does not work check the name of the filters file in the showeq.conf file in the shared files directory.
