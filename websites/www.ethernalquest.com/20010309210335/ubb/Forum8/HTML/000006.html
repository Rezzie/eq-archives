Ya I found the money values at 0087B560.
If you change this value it doesn't change on display. But then when you drop the money it will ask you how much to drop.
What it asks can be as high as your changes.
However once you drop the money it reverts to its original value. I've tried changing the display value to match the hacked value and that didn't work either.

These values are 64-bit comp values. If you go up a few pages from that offset you can change the visual values.

About the game a little bit.
The storage places are all linked. All your characters on one account can share all the human storage places.

I created a second directory and made a new account on it. When I logged in under the 1st accounts name there wasn't much in inventory which leads me to believe that alot of stuff is stored in the encrypted inventory file and some of your stuff is stored server side.

So I think the key to hacking this is those charactername.inv files. I bet if you could get money in those files then it would appear on character or in storage.

One other tip for working with stuff. Once you have activated a item slot you can type in the value you want. So say click out 1 silver arrow at the weapons store and then type in 100 and you will buy 100 units of those.

Soooo. Poke around in memory some but the key to hacking this has got to be in those inventory files. Where do we go from here?
Anyone got any tips on working with encrypted files?
