THINGS YOU NEED TO HAVE

The NEW eqmain.dll/eqgame.exe
The emu
Circuit Gm Client
A hex editor

1. Copy the client into your EQ Directory
2. open loginsever.js
3. change the version to 7-18-2001 18:00
4. make loginserver.js Read only
5. open up loginserver.exe
6. type in the old version
7. change the old version string to the new one (see above)
