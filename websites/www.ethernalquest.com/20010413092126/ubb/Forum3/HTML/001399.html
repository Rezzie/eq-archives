Fixed my problem. Since I am using Windoz dial up I was using some Gateway software on my PC to share internet connection for my EQ machine. Turns out the Gateway software I was using was at fault (Avirt gateway). I uninstalled it and installed Winroute (as other posts had mentioned using this particular software) and Viola! Data everywhere! I must say, the information SHOWEQ yields is wonderful. Mob Levels, XP, etc. I love it. Bravo again on a cool and useful product!
