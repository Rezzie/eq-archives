Well, if I read your post correctly your problem is the EQ packets never go to the Linux system.

The Server sends the game info straight to your windows box, who does not forward the info to your linux system because the packets were meant for the Windows Client.

Basically you need to make your Linux system either the gateway, or find another way to route packets through it.

Using a USB DSL modem makes it tricky.. Not sure it will be possible to hook that up to your linux system as a gateway.. (Need to hear from someone more experienced in Linux to answer that)
