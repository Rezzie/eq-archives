I did some quick tests with Macro Express 2000 (you can download it from download.com (and astalavista.box.sk has a key generator)) which appears to work quite well with pretty much every program. This should work very well for all crafters, you can make multiple macros with different delays and loops..

I made a test macro (ctrl+f12):

<REP3:01:000001:000001:0010:0:01:>8<DELAY:10.00><ENDREP>

Which means: Repeat (10x) key '8', wait 10 sec.
Adjust the repeat and delay as needed.
