adenine!
Zombie 		posted 04-21-2001 12:56 AM         
Yep you should be able to use ShowEQ fine on that machine. I've heard of it running on systems as crappy as a 100 MHz 486. Basically you only need a swap partition (should be the same size as your RAM unless you have a whole lot of it) and a / partition. If you want to you can make partitions for other directories like /boot and /usr but you don't really need to. These are all the packages I can think of that you'll need to install ShowEQ (there could be others though...):
XFree86 libraries (just install them all)
gcc
gcc-c++
libstdc++
libpcap
libtool
autoconf
automake
make
But you shouldn't need to remove any of the Development stuff anyway to make it under 600MB.
