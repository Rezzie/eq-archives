You have to open up a new zone server window for each zone. Anyhow, I check my memory load with everything running, then I opened a zone server up. It took up 2% of my memory load. I have 512 MB of RAM. So that means on 512 MB of RAM (not counting other programs that run in the background) you could have about 50 zone servers running before performance severaly degraded. If you really wanted to run an EQ server and have all the zones up... your best bet would probably be to go with 3 computers with 1 gig of RAM each and just run about 30 zones on each... (That is assuming that EQ + Kunark are only about 90 zones total). You wouldn't need that big of a processor, just like a 200 MHz would do fine. The only problem with that is if one of the computer crashes... then you have 1/3 of all the zones down. Right now RAM is dirt cheap... and 200 Mhz processor isn't that expensive. If you build it yourself you could probably make 3 200 Mhz computers with 1 gig of RAM each for about 1,000-1,500 dollars depending on where you buy your parts. As long as you have a good connection to the internet.
