There are 3 parameters for every funcction in C++ (main is a function too). You need to specifie the type of value it will return to what ever called the function (void for it to return nothing), you need to specifie the name of the function (main is the name of the main function.. duh) and finaly you need to specifie the types of value it will receive from whatever calls it (void for no arguments). Some compiler will allow you to write nothing instead or writing void, but you will never get an arror for writing it, so why take a chance... write it.

Since your program doesnt return any value to the OS, your main function should be a void (or leave it an int and return 0 as the other guy said if you really want).

Also, since its not getting any arg passed to it, instead of leaving the () blank, try (void).

this should go :

void main(void)
{
