Lots of misinformation in this thread.

If you modify your stats in resident memory, there is no way for a GM to tell.

First, the various EQ files don't contain any functions to directly read memory (last I checked).

Second, the GMs wouldn't have reverse engineered the memory structure -- and hence would not have a clue where to look in your memory.

Now, this really isn't an issue, since modifying your stats has absolutely no effect whatsoever (other than graphical) except to modify movement (encumberance, falling damage, etc).

Additionally, whoever said giving yourself haste affects backstab refresh is only giving half the story. You can acheive the same effect by directly modifying the "Backstab" refresh. The button refreshes client side, but there is still a server side filter. All you really did is give yourself a backstab every 12 seconds, rather than every 10 seconds, in the example quoted.
