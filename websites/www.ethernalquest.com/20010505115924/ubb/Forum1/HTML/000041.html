Kai,

Don't you think it would be easier to start with something like CrystalSpace and develop from there?

I think the 3D technology in EQ is a little out of date. With the features of other publicly available engines, why take so much longer to reverse engineer when you can just take an existing codebase and build on it?

I would love to work on a MMORPG project where I can give creative input. I am not a programming god either, but I can read and modify similar to what Kai stated his skills were.

But, why develop an emulator when your client uses a 3-year-old 3D engine?

How about reverse engineering a CLIENT instead.. and add graphics features and automation? That would kick some ass.
It would also enable a hell of a lot of cheating with a few scripting features.
Imagine coding something like this in an EQ scripting language:

while (1) {
if (encumbered()) {
sell(); // triggers a routine that
// walks to a shop (hardcoded loc) and
// sells items that are "non essential"
// (hardcoded list.. rusty, FS, etc)
}
else if (totalmana/currentmana < 40) {
meditate(2); // med for 2 minutes

else if (totalhealth/currenthealth < 40) {
cast(greater_heal);
}
else if (
seek_and_destroy(lvl-3,lvl-1);
// routine to pick a random mob of lower
// level in the zone (hardcode bad choices,
// to preserve faction etc),
// pick a path with random wandering thrown
// in to prevent detection by Verant,
// kill it, and loot. Would be a
// complicated routine, but doable.
}
}

You could leave this on all night while you're sleeping, gain 3 levels, and get tons of loot. Sure, there's a lot of caveats as far as you would have to be in a relatively safe, flat zone, etc. You could use it to forage thousands of pieces of food, make thousands in plat by killing greens, etc.

Now, with that said, I think cheating to get ahead in the game would be boring and unethical. Why play? To me, I would program something like this just to have fun, and not use it on a regular basis.
I would write a bot to go to a newbie area and go around casting heals on anyone under 50% health, cast SOW, and say things like "I hate rain!" and "No one expects the Spanish Inquisition!"

I'm not advocating something like this, I just think it might be easier than a server emulator.
