casey
Zombie 		posted 08-22-2001 12:23 AM            
there is no windows emulator, but 2 peices of software you may be interested in are

VMware: emulates a complete x86 machine, on which you can run whatever you want (linux, windows, beOS, anything else you can boot on a x86 box).

Wine (Wine Is Not an Emulator): An implementation of the win32 api under linux, lets you run some windows apps under linux.

VMware has no 3d support, so a lot of games are out of the question.

Wine has support for openGL, and directX support is kinda there, but nothing to brag too much about atm. Half life runs under wine (in openGL mode) and some other popular games as well.

Other games have native linux support such as all the quakes, doom, tribes2, descent (all i think), and others. see lokigames.com for a bunch.

But if you really want to stay on top of gaming, windows is really the only way to do it. :/
