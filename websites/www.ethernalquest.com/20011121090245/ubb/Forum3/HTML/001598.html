wannabe:

i need to look at the instructions i made again, the elusive line that screws up may have moved, the line you want to delete is

typedef int charset[2]; const charset x;

that line causes configure to detect a broken const (bad). after editing ./configure do

make distclean
./configure
make
make install

dont re-run autogen as that rewrites ./configure to the original version. you need to make distclean and reconfigure to generate some files needed to build.
