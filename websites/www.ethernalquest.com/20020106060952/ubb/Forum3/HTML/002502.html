Showeq is not available on any Windows platform. Basically the whole idea is that it exists, but you have to jump through some hoops to get it to run. If it installed from a double click then VI would expend the effort necessary to stop it. SEQ and VI live in an uneasy truce. Shots are fired by VI, but so far none have been mortal wounds. Think of it as a cold war. As soon as there is a readily available windows version it would be like the USSR putting missiles on Cuba.. too damned much to ignore.

If you understand this you will understand why no windows version exists. There will never (again) be a windows version, because, if there was, VI would do whatever they could to put the kibosh on it. They still will try but it's not a high priority. We are the rebel alliance. If there becomnes too many of us though the evil empire will whip out the fully functional death star and start acing planets. None of us want that.

This is not a technological limitation, this is a political limitation, and thus is not one that can be overcome as a technological limitation could be. This cannot be overcome. The only possible way would be if SEQ were a paid for product and VI got a cut. Since SEQ is under the GPL this would be difficult, if not impossible to do now.
