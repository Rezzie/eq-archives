Answered my own question. Here is how I fixed it. Hopefully this will help others.

Download the GLU library from the Mesa3D site at Sourceforge:
Mesa3D

Grab the file named:
oss-opengl-glu-20000925-1.i386.rpm

Install it and try compiling OE again
