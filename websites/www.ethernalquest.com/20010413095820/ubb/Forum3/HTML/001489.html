It looks like your packet are never being sent out to the network. You are using the ISDN modem on the Win98 machine to access the internet right? If so, then there is no reason for your EQ packets to ever show up on your local network.

Try putting the ISDN modem on your linux machine and configuring it for IP masquerading. Its a bit of work, but using the linux machine as the gateway is a common way to get things working.

Sounds like you got the machines in the wrong order.
You should have the linux box connecting to the internet, then have the Win98 box connected to the hub that the linux box is on.

The information has to go through the linux box in order for SEQ to be able to read the EQ packets, then forwards them on to the win98 box running EQ.

Good luck.
