This is expained in previous posts, so keep reading, or use the search feature. It's setup just like you said:

1. You install software on your windows machine that emulates a separate machine (sets up a portion of your system to act as a separate machine.)
2. On this "virtual machine" you install the LINUX operating system, just as if you were installing it on a separate piece of hardware.
3. You then install ShowEQ on the virtual Linux machine.

All the information you need to know is available in this forum. You will probably have to do some reading. You may have to do a lot of reading. There are links to web-sites that will give you tutorials on Linux and other related subjects, like networking.
