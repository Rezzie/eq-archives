Ok..first off the inet connection must go through the linux box.
You dont need a hub or a switch. A crossover cable will work fine.

It should look something like this.

Inet---Linux---Win9x

Your ipmasking must be working. Try pinging your win box. If that works, from your win box, ping the linux box. If that works, try to get out on the inet from you windows box.

If all that works, then the problem is most likely with the config for showeq. Check your showeq.conf file and make sure the ip and the ethernet settings are correct.

Hope that helps.
