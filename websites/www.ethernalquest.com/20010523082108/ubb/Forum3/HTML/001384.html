Really.. Please..

It's obvious that most of you are gamers.. I installed SEQ on a SuSE machine in about 45 mins..

1. Launch yast.. (I hopy you know what this is..)

2. With yast, install the qt2 packages (something like qt2lib, qt2devel, etc..)

3. Download and unpack the SEQ code..

4. Run ./configure --with-qt-dir=/usr/lib/qt2 (this option is quite important..)

5. make

6. make install


Now, that wasn't so hard, was it..
