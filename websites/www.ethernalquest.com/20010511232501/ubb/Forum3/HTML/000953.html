cvs is very easy to use :) and i will show u how right here!
it was already installed on my system, so i will assume it is installed on yours as well.

i actually wrote a little script that will update showeq just by typing say "./update.showeq.sh"

u can gleen the commands from here as well :)

easy step by step!
cd /tmp
mkdir ShowEQ
cd ShowEQ
cvs -d :pserver:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq login
<press enter>
cvs -z3 -d :pserver:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq co linux
cd /tmp/ShowEQ/linux
cvs update -P
./configure
make
make install

it is that easy...
a few notes, that cvs stuff should all be one line... the cvs update -P will remove empty directorys (cause for some reason, u can't remove them from the cvs server).

#!/bin/bash
cd /tmp/ShowEQ
rm -Rfv *
cvs -d :pserver:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq login
cvs -z3 -d :pserver:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq co linux

cd /tmp/ShowEQ/linux
cvs update -P

#./configure
#make
#make install

u could copy this to a file by doing
pico <filename.sh>
<insert all that jazz here>

ctrl-x (choose yes)
chmod +x filename.sh

then ./filename.sh

if you set it all up, it will remove a bunch of files, then download them again. there is also a way to just update all files, but i dunno what it is :)
