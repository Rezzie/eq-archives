Just FYI, the Win2k version of ICS (if you can call it that) does work with ShowEQ. I have my network set up thusly:

cable modem-->Win2k server box-->hub-->4 other computers, including EQ game box and linux box.

This works like a charm, all I had to do was specify the address of the box on which I play EQ (192.168.0.XXX) in the .conf file.

I wish I knew the exact differences between how Win2k handles NAT translation and how Win9X does it, but hey it works for me and it may be easier for some to upgrade to Win2k rather than install Linux.
