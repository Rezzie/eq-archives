this is a VERY simple and crude ass script i wrote to update ShowEQ to the CVS (which is the version that will work!). it assumes one or two things...

1.) it is in the dir /tmp
2.) the dir /tmp/ShowEQ exists (CASE sensitive)

to use it, open a file and paste this into it (pico update.showeq.sh)

then, make it executable (chmod +x update.showeq.sh)

then, you might have to chown it (chown root update.showeq.sh). i run the update as root personally.

after you have done that, goto tmp (cd /tmp) and run the file (./update.showeq.sh)

after that, goto the dir it downloads to (cd /tmp/ShowEQ/linux) and do (more CHANGES) to see if anything new has happened. then, if it has...

right now to compile it you do (make -f Makefile.dist ; ./configure ; make (-j3 if u have dual cpus) ; make install

you also need to download libEQ.a and put it in ... /usr/lib/ (or a few other places) before you run ./configure.

hope it helps :)

#!/bin/bash

cd /tmp/ShowEQ
rm -Rfv *
cvs -d :pserver:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq login
cvs -z3 -d :pserver:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq co linux

cd /tmp/ShowEQ/linux
cvs update -P
