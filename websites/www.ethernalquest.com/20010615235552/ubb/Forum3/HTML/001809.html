Here is what I did to get it installed..actually very simple all in all. Do a custom install and select as many components as possible from the non-server side. In fact if you have disk space just select them all The most important ones to hit are

make
gcc
autoconf
automake
Xfree86 dev
libpcap
zlib
qt-2
qt-devel
libstdc++
egcs-1
egcs-c++
egcs-cpp
libtool

I've shortned them to save space but here's the catch for some reason some of these don't appear in the initial set up (at least I couldnt find them) You have to add them with the "Software Manager" on your desktop after your done the install if Mandrake. Just do keyword searches using the first few letters and you'll see all the packages.

Once all the packages are installed

Save the libEQ.a to /usr/local/lib/libEQ.a from http://libeq.trifocus.net/libeq/i386/linux

Then use cvs download by doing the following

mkdir /root/showeq
cd /root/showeq
export CVSROOT= server:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq
cvs login (just hit return for password)
cvs co linux
cvs logout
cd linux
make -f Makefile.dist
once thats done
./configure; make
make install
cd /usr/local/share/showeq
cp showeq.conf.dist showeq.conf
cp filters.conf.dist filters.conf
edit the showeq.conf file to the ip of your eq machine
showeq to fire up the program
