I have read a few posts here from people that can no longer use DX Windows.

Here is my history with this awesome little program

I DLed DX Windows ver. 1 beta 2 a little after it was released. I had a TNT 8 meg board in my system and DX Windows worked great!

I changed my video board to a TNT2 32 meg board and DX Windows started having problems. It would still load and work, but after a period of time, 2 mins up to an hour, it would lock up my system. So, I had to stop using it

Then, there was the patch that appeared to "break" DX Windows and I read a few posts stating that it still worked but only to alt-tab out and then back in. I tried it this way just because I was desperate and it worked great for me. It actually seemed to stabalize everything and I could adjust gamma

A couple of days later they did the emergency patch to fix video problems and once again people were posting that DX Windows did not work, even for Alt-Tab.

It worked fine for me after the second patch at first... Then for some reason it stopped working again But tonight I was messing with it and found out how to make it work now.

I ahve all my settings as they should be, mip-mapping off, level detail off, same res off and my res is set to 640 x 480. But, the one thing that allows me to use DX Windows now is having winamp streaming MP3s when I start up EQ. This cause the EQ sound not to initiate and makes it so DX Windows will allow me to Alt-Tab in and out.

I hope this helps some other people out there and possibly is info that Ashran can use.

Thanks

P.S. Another poster commented that DX Windows had saved EQ for him, this is also true for me, thanks a ton Ashran

As far as winamp goes, there are two relevant output modules: waveOut and DirectSound. The perk of using DirectSound is that it could let you hear sound from multiple programs at once, and it may have lower CPU utilization. But if your audio card doesn't support DirectSound (doesn't support well, that is... like most ISA sound cards I've come across), you need to use waveOut.
The point is that if winamp uses waveOut, Windows won't let EQ initalize sound. This seems pretty interesting. I'll probably play around with this myself. Thanks meandmyself!
