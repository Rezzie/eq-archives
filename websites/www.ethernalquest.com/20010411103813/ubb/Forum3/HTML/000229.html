Ok, For NT you need a program to foward the internet stuff to your Linux box. Everything is set the way it should be it looks like. I would sugguest a program such as SyGate Ver 3+. There are cracks/reg available for it.
One thing though , the server will have an ip. prob 192.168.0.1 so that would be the gate way for the Linux box. Make sure you have your dns #'s entered as well.

NT:
IP 192.168.0.3
Netmask 255.255.255.0
Gateway Empty
DNS Empty
workgroup name: workgroup

Linux (Mandrake 7.0)
IP 192.168.0.2
Netmask 255.255.255.0
Gateway 192.168.0.1

DNS Empty - You need to add your dns #'s from your isp or how ever u connect. If unsure check out your local isp/etc web page or phone them up.

workgroup name: No need for one.

Also should IPv4 be enabled .. well i dont see what good it will do if the internet connection if from your NT box. Thats all I can sugguest I hope its been some help.
