You can still get your setup to work (assuming your CPU/memory isn't so poor you can't run X-windows on it):

Just move your modem to the Linux box (if it's not a winmodem -- otherwise buy a cheapish external modem) and set that up to do IP Masquerading. Run EQ on old box as before, having set a default route on it to point to the linux box. Run ShowEQ on Linux box. done!
