The screen is not minimized.. What happens is the eqw.ini file gets updated somehow and changes your window position so that it is actually open but out of screen area, effectively taking it out of view. You can fix this by editing the eqw.ini file and resetting all window resolution start coordinates to x=0, y=0. Why this happens I don't know but it only happened to me once and I havent had a problem since. Hope it helps.
