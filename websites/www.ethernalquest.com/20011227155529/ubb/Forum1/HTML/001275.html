You will not be able to ping the eq servers as the firewalls verant has setup block all icmp traffic, and you will not be able to traceroute on windows past thier firwall, for the same reason, but Unix traceroute is UDP based and seems to get through.

Also, the stats you get connecting to these boxes will not measure the performance you get for EQ, you need to test the connection to the zone servers themselves, and there are quite a few fro each world, but the stats should be similar since the world is going to be hosted at the same site.
