First, you need to know a little about coversions. The number systems used were hexadecimal (that's the numbers that also had letters) and binary (the ones and zeroes). It's relatively easy to convert between hex, binary, and decimal (our "normal" number system). I can talk about how you do the converts more later, but I don't want this first post any longer than necessary. Just remember that these converts are pretty easy.

Second, you need to know the ASCII code or have an ASCII chart. A chart can be found here: http://wls.wwco.com/ref/ascii.html ASCII is the standard code for characters on a personal computer.

About the only thing that you need to remember about ASCII is that the characters on your keyboard are on this chart and they all have a numeric representation. For example the letter 'a' has a value of 97 in ASCII.

So, basically with the platinum trainer thread, there were all those hexadecimal (hex) and binary numbers on some people's posts. Some didn't translate or were silly little pictures, but some of them said a lot. You just had to know how to translate from hex or binary into ASCII.
