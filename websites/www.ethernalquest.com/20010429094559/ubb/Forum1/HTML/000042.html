A hacked spdat file will be "patched" (and you will be logged) when you run the patch program. Once the patch server "updates" your spdat file to current (non-hacked) version, it is loaded into memory.

The only way to "hack" the spdat information is to edit it in memory.

However, I have heard a rumor that at random intervals, the server performs a checksum on the spdat in memory to look for "differences".

I can't verify the truth to the "checksum check". Use memory edits at your own risk.
