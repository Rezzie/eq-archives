Yes, delete the *.opt files.
You did not read the FAQ.
Section 2.1, Step 6.
6) Kill all your .OPT files and Client.ini

Go read this section again.

Here is a short and simple guide.

1. COPY all files from your everquest folder to a new folder named EthernalQuest.
2. Unzip the EthernalQuest.zip into this folder. Double check and make sure it did not create a folder named ethernalquest INSIDE the ethernalquest folder you had created. If so. just drag all those files from the new one to the one you created.
3. Unzip the emulog.zip into the ethernalquest directory.
4. Unzip the EQfiles.zip into the ethernalquest directory.
5. Delete all your old .opt files.
6. Run Zoneserver, Loginserver, Worldserver.exe files. This can be done in any order.
7. Run Emulog.exe
8. Make up anything for station name and password.
9. Log in like normal.
10. Then after you get this to work, then you can use the hexeditor on the zoneserver.exe and make it so the #zone command works and you can change zones.
11. If you do edit the zoneserver.exe correctly, you will be able to issue a command such as #zone tox. This will tell you that you have issued a GMN command and so forth.
12. /q, then log back in. You will now be in a new zone.
13. Enjoy.
