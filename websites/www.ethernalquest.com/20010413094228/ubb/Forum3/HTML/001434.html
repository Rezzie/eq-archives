It is a boot loader, used to manage which partition and OS to load, usually accompanied by a menu. Boot loaders are not normally located on a partition on the disk. They are stored in the Master Boot Record. This is why you often have to disable "boot virus protection" if you install linux.

Boot from your windows 95/98 boot disk (you DO have one, right?) and type at the prompt:
fdisk /mbr
This will replace the Master Boot Record with a default boot record, thus wiping out the loader. From my experience it works on multi-drive systems with no troubles (not sure if it'll replace the secondary disk's boot record, but that one doesn't really matter).

Installed on c? I've never used mandrake, being an RH fan myself, but linux should not have been installed to a lettered drive. Although I have heard of an install option that creates a virtual partition inside a FAT partition, this is slower and less stable than a dedicated partition.
