I would put the two NICs in the Linux box. And use IP Masq to get this going.

To clarify Kintar's suggestion of the Uplink port, that would be the port that has a switch on it. Either X or not X. It is probably set to X, which ALL the other ports will be set to by hardware. This is likely the first or last port on the device. In order for this hub to communicate to the hub upstream from it, you will need to switch it to a non-X setup. This is just an internal crossing of cables so that the hub talks correctly to the end device, but hub to hub communication does not require this.

Hope that helps,
