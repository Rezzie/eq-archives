Here is what I have found to work for me. Make sure you update your everquest. Then delete or backup your eqmain.dll. Either way you don't need it for the emulator. What you should do is run the everquest.exe and then wait for it to update all of the files. Then before you hit play online you should unzip the files from the zip with the eqgame.exe and the eqmain.dll. Then delete the eqmain.dll and then hit play online. It will come up to a screen where it ask for username and password and server. So you just type in whatever and put the ip address for the server. After you run it you will be able to run it like that afterwards using the emulog.exe not the emulogon.exe.
