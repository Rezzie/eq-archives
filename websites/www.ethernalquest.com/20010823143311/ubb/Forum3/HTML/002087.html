Frequently Asked ShowEQ Questions

Q. Where do I get the latest version of ShowEQ?
A. You can download a tarball from http://seq.sf.net, however, I suggest using CVS so you can always have the latest version. Here is how to use it:
cd /root/showeq (or wherever you want to keep the source)
export CVSROOT=:pserver:anonymous@cvs.seq.sourceforge.net:/cvsroot/seq
cvs login
(Hit enter when it asks for a password)
If this is your first time using CVS:
cvs -z3 co linux
If you have already used it at least once:
cvs -z3 update linux
cvs logout
cd linux
make -f Makefile.dist
./configure && make
make install
(Edit the configuration and filter files, if necessary)

Q. Why do I get xxxxx error when configuring/compiling ShowEQ 2.3?
A. 2.3 is an old version and would not work even if you did manage to get it compiled. See above for how to get a newer one.

Q. ShowEQ is going crazy! There are vector lines all over the place and spawns are showing up as gibberish! Did Verant change the encryption?
A. Probably not. Do not post a new thread unless you are absolutely sure there is not another about it. Look through the last few pages of threads to see if there are any about changes on the test server. More than likely, someone will have posted how to fix the problem in that thread. If you are not comfortable editing source code, just wait until the changes are added to CVS.

Q. How do I edit my ShowEQ configuration files?
A. I suggest using vim. Here is how:
Open a command window if you're in X
vim /usr/local/share/showeq/showeq.conf
Hit the A key
Edit whatever you need to.
Hit the Esc key
Type :wq and hit Enter. (:w saves the file and :q quits, that just combines both commands)

Q. Can somebody make a Windows version of ShowEQ?
A. No.

Q. Why do players only update when I get near them?
A. Because the servers only send player update information when you are within a certain radius of the players. This is not a ShowEQ-related problem.

Q. Why does it say <=0 for all NPC hit points?/How do I get ShowEQ to display NPC hit points?
A. Because no HP data has been sent for those NPCs/You can't. Again, not a ShowEQ-related problem. Hit points are now sent only as a percent value, and only when you get within a certain radius of the NPC.

Q. Why doesn't ShowEQ display what loot the NPCs are carrying?
A. Because it can't. The server only sends data about what is visible on the NPC, not loot such as coin.

I hope to add more to this at some point, and if you have something that you think should be added, reply with the question and answer and I will edit it into this post and give you credit for contributing it. :)

Q: I have showeq configured and on my network at home but nothing is being displayed. (or something similiar)

A: Make sure you have a *hub* not a switch at home. Hub's broadcast the packets across all ports while a switch only sends the packets where they are supposed to go.

A2: Make sure you edited showeq to point at the IP address of your EQ box. You can specify the IP address in the command line of showeq like this "showeq -i eth0 <IP>"

-------------------------------------------

Q: What is and where can i get a filters.conf file?

A: Filters work against the mob name and are used for things such as highlighting rares, warning about uber mobs, filtering out low level mobs, ... You can download a fairly comprehensive file at www.geocities.com/tucciim/filters.conf.txt .
