Zombie 		posted 05-20-2001 08:14 PM         
If everything is plugged into a hub it will work. You don't *need* a hub if you're using the Linux box as a gateway; in that case you can use a crossover cable to connect the Windows and Linux computers, but you might as well use one anyway in case you want to add computers to the network in the future. If your Linux box is the gateway you can use a switch to connect the computers in the LAN, but simply plugging everything into a switch will not work.
