*grin* Well, let's see.

1) It's an entirely different operating system. You don't have to get rid of windows to use it, but you can't run linux and windoze at the same time.

2) Unless you want all the manuals and tech support, you don't have to buy anything. You can get the entire distribution from places like ftp.cdrom.com and www.redhat.com.

3) YES!!!

4) It depends on how much effort you're willing to put forth. It's not as dummy-proofed as windows, but it is fairly intuitive once you're thinking on the right wavelength, and a heckuvva lot more powerfuller. (<--- please note, bad grammar used for emphasis )

The short of it is this, however: If you're going to install linux to use ShowEQ, you need two computers. One to run Windoze (and EverCamp), and one to run ShowEQ on Linux.

Hope it helps!

Ahh, yes, if using a standard modem for dial up, then here is the preferred method:

1. Put modem in PC running Linux.
2. Configure Linux for PPP dial in.
3. Configure Linux for IP Masq (As Per HOW-TO)
4. Set Windows PC to have default gateway of Linux Box IP address.
5. You are good to go.

I understand that step 2 can be quite a pain. Been on routers long enough that I have never needed to try it myself.
