Well, I'm sure someone will correct me if I'm wrong but I beleive All the showeq files (except libEQ.a) are in /usr/local/share/showeq and the showeq executable is in /usr/local/bin by default. So all you should have to do is remove /usr/local/bin/showeq and then remove the directory /usr/local/share/showeq and all it's contents. also if you want to remove the libEQ.a library it's /usr/lib/libEQ.a

Also you'll probably want to remove all the source files but those will be where ever you extracted the Tar ball or downloaded the CVS tree to.
