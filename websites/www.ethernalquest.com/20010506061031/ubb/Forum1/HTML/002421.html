Hello! I'm not sure if anybody remembers me... been around for a while. But ya, good to see you Draigon!
And about B&W, the game is great. I'm sure you've all figured this out by now, but if not, here's a little tip:

hex edit the file c<ID number>.err and physique_c<id number>.err in the /scripts/creaturemind directory (id number can be found in the registry under your profile). The first byte in physique and 5th (I think...) in the other file should be the same. You can change these to any numbers between 0x00 and 0x10 to change which type of creature you have. (only way known to get the ogre is to change them to 0x0D).

Anybody have any info on editing the meshes yet? It would be nice to make my own levels... the scripts seem straightforward (CREATE_NEW_TREE(x,y,z,size) type of thing).
