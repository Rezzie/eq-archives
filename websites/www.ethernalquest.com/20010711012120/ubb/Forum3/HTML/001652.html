Decaying Skeleton 		posted 05-11-2001 12:35 AM            
try this:

sed -e s/\*/\.\*/g spawns.conf > filters.conf

That will change all of the *'s in your file to .* (the reg-exp equivalent). You might also want to run something like this:

sed -e s/$/\/

which will add a / to the end of every line, so that your filters will only match the name of spawns intead of also matching their phat lewts (i.e. Oracle.* won't match C:OracleRobe)
