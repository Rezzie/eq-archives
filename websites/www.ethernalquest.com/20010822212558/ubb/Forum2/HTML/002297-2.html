You have to have two files.

D3deverquest.exe
eqhost.txt

place those two in your Eq directory THEN run.

If you wanna use YOUR own server:

Open eqhost.txt

change it to look like this:
Lets say your IP is 156.98.67.9 port 5999

[Registration Servers]
{
"eqlogin1.989stu.com:599"
}
[Login Servers]
{
"156.98.67.9:5999"
}

Simple huh=)

Now IF you STILL did not get it the format is this:

[Registration Servers]
{
"eqlogin1.989stu.com:599"
}
[Login Servers]
{
"'YOUR IP':5999"
}

What is a IP?

When you run Loginserver.exe(asfter you hacked it, changed the version in Loginserver.js and make that read-only)

There will be somthing called "Local Machine name " and that is your IP

What is the "5999" for? It is the port DO NOT change it.
