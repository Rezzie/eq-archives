Well I'm assuming you play EQ on the primary machine? In that case it won't work for ShowEQ because since the EQ packets are sent to that computer's IP, that computer does not forward them to the rest of the network. If you don't want to use multiple IP addresses the way to set it up would be to replace the current primary computer with the Linux box and set up Linux for IP Masquerading (see WhiteFire's networking guide or the HOWTOs at http://www.linuxdoc.org).
