One that would log into your account and play while you sleep to get you levels. My interest in this comes from something I wrote for Xenix (like Unix) as a kid. It was basically a MUD. I think it might be possible to do this in EQ by understanding how to manipulate the packets going to and coming from the server. Just a thought .

Learning a programming language is like learning any language, it takes practice.

To see the quickest results, and be able to write usefull applications quickly, learn VB first.

But dont stop there.. to be able to do everything you want to, you will want to learn C++ also.

C++ is a very powerful language. It is the language windows was written it. You can do just about everything in windows using C++. But with this power comes complexity. Just changing the background color of a form is a pain in the arse, where in VB its a no brainer.

Another nice thing about VB is that things that you normally cant use, you can amost always find a component that does. Want to make a form round instead of rectangle? Can't do it with VB.... unless you use an activeX control!

To sum up.. learn VB first. It will get you into real programming quicker. Just dont stop there.
