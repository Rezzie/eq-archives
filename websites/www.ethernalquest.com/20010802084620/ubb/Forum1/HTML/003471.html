Phi
Zombie 		posted 07-31-2001 12:34 AM         
If you don't get either expansion, you can't pass level 50, if you get either one you can go up to 60.

If you don't get Kunark, you can't create an Iksar character, and lose out on many greatly populated exp. zones (Lake of Ill Omen for levels ~17 to ~30, Dreadlands for 40+ and more)

If you don't get Velious, you can't use the interface enhancements that it comes with (although IIRC you can install Velious off someone elses CD and get the interface enhancements, but you still can't GO to Velious without your own CD key) and Velious items are fairly awesome for when you can get them, and it is usually much easier to get them yourself than rely on buying them (and some of the nicer stuff from Velious is no drop, meaning that you can't trade it, so you have to be able to go and get it yourself.)
