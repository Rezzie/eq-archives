Heh - depends upon how many ports your router has. If the number of ports >= the number of computers you're good to go! If the number of ports is < the number of computers, you need a hub.

If you need a hub, don't go cheap. Spend the $100 (or so) and get a 10/100 Hub. I highly recommend the 3Com 10/100 Switch because it allows SEQ to function (I know because I own one).

If you buy a switch other than the one I recommend, beware. Many switches advertise discrete BW to each port. This means SEQ won't function.

Good Luck!
