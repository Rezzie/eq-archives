I am finding the text window of great use as it provides me with flexibility not readily available within EQ. I find it incredibly handy that I can check and uncheck specific types of communications and the window updates.

Please consider this and let me know what you think:

Here's what would allow me to take the greatest advantage of this window-

1. Multiple-windows! Let me start additional windows so I can have one window just-for-auction and another customized by the way I check the check-boxes. Let me have as many windows as I want to start and my system can manage!

2. Color! Let me associate color with a window's text and specifically with each option I can check, similar to EQ.

3. Search! Let me put key words into a dialog box (separators like ; to keep together, space to separate words, "exactly this text", AND, OR). To help communicate what I'm looking for I'll give this example. I start a new text box and select the auction-check-box. I hit the criteria button and type in ( WTS AND HQ;Bear "Sarnak Tabbard" ) ignore "(" -- this would highlight in a color I choose a line of text that said "WTS HQ Bear send tells please" and not highlight "WTB HQ Pelts" it would also highlight any line with Sarnak Tabbard in it.

If this, or some functionality is already there, I'm still researching - just point me in the right direction.

Your time and consideration sincerely appreciated.
