Just followed curious necro's corrected instructions (as follows) and zone into luclin zones with brilliant color on all the mobs!

Started with a clean source tree in a fresh directory. Redhat 7.2 with all the standard libraries.

(the following posted by curious necro)

cd /usr/local
wget http://www.base-nine.org/seq/SEQ3.1.2.tar.gz
wget http://www.base-nine.org/seq/3.1.2_upgrade_to_121301.tar
tar -zxvf SEQ3.1.2.tar.gz
tar -xvf 3.1.2_upgrade_to_121301.tar
mv everquest.h linux/src
mv decode.cpp linux/src
cd linux
make -f Makefile.dist
./configure
make
make install
cd /usr/local/bin
./showeq

Thanks man! I love it.
