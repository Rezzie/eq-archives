I ran into this problem with my POS (Piece of Shhhhh) laptop running RH 6.2. I had a 10/100Mbit hub and my Win98 system is a 10/100Mbit nic card but my laptop is only a 10Mbit nic. The packets will not get passed from the 100Mbit thru to the 10M nic. Try setting your 100Mbit nic to only 10Mbit. Or get a 10Mbit hub. This is my config and it works pretty good.

        Internet(Cable Modem)
                     |
                     |
 Linksys 4 port router/firewall(DHCP)
                     |
                     |
                10M Hub
                   /   \
                  /     \
                 /       \
       EQ Win 98   SEQ Linux
        10/100M       10M

Decaying Skeleton 		posted 09-18-2000 05:06 AM         
Make sure you plug your 10M hub into any port other than the uplink port on the linksys. BUT be sure to plug the network cable into the uplink port on your 10M hub and if there is a little switch on your hub that says NORMAL or UPLINK, put it to uplink.

Also make sure your linux box has the same gateway number (IE 10.0.0.2, not the words such as PROXY-SERVER.ANYWHERE.COM). Use traceroute to get the ip of your gateway if needed.

It should work. Hope this helps.
