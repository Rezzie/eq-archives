well, i dunno if this has been posted already in the past, but what the hey. my friend showed me some totally l33t winamp plugin that lets you run songs thru eq without use of eqw, etc.

here's an excerpt from the makers themselves:

Sneaky Amp is a funny little Winamp plugin that allows you to control Winamp's various functions without direct acces to Winamp.

That sounds awfully complicated, but it's not. All Sneaky Amp does is watch the keyboard, and when you type certain commands it manipulates Winamp for you. Quite useful for using Winamp within alt-tab unfriendly games (such as Everquest) or just for playing .mp3s without having to dig through tons of mp3 folders.

Examples using the default options:

-If you wanted Winamp to start playing, you would type winamp play"".

- If you wanted Winamp to play the song with "Dance Fever" in the filename, you would type winamp play dance fever""

- If you wanted Winamp to play a random 10 songs with "fish" in the filename from the directories with "bean" in them, you would type winamp play fish rand 10 -bean"".

For full instructions, read the SneakyAmp_ReadMe.doc file that is installed in Winamp's plguin directory.


so, sorry for bringing up an old topic if its been mentioned already, but i found this little prog to be very tight.

oh, and the URL for it is:

http://www.execpc.com/~zor/SneakyAmp/download.htm
