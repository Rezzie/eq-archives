1. Print out the instructions for installing OE on RH7.2, you'll be following those almost to the "T"

2. check for the following libraries:

rpm -qa | grep 'pcap'
This should produce:
libpcap0-0.6.2-1mdk
libpcap0-devel-0.6.2-1mdk.

You can get libpcap on disk1 and pcap-devel is on disk2 of the Mandrake 8.1 install disks.

rpm -qa | grep 'Mesa'
This should produce:
Mesa-common-3.4.2-2mdk
Mesa-common-devel-3.4.2-2mdk
Mesa-3.4.2-2mdk
Mesa-demos-3.4.2-2mdk

I believe you need all of these, so check disk 1 and disk 2 for the appropriate RPMs.

3. If you have an Nvidia card, then you can download the NVIDIA_kernel and NVIDIA_GLX files and install them. Also download the README and make sure you modify your /etc/X11/XF86Config-4 file as specified in the README.

4. Go into X and run gears --info from an xterm. If this works, then your OpenGL is working.

5. Now download qt-3.0.1 as specified in Slicer's instructions and compile it the way he says to.

6. Now grab odinseye and daoccrypt and install/compile them per Slicer's instructions. Don't forget to set the QTDIR.

7. Run/test OE as specified in Slicer's instructions.

Enjoy!
