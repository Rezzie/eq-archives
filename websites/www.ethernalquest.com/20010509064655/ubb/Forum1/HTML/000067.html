It is simple... just read the IPCHAINS doc. There is an easy 3 step howto.... Here is the information for it.

# echo 1 > /proc/sys/net/ipv4/ip_forward
# ipchains -A forward -i ppp0 -j MASQ

You also need to define the default route for your output.
# netstat -rn
and see if the default gateway goes to the internet. If not you want to issue the following command

# route add default gw ipaddressofg
