I think your best bet here is to run to the local computer shop and pick up a 4port hub.

If your budget allows I would suggest a linksys DSL router/firewall device. This product is designed specifically for DSL/Cable internet connections. If you were using an x-over cable with dial up, and 98 I'm going to assume you were using Internet connection sharing.

While Inet sharing under 98 is "supposed" to work with any connection, I found it terribly cumbersome using anything but dial up.

Save the headaches, spend a few bucks, protect yourself, it's worth it in the long run IMO. Check this link out:
