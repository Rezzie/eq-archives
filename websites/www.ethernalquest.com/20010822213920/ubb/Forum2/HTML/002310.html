Cool! thanks for figuring this out Septimus.

Let me explain what happens when you say "Append Item". I simply add that item to the end of the file and I expect that there isn't already an item with that itemnumber in the file you append it to. I don't have a replace or update item functionality yet (sorry). But I'm glad that Septimus' method works!

Also, the "Save Item" is if you want to store that item in it's own file. So you can, for instance, make an item you like and save it as "Dagger of Cheating.db" and then you can load it up later and stuff... A future release will have lots more functionality, promise Just can't promise when, I'm putting all my free time into the EMU.
