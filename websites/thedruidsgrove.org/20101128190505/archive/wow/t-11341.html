Does anyone know of a good replacement for DruidMod?

I've used it for a long time but it hasn't been updated in ages and now doesn't work at all in 2.0.1.

I liked it mainly because it gave me macros to replace my spells which when used would selfcast if I had a enemy or nothing targeted or would cast normally if I had a friendly selected.

I know I can alt+hotkey for selfcast but just hitting a number and having it react smartly is VERY nice.

It worked for all 3 types of heal, remove poison, and remove curse.

There is a key binding to turn on and off auto self-cast, which does not require a hotkey to self-heal. I bound that to my "`" key, so whenever I PvP or solo, I turn it on. When I'm in a group/raid, I turn it off.

If you're planning on having it on all the time, you don't need to bind it. You can go to your interface options screen (forgot if it's in basic or advanced), and check the Auto Self Cast option.

Actually with help from wowwiki I found out that this is really easy with the new macro system.

/cast [help] Rejuvenation; [target=player] Rejuvenation

If the target is friendly cast Rejuvenation else selfcast. Replace "Rejuvenation" with the name of whatever spell, buff, decurse, or depoison you like.
