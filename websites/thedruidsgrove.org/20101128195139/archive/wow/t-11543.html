Yeah it has a learning curve to it, but once you figure it out it can be quite useful. I've got it doing two main things for me currently. First off I have outfits set up in Cat, Bear, and Healer sets. When I exspect to be in that form for any length of time I have a main set button I use to put it on. (Name your sets with a set prefix to keep them organized for upkeep. aka gear-cat, gear-bear, & gear-Heal) Second I have weapon/Idol sets. These are set to change as I shift and use the weapon and Idols that will be most useful in the form I'm currently in. These can be set on ItemRacks event tab for druid forms. I have seperate sets for these. Even have fun sets like fishing sets and such.

To get the sets button you Alt+click your picture in the character screen. I also have my trinkets set out so that they can be switch/clicked from that hotbar so my other bars are freed up for other things.
