Elysians are undead so you should have no problems killing them as a 57 necro, main problem you may have is single pulling them with no harmony. Best place to pull to is the corner as there are no spawns there and no wanderers that path through. If you agro kite just make sure there are no druids root doting in your path since we normally don't kite elysians and wouldn't be looking for that.

I didn't use harmony once when hunting elysians. They seeinvis, so look for one, and proximity aggro it. Run it to the valley, kite it, loot, repeat.
Kineada
Go to safe valley. Invis, SoE (or SoW/lev) and pull from the north side of the valley (wanderers). Elysians see invis so it's possible to aggro pull them through a crowd of regular mobs. Just run close to one then back up. If it's chasing you, run back to safe valley and kite it. If not, run up again and back up. Takes a bit to figure out how close you need to be to gain proximity aggro without getting hit (which drops invis which gets you gang raped).
