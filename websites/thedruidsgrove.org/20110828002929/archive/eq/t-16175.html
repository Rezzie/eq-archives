Greetings,


We are currently seeking a new druid class correspondent.

The class correspondent is responsible for helping the Community Team collect information about the class to pass along to the development team. This could range from bugs found, class balance issues, or general questions about the class.

If this sounds like something you would enjoy doing and you have a good feel for how your class works please send me (Zatozia) a PM with the following information:
Character Name:
Server:
Level of your main Druid Character:
Guild Name:
Guild Website: (if applicable)
The names of any other EQ fan sites you regularly post on and the handle you use on those EQ fan sites.
Brief description on why you would like to be your class's correspondent.The deadline to submit your application is 8/14/2007. Please note that we are only accepting applications not nominations.


Thank you and good luck!
