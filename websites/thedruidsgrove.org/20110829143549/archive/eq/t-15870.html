I had a similar problem but is wasn't Vista, it was the screen resolution. If you have a widescreen monitor try lowering the res. to a common one like 1024x768 and see if EQ works (that fixed it for me). You can than edit your EQclient.ini to see if your system will allow EQ to run in a higher (wider) res.
