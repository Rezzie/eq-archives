Palarran
01-02-2008, 04:41 PM
The type 1 and 2 augment slots are "hidden" aug slots related to how power source bonuses are implemented on Infusible armor. All infusible items (items that are affected by power sources) have these aug slots.

On any item, each aug slot has a corresponding field that indicates whether the aug slot should be displayed (and whether augments can be inserted normally). Think of the type 30 aug slot for the epic 2.0 to be used by the epic 2.5 augment; I believe this was the first time an invisible aug slot was used.

Lucy has started to hide aug slots marked as invisible. Magelo has not. On Lucy you can look at the raw data and see:
aug1type = 7 (first aug slot is Type 7)
aug1vis = 1 (first aug slot is visible)
aug2type = 13 (second aug slot is Type 13)
aug2vis = 1 (second aug slot is visible)
aug3type = 1 (third aug slot is Type 1)
aug3vis = 0 (third aug slot is NOT visible)
aug4type = 2 (fourth aug slot is Type 2)
aug4vis = 0 (fourth aug slot is NOT visible)
