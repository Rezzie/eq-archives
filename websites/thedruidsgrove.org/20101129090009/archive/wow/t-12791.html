Avatars

There is currently a bug with setting avatars on the Grove's WoW forum. To set an avatar, please go to our EQ forums (http://thedruidsgrove.org/eq/forums/), and log in with the same username and password you use here. Go to User CP > Edit Avatar and you should be able to upload an avatar, which will then also show up on the WoW forum here.

Signatures

From the posting rules (http://wow.thedruidsgrove.org/forums/showthread.php?t=8002), on signatures:

Signatures can consist of multiple images which should be kept to a total size of under 85k and max dimensions of W:500 x H:150 pixels. Signatures which fall outside of these boundaries and cause malformations of the forum structure will be immediately removed. As always, Moderators have full authority to remove any signature.
