I did this for awhile, as this model has moves for ravage, claw, and ferocious bite. However, changing the model is a global change. What it requires you to do is unpack the mpq files and then locate the specific model you want. Change the name of the file to the same as the current druid skin. Then delete everything else but these files and their respective hierarchy. DO NOT REPACK THE MPQ as blizz checks for changes to this and will require you to uninstall and reinstall. Then, if you did it properly, you will appear as a tigon on your screen only. All other druids that would use the same model will appear as a tigon too. It's coll for a bit but then gets annoying.
