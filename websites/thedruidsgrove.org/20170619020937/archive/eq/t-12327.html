Logitech released their new Gaming Keyboard (http://www.logitech.com/index.cfm/products/details/US/EN,CRID=2288,CONTENTID=10717)last week. I saw them at Best Buy so I picked one up.

I have to say, This keyboard rocks. It looks good, feels good to type on and has a ton of great features. Notable features include:


The Ability to disable the "windows" key to prevent accidental deaths because you fat fingered the keyboard.
Two USB Ports on the Keyboard itself so you can attach your mouse and joystick easily.
18 programmable keys to record macros
back lit keys so you can read the keyboard in low light.
No stupid "F-Lock key" that changes your F keys to other functions.
A LCD display that shows Time/date, System CPU/RAM usage, and will soon be able to display gaming information such as health, ammo, etc.


If you're looking for a good keyboard, I recommend this one.

I'm not sure if you can use the G keys seperately and map them from within a game. I'll try when I get home tonight.

But.... There are two ways to program the Gkeys.
1. There is a program button on the keyboard that lets you map keys /macro's on the fly.
2. The applet that comes with the board, lets you create profiles for each game and it lets you create your keys/macro's.

There are only 18 G-Keys (all on the left side) but, each key can have 3 macro's programmed to it (via a select key above the G Keys) per profile. Profiles can be automatically loaded when you start your game. So, you really have 54 keys per profile, which has no limit.

When programming macro's, you can have complete sentences programmed. You can even have it retain any delay between keystrokes.
