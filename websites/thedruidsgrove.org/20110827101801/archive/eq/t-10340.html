First of all, the second link you presented lists a typ7-item.
None the less, the different aug-types are based on what kind of item it is.

Type1-3 = LDoN bought-items
Type4 = Weapons
Type 5-6 = I'm not really sure if they exists. Never seen one and too lazy to search the net.
Type7 = EP-gear and below. Some GoD. Some OoW. Some DoN (lower lvl of gear).
Type8 = PoTime and equally (or better) gear.

The more slots the better (in that regard), but to determin if an item with many slots is better than an item with only one slot you need to add what augments you can reasonably add to the items in a near future.
