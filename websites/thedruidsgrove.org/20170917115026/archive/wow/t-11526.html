You're wasting mana if you stack lifebloom. Only the first cast gets any benefit from your +heal gear and the blast of health at the end doesn't stack. The only time stacking lifebloom is good, is against a dispel/purge happy opponent in PvP. It heals for 600 or so each time part of the stack is purged/dispelled.

You're better off tossing on a rejuv and one lifebloom and letting the bloom tick for the full amount.

Lifebloom is far more mana effective if you use it once, let it expire, then use it again. Reason being, +heal only counts on the first time you apply it. On subsequent application, it just gets the base heal for the spell, which is minimal. And if 3 are stacked, it doesn't 'bloom' for 1800 at the end, but rather 600.

So tis better to rejuv, lifebloom, cast a ht, re-lifebloom, repeat.
