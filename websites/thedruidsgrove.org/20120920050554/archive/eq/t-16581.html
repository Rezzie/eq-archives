Can you please use the same code for Illusion : Fairy too please? They're huge when you use the illusion too.

The change we have to make would fix that problem too. The problem is that there is no way to specify height on an illusion so the models use the default height which either has to be hardcoded per model or defaults to the standard human size (6 foot). Illusions that change size, like gnome, have a hard code inserted that override the height. The change we're making will allow us to specify default size by model in data so we can set that information for all models and fix this problem with all illusions.

Rashere
