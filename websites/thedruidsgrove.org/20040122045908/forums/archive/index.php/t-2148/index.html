Jarilon
The one time I did this it was in a group that had two 65 enchanters, both with the high level charm spell. I would harmony so that we pulled the Wrulon single and sic both pets (Guard or Guardian ) on them. We had a melee near the fight calling the AE's and the rest of us waited a ways off out of the AE range. When the AE was called, those that could would run in and nuke, dot, heal, etc. It was quite an effort just to kill one mob but it worked. We used the same tactic for the Recurso (sp?). Hope this helps.
