I just did this a few months ago.

- Make sure you install all the CD's in Order (original/kunark/velious/luclin/pop), with no patching in between.
- Then patch those (with no extras checked in the Advanced page of patcher).
- Then check off LoY, close and re-patch.
- Then check off LDoN, close, and re-patch.
- Then check off GoD, close and re-patch.

I also re-booted between each CD and each patch.

Hope that helps
