There are 3 issues:

Compatibilty mode:
1) Open up windows explorer, navigate to your ..\Everquest folder.
2) Find and right click EverQuest.exe, click properties, click the compatibility Tab.
3) Place a Checkmark on the following but don't click OK yet:
-Run this program under compatibility mode for Windows XP(Service Pack 2)
-Disable Desktop composition
-Run this program as administrator
4) Click "Show settings for all users", place the same Checkmarks as step 3, click OK
5) Click OK the properties dialog opened in step 2 and waited on in step 3
6) Repeat steps 2 thru 5 for eqgame.exe

You will also have to get Directx9 reinstalled as Vista comes with the upgraded Directx10. You need to download the June 2007
DirectX redistributable at http://www.microsoft.com/downloads/...&displaylang=en

Dual Core issue:
EQ doesnt play well with dual core AMD chips. I assume multicore Intel chips have the same problem but don't know that for sure. Symptoms for me were flakey graphics like turning left or right extremely rapidly or moving forward in almost a warping fashion.

The quick fix is to set affinity for eqgame.exe to a single processor in Task Manager, but you have to do that every time you restart EQ. The better way is to download and install the Dual Core Optimizer from AMD website.

www.amd.com - DualCoreOptimizerSetup.exe
