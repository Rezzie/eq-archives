Hello new druid ! :>

You will find it a lot easier to help your wife if you can tank instances, so I would suggest Feral for now, since that will allow you to tank. Balance is a good levelling spec too but doesn't allow you to tank, however you might prefer it if you want to have a similar sort of downtime for drinking etc as your wife's mage.

For feral gear, initially you want stats that are good for cat form so that you can kill things quicker while questing. So for cat stats, get agility, strength, attack power, crit chance, etc. Those are roughly in order of importance, from most to least important.

When you start tanking instances, especially once you are level 66+, you will want to start using some bear gear, which includes heavy armor, lots of stamina and lots of agility.
