Here is a guide to the value of ac vs hp, and avoidance.

http://reborninfire.proboards91.com/index.cgi?board=guide&action=display&thread=1192313507

The basic point for shield ac is this: shield ac is special, its not subjected to the soft cap on worn ac, this guide states that
10ac on your shield = 250hp.
So a 30ac gain on shield is, according to this guide, equivalent to 750hp.

Note: this is true for shield ac only, not for other secondary slot items with ac.

so go out and get that best shield and ac aug! ( and if anyone has any info that contradicts this guide, please post. )
