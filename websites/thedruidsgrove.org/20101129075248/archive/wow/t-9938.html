Thottbot has nearly every quest ever discovered in game as far as I am aware. Usually there are some comments below the quest to help out visitors to the site as well. In the search box just type in the name of the quest and if it is a series type of quest, all the quests in the chain should be listed there as well.

Allakhazam has pretty much the same information with perhaps a bit more - I have been using Thottbot since beta, and never have I hit that site and not found what I was looking for.
