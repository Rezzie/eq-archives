The description of Spell Casting Subtlety reads: "After training this ability, NPCs will notice your magical activities 5, 10, and 20 percent less." Arguably, this is one of the better GoD AAs so I went out and tested it to see if the description holds true. (Note, this ability costs 3/6/9, as opposed to the caster version which costs 2/4/6.)

The test was rather simple. I generated a high degree of taunt with one character on the usual Guinea pig (mist panther) and then attempted to taunt off with snare, counting the number of casts it takes. I used two different taunt values and repeated each test twice (it was the same in both instances, each time). Positioning of the characters was about 75 paces and remained constant. Memory blur reset the subject after each test until it retuned to indifferent.

Premise: if snare generates X taunt per cast, and if, per the description, SCS makes the target notice the snare 20% less, then we would expect post-SCS snare to generate .8X taunt per cast. That would mean it would take <b>25% more</b> hits of snare to generate the same taunt.

<b>Test 1</b>: Taunt is Tashan + Slow + Tashan + Slow. Without SCS, it took 5 casts of snare to taunt off. With SCS3, it took the same 5 casts of snare to taunt off. This test suggests that the actual effect is >= 0% and <= 17%. This is inconsistent with the premise; if it took 5 casts before, and you now have to cast 25% more, you would expect it to take 6 or 7 casts, depending on how much you taunted past the break point the first time.

<b>Test 2</b>: Taunt is Tashan + Slow + Cripple + Tashan + Slow + Cripple. Without SCS, it took 6 casts of snare to taunt off. With SCS3, it took <b>7</b> casts of snare to taunt off. Clearly SCS3 is doing <b>something</b> here. This test suggests that the actual effect is > 0% and <= 25%.

While my crude test doesn't answer the question definitely, it answered my two most pressing concerns: (a) this ability does do <b>something</b> to affect taunt and (b) that effect is to lessen the taunt (believe it or not, there was a time where this ability was rumored to work in reverse). I hope someone else has the time to test this skill more thoroughly so that we can verify that the in-game description is correct.
