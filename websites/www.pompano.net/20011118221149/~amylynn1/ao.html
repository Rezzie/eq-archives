Attribute Calculator


The Attribute Calculator lets you find out how many IP (Increase Points) it will take you to gain any amount of attribute levels!

Download the calculator here (Version 1.1 - 7 kb)

This is a Visual Basic Program. If you are running Windows 95/98 (NT/2000/ME should not need these files), and the program will not run, download this support file, unzip it, and put it into your c:/windows/system directory. If you have a newer version of this file, do not replace it:

msvb6dll.zip (MSVBVM60.DLL - 687kb)
