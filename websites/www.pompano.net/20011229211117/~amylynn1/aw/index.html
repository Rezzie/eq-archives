AWMapEd - Advance Wars Map Editor


AWMapEd is a map editor for the fantastic Game Boy Advance title, Advance Wars! The Editor will allow you to recreate the maps seen in the game (to allow more precise strategies), or to create maps of your own that you can share with others!

Download AwMapEd here (Version 1.1 - 134 kb)

This is a Visual Basic Program. If you get an error that says any file is missing or 'out-of-date', and the program will not run, download these two support files, UNZIP them, and put them into your c:/windows/system directory. If you have a newer version of these files, make sure to back up your newer files before replacing them with these:

msvb6dll.zip (MSVBVM60.DLL - 687 kb)
sysfiles.zip (8 Support Files - 700 kb)


There are also more details instructions in the readme.txt file included with awmaped.zip!
