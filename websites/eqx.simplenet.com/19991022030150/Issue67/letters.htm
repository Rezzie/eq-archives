Troll in Kelethin Part 1 - Review

Fantastic read. Love it. Two thumbs-up. Five stars.

M. Steele
(Nukulbonz - "Yu bleed gud fer Nuk, no cry much lil frogz." - Troll Shaman of Innoruuk - PvP Server)

While having to admit that the game is quite an accomplishment, I am already very much addicted, there seems to be something inherently wrong with releasing the game before what impact the loads on the logins had been established. The Phase Four beta seemed to be about just that..and the performance in the first few days was dismal at best. Although the logins seemed to get better as time went on..the last night of the beta should have been some sort of indication of what was to come. Zone crashes, lag, and the like were very prevalent on the Tarew Marr server. In fact the last day I noticed a good many graphic bugs (Floating trees, teleporting monsters, etc.) and felt the beta probably should have gone on a bit longer. The bottom line is this is a product..like any other. "Ironing Out" should have been done in the betas..and as of this writing the login issues have not been resolved. If this is not solved soon ( today is 3/21 ) Sony should think about extending the "Free Month" until it is..

Thanks..
Frank Pupo (Sallow)
