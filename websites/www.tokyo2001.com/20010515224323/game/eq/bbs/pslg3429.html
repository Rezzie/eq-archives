・Hero Bracer 貰うために必要な faction

Steel Warriors ( FreePort & Qeynos Warrior ギルド ) が kindly に
近い amiably が必要

・Steel Warriors の faction の上げ方 from indif or app

FreePort での DeathFist Belt Quest か Nro の Dorn B'Dynn を殺す
amiably になると Qeynos Steel Warriors からの Quest も実行可能
でも効率は Deathfist Belt が一番かな？
Wiz か Dru なら Kobold Hide Quest も楽ですね( 要 amiably )

・Mino Hero のスポーン条件

Place Holder は、地面にささった歯車のところにポップする skelton と
Mino Cave 入ってすぐの minotaur slaver です
ただし、minotaur slaver の殺し方が重要！( 考えてみて下さい )
Track してるとわかるんですが、minotaur slaver を手順どおりに殺すと
当たりの場合は Minotaur Hero が、ハズレの場合は Mino Cave 奥に
Qeynos か Ak'non 絡みと思われる NPC がスポーンし、すぐ殺されます
( ログとってないので名前失念 )
Quest の臭いがプンプンしますが、レベル的に cave 奥探検不可能(泣)
