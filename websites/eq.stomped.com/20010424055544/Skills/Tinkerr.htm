    Tinkering Recipes

    If you wish to contribute recipes for creating items, please e-mail me

    (Thanks to Tolthis for sending most of the recipes and book reviews)

    For a discussion on tinkering, go to the Tinkering Strategy Discussion Section.

    The following are recipes for creating various items via tinkering:
     

    (NOTE:  All tinkering is done with a TOOLBOX to use to combine the items. This is for sale only in the Windmills outside of Ak'Anon, but you can also have a blacksmith make you one.  Velious timkered items need to be made in the delux toolkit, which is purchased in the pirate camp in the Iceclad Ocean)
    • Animated Bait = Cork + Spriket
    • Aqualung = 1 fine coral mesh + 1 Silk Lined Steel Helm + 1 metal twine + 1 Gnomish bolts + 1 metal rod + 1 fresh fish
    • Cams (improved longbow)  = Gears + Gnomish Bolts + Grease.  (In Gimblestan Vol XIV)  Cams are pulleys that can be used by Fletchers to make composite bows.  By adding one or two cams to the usual staff and string combo in a fletching kit, they can make a composite bow that shoots much farther.
    • Catapult = 1 shaped ashwood recurve  (hemp string ,ashwood stave,planing tool) + 1 sproket + 1 gear + 1 metal twine + 1 fire water   (gnomish spirits + water)
    • Collapsible Fishing Pole = 3 Metal Rods + Gnomish Bolts
    • Compass =  Static Orb + Skewer + Pie Tin + Water  + Cork.  (In Gimblestan Vol XIV)
    • Firewater = Gnomish Spirits + Water
    • Flameless Torch = Large lantern + Firewater + Metal Twine.  (In Gimblestan Vol XIV)
    • Footwarming Boots = Unstained Smal Fine Plate Boots + Firewater + Sharkskin Toubing + Two Metal Rods + Metal Twine + Gnomish Bolts
    • Gemcutter = 1 metal rod + 1 diamond dust + 1 standard bow cam + 1 metal fastening + 1 firewater + 1 grease
    • Gnomish Fireworks = Firewater + Bat Wings + Metal Shaft
    • Gnomish Vanishing Device = Diamond + Small Lantern + Steel Wire + Gnomish Bolts + Reflective Shard + Haze Panther Eye + Class five mana battery (can be

      • recharched by replacing battery)
    • Mechanical Lockpick =  Lockpicks + Gears + Sprockets + a Rat Ear.  (In Gimblestan Vol XIV)
    • Powered Gloves = 1 Steel lined gloves + 1 sprocket + 1 gear + 1 metal twine + 1 firewater (water & gnomish spirits)
    • Rebreather = 1 aqualung + 1 lime coated meshing + 1 sharkskin tubing + 1 medium size platemail helm + 1 firewater + 1 metal fastening
    • Spyglass = Reflective Shard + Metal Rod + Bottle + Metal Twine + Collapsible Fishing Pole.   (In Gimblestan Vol XV)
    • Stalking Probes (Don't ask.  I don't know.  Read the caution about exploding Scouts though) =  2 Sets of Gears + Metal Rod + Bottle + Firewater.    (In Gimblestan Vol XV)
    • Telescopic Eyepatch = Low quality Dire Wolf Fur + Gnomish Bolts + ReflectiveShard + a Velium lens
    • Thermal Cloak = Tattered Gnomish Cloak + Metal Rods + Firewater  (In Gimblestan Vol XV)
    • Watchman Boots = Unstained Small Fine Plate Greaves + Cogs + Sprockets + Grease + Faun Hoof + Class five mana battery + Metal fastening + metal twine + gnomish bolts
    • Watchman Bracer = Unstained Small Fine Plate Bracer + Cogs + Sprockets + Grease metal rod + glob of tar + Class five mana battery + Metal fastening +

      • metal twine + gnomish bolts
    • Watchman Breastplate = Unstained Small Fine Plate Breastplate Cogs + Sprockets + Grease Gnomish Vanishing Device + Class five mana battery + Metal fastening + metal twine + gnomish bolts\
    • Watchman Gaunlets = Unstained Small Fine Plate Gauntlets + Cogs + Sprockets + Grease Block of living granite + Class five mana battery,  two Metal fastenings + metal twine + gnomish bolts
    • Watchman Greaves = Unstained Small Fine Plate Greaves + Cogs + Sprockets + Grease + metal rod + Holgresh Wing + Class five mana battery + Metal fastening + metal twine + gnomish bolts
    • Watchman Helm = Unstained Small Fine Plate Helm + Cogs + Sprockets + Grease + Sifaye Dust + Velium Lens + Class five mana battery + Metal fastening + metal twine + gnomish bolts
    • Watchman Vambraces = Unstained Small Fine Plate Vambraces + Cogs + Sprockets + Grease Holgresh Fur + Class five mana battery + Metal fastening + metal twine + gnomish bolts
