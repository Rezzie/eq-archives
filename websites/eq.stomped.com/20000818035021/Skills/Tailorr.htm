Tailoring Recipes


If you wish to contribute recipes for creating items, please e-mail me

 

The size of armor depends on the pelt:

Large armor (Barbarians, Ogres & Trolls) = Bear pelts
Medium armor (Barbarians, Elves, Erudites, Half-Elves & Humans) =  Cat pelts.
Small armor (Dwarves, Elves, Gnomes & Halflings) =  Wolf pelts.
 

Below are recipes for creating various items through tailoring:

  • Cured Silk Cap = 1 silk swatch + 3 Heady Kiolas + Pattern
  • Cured Silk Cloak = 2 Silk swatches + 2 Heady Kiolas [brewing skill] + Pattern
  • Cures Silk Gi = 3 silk swatches + 2 Heady kiolas + Pattern
  • Cured Silk Gloves = 2 silk swatches + 2 kiola + Pattern
  • Cured Silk Gorget = 1 Silk swatch + 2 Heady Kiolas + Pattern
  • Cured Silk Mask = 1 Silk swatch + 1 Heady Kiola + Pattern
  • Cured Silk Pants = 2 silk swatches + 3 Heady kiola + Pattern
  • Cured Silk Mantle = 2 Silk swatches + 2 Heady Kiolas + Pattern
  • Cured Silk Sandals = 1 silk swatches + 2 Heady kiola + Pattern
  • Cured Silk Sash = 1 Silk swatch + 2 Heady Kiolas + Pattern
  • Cured Silk Sleeves =  2 Silk swatches + 2 Heady Kiolas + Pattern
  • Cured Silk Wristbands = 1 Silk swatch + 2 Heady Kiolas + Pattern
  • Hand-made Backpack (10 slots instead of 8) =  High quality bear skin + backpack pattern.
  • Patchwork Armor = Ruined Pelt (per size needed) + Pattern for type of Armor
  • Raw Silk Armor = 1 Silk Swatch (two for leggings and tunic) + Pattern
  • Reinforced Leather Belt = High Quality Pelt + 2 Steel Bones + Pattern
  • Reinforced Leather Boots = High Quality Pelt (per size needed) + 3 Steel Bones + Pattern
  • Reinforced Leather Bracers = High Quality Pelt (per size needed) + 1 Steel Bone + Pattern
  • Reinforced Leather Cap = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern
  • Reinforced Leather Cloak = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern
  • Reinforced Leather Gloves = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern
  • Reinforced Leather Gorget = High Quality Pelt (per size needed) + 1 Steel Bone + Pattern
  • Reinforced Leather Mask = High Quality Pelt (per size needed) + 1 Steel Bone + Pattern  [may be broken]
  • Reinforced Leather Pants = High Quality Pelt (per size needed) + 3 Steel Bones + Pattern
  • Reinforced Leather Shoulderpads = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern
  • Reinforced Leather Sleeves = High Quality Pelt (per size needed) + 2 Steel Bones + Pattern
  • Reinforced Leather Tunic = High Quality Pelt (per size needed) + 4 Steel Bones + Pattern
  • Shadow Silk = Shadow Wolf Pelt + Silk Swatch + Scroll of Gather Shadows.  Add extra pelt for large.  Add extra scroll for Inky Shadow Silk.  (These are used for the Necromancer Armor Quests in Solusek Ro)
  • Silk Bandage = Combine 2 silk threads.
  • Silk Thread = Combine 2 Spiderling silks
  • Silk Swatch = Combine 2 spider silks
  • Studded Belt = Medium Quality Pelt (per size needed) + 2 studs + Pattern
  • Studded Boots = Medium Quality Pelt (per size needed) + 4 studs + Pattern
  • Studded Bracer = Medium Quality Pelt (per size needed) + 2 studs + Pattern
  • Studded Cap = Medium Quality Pelt (per size needed) + 3 studs + Pattern
  • Studded Cloak = Medium Quality Pelt (per size needed) + 4 studs + Pattern
  • Studded Gloves = Medium Quality Pelt (per size needed) + 4 studs + Pattern
  • Studded Gorget = Medium Quality Pelt (per size needed) + 2 studs + Pattern
  • Studded Mask = Medium Quality Pelt (per size needed) + 1 stud + Pattern
  • Studded Pants = Medium Quality Pelt (per size needed) + 4 studs + Pattern
  • Studded Shoulderpad = Medium Quality Pelt (per size needed) + 3 studs + Pattern
  • Studded Sleeves = Medium Quality Pelt (per size needed) + 3 studs + Pattern
  • Studded Tunic = Medium Quality Pelt (per size needed) + 5 studs + Pattern  [may be broken]
  • Tailor Made Whip = 1 Medium Quality Wolf Skin + 4 Heady Kiolas + 1 Whip Pattern
  • Tailored Quiver:  High Quality Cat Pelt + Quiver Pattern.
  • Wu's Fighting Gauntlets = 4 Heady Kiolas + 1 Greater Lightstone + 1 Vial of Visous Mana + 1 Silk Swatch + 1 Glove Pattern
