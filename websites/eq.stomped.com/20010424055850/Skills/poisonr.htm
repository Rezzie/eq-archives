    Poison Recipes

    If you wish to contribute recipes for creating poisons, please e-mail me

    Note.  This is a new skill, so these recipes are very preliminary.  Many are conjecture and not yet tested.  Also, nobody knows the effects of the various poisons yet. Use them at your own risk.

    Dyes

    • Dye = Dye Extract + 1-3 resins (to alter shade)
    • Dye Extract (brown) = Flask of water + Russet Oxide
    • Dye Extract (orange) = Flask of water + Jack-o-lantern Fungus
    • Dye Extract (red) = Flask of water + Iron Oxide
    • Dye Extract (black) = Flask of water + Charcoal
    • Evergreen Extract (green) = Flask of water + Evergreen Leafs
    • Permafrost Extract (blue) = Flask of water + Permafrost Crystals
    • Sarcoscypha Extract (red) = Flask of water + Sarcoscypha Fungus
    Poisons

    • Aching Blood = Lined Vial + Constrict Suspension + 2 Lacteras
    • Asp Poison  =  Poison Vial + Suspension + 2 Asp Poison Sacs
    • Atrophic Sap = Lined Vial + Constrict Suspension + 2 Alocasia Roots
    • Basalisk Poison = Lined Poison Vial + Suspension + 2 Basalisk Eyestalks
    • Brain Freeze = Sealed Vial + Constrict Suspension + 2 Gelsemium Roots
    • Crystal Eritus = Lined Poison Vial + Suspension + 2 Crystalized marrows
    • Cyclan Butil = Lined Poison Vial + Suspension + 2 Cyclamine Corms + Snake Venom Sac
    • Delusional Swiftness = Vial + Suspension + Giant Wasp Poison Sac + 2 Vampire Bat Salivas
    • Eyeburn Solution = Lined Vial + Constrict Suspension + 2 Coyotetails + Muddite Mud
    • Festering Nettle = Lined Poison Vial + Suspension + 2 Ashroots
    • Kinetic Suppressant = Lined Vial + Constrict Suspension + 2 Delphiniums + Basalisk eye Stalk
    • Lethergic Bliss =  Lined Vial + Constrict Suspension + 2 Whore's Banes
    • Lixt Wing Dust = Lined Poison Vial + Suspension + 2 Embalming Dusts
    • Maddening Sap =  Lined Vial + Suspension + 2 Succulent Saps
    • Putrid Desil = Lined Poison Vial + Suspension + 2 Putrid Biles
    • Rancid Wolfkiller =  Lined Vial + Suspension + 2 Dogbanes + Chunk of Meat.
    • Regalis Poison  = Poison Vial + Suspension + Urticating Hairs + Froglok Poison Gland
    • Retinal Deactivator =  Lined Vial, Suspension + Coyotetail + Asp Poison Sac.
    • Solvent Gangrene = Vial + Suspension + 2 Grave Molds
    • Snake Poison = Poison Vial + Suspension + 2 Snake Poison Sacs
    • Spider Poison = Poison Vial + Suspension + 2 Spider Poison Sacs
    • Spine Break =  Lined Vial + Constrict Suspension + Thorny Ergot + Delphinium.
