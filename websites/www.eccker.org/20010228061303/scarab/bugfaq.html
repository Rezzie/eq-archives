Message Board Help

REQUIRED FIELDS

The only two fields that you MUST fill out are the Name and Subject fields. If you do not enter any text in the Message field, a red "NT" (meaning "no text") will automatically be added to the post's listing.

HTML

HTML codes can only be used in the Message field. Any HTML codes used in the Name or Subject field will be stripped out by the message board system.

Here are some sample HTML codes that you can use in the Message field:

CODE
RESULT
<center>Centered</center>
Centered
<b>Bold</b> Bold
<i>Italic</i> Italic
<font size="+1">BIG</font> BIG
<font size="-1">small</font> small
<blink>Blinking</blink> Blinking
<font color="#FF0000">Red</font> Red
<img src="http://www.yourserver.com/loungelizard.gif">
<a href="http://www.eccker.org" target=_blank>My Homepage</a> My Homepage

Using the Optional Link URL Field

If you do not want to use HTML tags to add a link inside the Message field, you can do so by using the Optional Link URL field. There are two rules for using this method to add a link to your message; you must enter a full path for the Optional Link URL field, and you must enter something in the Link Title field. Here are some examples for entries in the Optional Link URL field:

  • Valid Entries
    • http://www.yourserver.com
    • http://www.yourserver.com/
    • http://www.yourserver.com/dir1/subdir1/index.html
  • Invalid Entries
    • mypage.htm
    • www.yourserver.com
    • www.yourserver.com/mypage.htm

Using the Image Link Field

Like the Optional Link URL field, the Image Link field can be used in lieu of adding HTML image tags in the message body. The only rule for using this field is that you must enter the full path to the image file. Here are some examples for the Image Link field:

  • Valid Entries
    • http://www.yourserver.com/screenshot1.jpg
    • http://www.yourserver.com/dir1/subdir1/mypic.gif
  • Invalid Entries
    • myimage.gif
    • www.yourserver.com/myimage.gif

Copying an inserted picture to your hard drive.

To save an image that has already been posted to the board, right-click on the image and select "save image as" from the resulting menu. Note that the precise wording of this menu option will change from one browser type to another.

AUTOFILL feature.

The Autofill option will allow you to save posting information and have it automatically entered the next time you load a POST MESSAGE form. The SAVE DATA button stores the currently entered information for the Name and Email fields. You need not fill out both fields in order to save the data. Any saved information will automatically be filled-in the next time you access the POST form. To clear this information from your computer, click the CLEAR DATA button.

This feature is also available on the Reply post forms, but ONLY on those posts made after April 17, 1999 at 10:30pm Eastern (when the feature was added). Also note that you will have to reset your information if you change browsers, go to a different computer, or go to another guild's message board (even if it supports Autofill, the data is stored in separate places for each guild).
