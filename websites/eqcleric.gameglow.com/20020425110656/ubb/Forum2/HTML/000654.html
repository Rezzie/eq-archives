Most 60 spells seem to drop randomly from mobs like IV, Rhags, Glyped/Exiled, AL, Creator, THO, Grieg, Burrower Parasite etc. This is from personal experience.

Some classes has a single spell that only seem to drop from one mob.

- Cleric Marzin Mark: HP (not personal exp)
- Magi pet heal: Grieg
- Wiz nuke: THO
- Nec dot: Rhag1 (unsure about this - but I only recall seeing it from one of the Rhags)

I dont know if there are more spells like these. Also note that wizards nuke Garrison hasnt been reported to drop at all - and similar for a Magician AE nuke.

I wont spoil the encounters in the detail you ask though - except to say that if you havent done any of them yet you should try Rhag1 first.
