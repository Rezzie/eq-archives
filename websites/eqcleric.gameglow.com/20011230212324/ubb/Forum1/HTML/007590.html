What your looking for Dalur is freezes, lockups and generally really crummy performance.

If thats the case, install the ViA drivers for Win98/ME (assuming thats the OS your running), preferably the ones that shipped with your mainboard.

If the bench shows you have sub-standard performance compared to similar systems, then you know you have a system problem as opposed to a EQ problem.

Trust me, if you got a problem in your setup, you will see it in the results. I ran mine and got a score of around 500, after doing massive setting changes just to get it to finish running. Installed the ViA drivers, and I jumped to 5800+ score and it ran on the default test settings.

I was also getting lockups trying to run any 3D program, constant crashing.

If you get solid marks, you can generaly be safe in saying your system (vid drivers, directx, board drivers, etc) is in working order.

i forgot had a similar problem long time ago, might want to check that too :-)

from nVidea FAQ:
When I try to play a game, my monitor goes to sleep/turns black but I can still hear the game running. How can I fix this?
This generally means that your graphics card is running the game at a refresh rate higher than what your monitor supports..
...or it's a Windows Error

for Win95/98 (copied and pasted from various sources, worked for my WinME too if i remember right, for Win2K/XP there are several programs to fix the refreshrate)
1. To manually set the monitor refresh rate, you must use the utilitiy Regedit. Click on the + to expand each folder in this order:
HKEY_LOCAL_MACHINE -> System -> CurrentControlSet -> Services -> Class -> Display -> 0000 -> DEFAULT
2. Right Click on the right pane of the Registry Editor, and click on "String Value".
3. Enter the name "RefreshRate"
4. Double-click on RefreshRate and enter the value -1 for automatic rate settings, or a number specific to the refresh rate. (i.e. 75 or so, refer to your video card or monitor documentation for information on refresh rates.) Click OK and then restart Windows.

All times are ET (US)
