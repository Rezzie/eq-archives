Popup adds are not that big of a deal. Just go find a program that shuts them down as soon as they open. I use one called POW. It was created by AnalogX and is a boon for killing popups. Only draw back is that you have to allow the popup to come up one time, then add it to the POW list so it doesn't come up again.

System overhead using POW is about 1% system resources used (my system: 550mHz, 128MB RAM).

Just FYI.
