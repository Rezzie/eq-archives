Regarding Gate, I might consider loading it if I was fighting in a place in which my gating to the zone might help people survive. For example, a few levels back I can recall loading it in Karnor when I was bound at the zone. Group had to make a run for it, I gate to the front and heal them as they run through. Besides that I don't load it in combat.

Of course, our roles change as group makeup changes, so I would never be so ignorant as to say Complete Heal is worthless, just in my opinion it's situational and rarely used in the groups I regularly hunt in.
