If you use version 1.1 of my UI(brown), download this file (http://home.attbi.com/~mbeiser/Files/1-9-03v1.1.zip).

If you use version 1.2 of my UI(blue), download this file (http://home.attbi.com/~mbeiser/Files/1-9-03v1.2.zip).

Extract it to the directory you have the UI in.

I had a few people express interest in my target window mod, if you want to try it out, download this file (http://home.attbi.com/~mbeiser/Files/Calealtarget.zip), and extract it to the directory your UI is in. All the annimations for the custom gauge are described in the xml file for the target window, so this will work in any UI. It will handle even the longest of NPC names in Vex Thall.

http://home.attbi.com/~mbeiser/Images/Calealtarget.jpg

Note that the target lifebar and casting gauge are divided into 10 sections. I find it somewhat usefull in determining Complete Heal timing. I like shaving things a little close. :D
