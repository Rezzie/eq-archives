never keep gate memmed as if i have need to use it, i can cast DA/DB sit down and mem it quick, the time it takes to cycle is about when DA/DB wear off, then i start frantically hittin the gate gem =)

For DA and DB i've come up with a hotkey that works fairly well for me.
DA/DB
/g GOING INVULNERABLE! NO HEALS TO OR FROM ME!!
/cast 2
/cast 1
/g I'M INVULNERABLE!! NO HEALS TO OR FROM ME!!

putting DB first in the list puts the one that uses more mana as the first one you use IF you have the mana, if you don't, you'll simply cast DA instead, saving you the precious seconds it would take for you to fumble to cast DA if you get the OOM message for DB. Also, if you've used DB already, this hotkey will automatically cast DA instead. put this hotkey in one spot and keep it there, so as time passes and you use it more, hitting that number will beocme second nature and save even more time in tense situations.
