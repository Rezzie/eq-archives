The command to sit is simply /sit

One limitation of the macros is that there is no delay or waiting for completion. They act as if you did all the commands yourself in rapid succession.

Thus:
/sit off
/cast 3

works properly, whereas

/sit off
/cast 3
/sit

would do the same thing, because the /sit command would hit during the casting of the spell, which, of course, is ignored because you are already casting.
