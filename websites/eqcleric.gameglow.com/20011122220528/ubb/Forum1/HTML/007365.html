How bout this change on test?:
Changes on Test Server
10/30/01, 9:24 PST | Kwill

Necromancer Steps of the Cazic Thule server has passed on this message from the Test Server patch news:

Also, implemented on the Test Server is the ability for the casters of group buffs to cast their group buffs on other groups. We had to implement a slash command to do this. The command is /targetgroupbuff. This command acts as a toggle and while it is *ON*, all groups buffs will resolve around the targets group. While the toggle is *OFF*, all group spells will resolve around the caster.

Interesting developments...we will see if these changes actually go live in the future.
