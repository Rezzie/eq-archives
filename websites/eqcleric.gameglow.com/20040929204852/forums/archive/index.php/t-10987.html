I think you're using a old version of the UI. The reason your group window doesn't show up is because the spell bar (where you mem your spells) is also used to show the group information. And the box that normally has the group information is being used to show the bags. Since a warrior doesn't get the option to have the spell bar, you won't be able to see group information.

The newer version of the UI fixed this problem by putting the spell bar back in a standalone window and putting the group information back in the group window (the one with the bags). So downloading the newest version of the UI should clear up your problem =)

Very basic tip:

Whenever I run into problems with some custom UI not letting me change something (like window sizing, position, etc) I just type:

/loadskin default 1

Make whatever changes I want, and then reload my custom UI by typing:

/loadskin custom_UI_ name 1

And you're back in business. Your changes will write themselves to your current UI_charactername_server.ini file and you're gtg.

(Another trick is to then copy that modified UI_charname_sever.ini file to a safe location so you can use it on whatever other characters that might want to use the UI. Obviously, you'd be renaming it to their character name.)

Don't forget to use the 1 or you're going to tear up your ini file and be forced to quit out, replace the ini and restart EQ. (major PITA)
