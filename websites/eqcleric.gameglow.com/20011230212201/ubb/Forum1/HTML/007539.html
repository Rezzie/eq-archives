I decided to run a few tests to figure this out.

A
118 wis + 0 mana
Cast 3 HB + 3 AoP - couldnt cast Root
total between 1590 and 1620 mana

B
200 wis + 560 mana
Cast 7 HB + 1 AoP + 1 MR - couldnt cast Root
total between 3135 and 3165 mana

C
255 wis + 710 mana
Cast 8 HB + 1 AoP + 1 MR + 1 PR - couldnt cast Root
total between 3605 and 3635 mana

Alright - this obviosly doesnt take the effect of specialization into effect - but from others tests I seem to recall that being in the order or 2-3% for the non-main spec.

I now looked at the fomula:
2*Level + Wis_upto_200 * Level/5 + (Wis-200)*Lvl/10 and get the following predictions:

A: 120 + 118*12 = 1536 - with a 3.5% spec deduction thats within the seen result.

B 120 + 200*12 + 560 = 3080 - with a 2% spec deduction thats within the seen result.

C 120 + 200*12 + 55*6 + 710 = 3560 - with a 2% spec deduction thats within the seen result.

So, by no means have I proven anything - but I think I have shown that its likely that each wis above 200 give 6 mana at level 60 now. If it gave 0 or 1 pre-patch I do not know. I would obviously love to see tests done by others to comfirm or deny my theory.

I am pondering getting Spell Casting Mastery too. However, when doing so I would like to make some real tests as to its efficiency.

My method of testing was to be this:
- Make sure I has less mana than needed to cast DL/BoF (for main spec/other spec tests respectively).
- Standing without an FT item, MC skill or mana regen buff cast it the very tick it was possible.
- Have spells loaded that cost various ammounts for 30 to 70 mana and see which I could cast thus getting an approximate mana saving from spec.

Now, to verify that this is indeed a useable method I logged on my level 16 magician (= no spec yet). I then cast a 160mana spell the very tick it was possible. Outcome - I was able to cast a 10mana but not a 15mana spell immidiately afterwards.
