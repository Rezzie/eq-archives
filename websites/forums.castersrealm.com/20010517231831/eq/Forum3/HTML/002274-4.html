a_mmorpg's_corpse_001
unregistered 		posted January 30, 2001 01:34 PM              
We've changed this item [a cleric] before because we thought it was overpowered. Unfortunately, it is still overpowered. Let me explain why.
This item is a cleric. If you use this item, it will cast a Complete Heal on your target with a casting time of 10 seconds. It uses mana to produce this effect.

A group of three characters, one cleric, one enchanter and one warrior, can defeat any opponent, as long as that opponent can not kill the warrior in under 40-50 seconds.

Let me restate that.

One enchanter and one warrior, using this one magic item [a cleric] , can defeat ANY opponent or any number of opponents that are not capable of killing the warrior in under 40-50 seconds.

I hope that the impact that this game dynamic has on the game is obvious to everyone. If we allow this item to remain in the game as it is, we will have to change the game so that monsters are capable of killing a warrior in 40-50 seconds. 20-25 seconds if the group consists of two clerics, an enchanter and one warrior, and it gets worse from there...

Of course, if we make the world so hard that this item [a cleric] is not overwhelmingly powerful, we would have to, essentially, rebuild the game, rebalance the classes… No single item should be so powerful that the balance of the game revolves around it. Adjusting the game to fit this item is not a choice. We need to change the item to fit the game.

We have had a fair amount of internal debate over this item. It really comes down to two choices. Eliminate the item entirely, or change the way it works. We would rather change the way it works. After much debate and some conversations with players that have characters that use this item, we decided on the following change.

The item's spells will be shared among the other classes. Following the druid and shaman balancing, remedy and divine light will belong to them. Ressurection has already been allocated to two other classes.
The item itself [clerics] will be removed from the game and with them the Complete Healing spell.

In practice, what this will mean is that the cleric with an enchanter slowing and casting c2 will no longer be able to heal anyone he wants every 40-50 seconds (med time on CH with c2). But he will be "balanced" and will no longer destroy the game dynamic of EverQuest.

There is one concern that we have as to this implementation, and I'd like your feedback on it.
Who wants icecream?

Yes, we know this is a 'nerf'. And, frankly, we're willing to nerf this item to save the game as a whole. We are sorry about ever creating an item that is so easily unbalancing to the game, and we're sorry that we have to change it again. For now, I can't tell you when this change will take place. It won't be until after we've finished the work that has us on hold for updating the game. When I have a better timeframe, I'll certainly let you all know. I can tell you that following the removal of the item [a cleric] we will consider the cleric class to be "balanced" and will not be nerfing it any further.

Why is it that Verant is perpetually plagued with short sightedness?

To be honest, we never really thought anyone would play with this item [a cleric], especially in this way. I mean, what sort of person would pay good money to sit facing a wall and stand up and heal at random intervals so that other people could enjoy the game?

Honestly? I think we should have deleted the item the first time it became a problem. No single item should exist that makes us worry about it being unbalanced. Those items have the power to destroy the game.

I look forward to "balancing" more classes in the near future.
