Okay, Verant and a lot of Enchanters say we're not a pet class. I'll accept that for a moment. What are we, then? For this example, call us the one and only animation class. So let's take a look at the differences between the Pet Class and the Animation Class.

Pet class pets take commands. Animation class pets do not.
Pet class pets can dual wield at 29. So does the animation.
Pet class pets are are lower in level (relative to the player) as the player's level rises. The same holds for the animation class.
Pet class pets taunt, bash, kick, dodge, etc. So does the animation.
Pet class pets take 50% of the exp if they do over 50% of the damage. Animations do the same.
Pet class pets can be equipped with weapons, and those weapons will proc as they would for any player. Animations have the same ability.
Pet class pets die when you zone. So do animations.
Pet class pets do not take appreciable experience when you are grouped with others. Neither do animations.
Pet class pets with an LD master behave like NPC's. So do animations.
Pet class pets have an incomprehensible weapon wielding logic. So do animations.

I could go on, but you see where I'm going with this. Some differences, many, many similarities between the pet class and the animation class. Now, let's take a look at the changes (the ones I know about, anyway) that have affected the pet classes and the animation classes.

Pet class pets got nerfed with regard to weapon delay. Animations took the same nerf.
Pet class pets got nerfed with regard to weapon procs. Animations took the same nerf.

Hmm, pretty similar there, too. It looks to me like we're pretty fraking close to being a pet class. Explain to me again how it is we're not?

Personally, I would LOVE not to be a pet class. I would much prefer to be an animation class, if ONLY it meant that Verant would not nerf our animations like the do the pet class pets. How nice it would be to be left alone. Unfortunately, Verant screws with our animations just as they do the real pets, and this is why I get so bent when I hear people say we're not a pet class. The whole discussion is (obviously) largely a matter of semantics, but to use "we're not a pet class" as an explanation for every boneheaded move Verant makes w/r/t our animations is an indication that someone is not thinking very hard.

Wait, there is one major difference between the way Verant treats animations and pets: when pets have been nerfed, there has been some sort of compensation to the pet classes. When Animations get the same nerf, Enchanters get screwed and stay screwed.
