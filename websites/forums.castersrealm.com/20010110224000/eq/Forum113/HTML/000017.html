After 40 start looking into getting a one hander - most if not all the mobs out there will be casters/gaters and you're going to need to be able to bash (for all you Human/Dark Elf SKs). Ebon Mace, MoSS, Lamentation or Jade Mace, Noctivagant Blade(sp?) These are the best 3 sk weapons out there that can be bought (MoSS and NB are Sk only too).

Large races get Slam, so I'd go with Zealot's Incardine Sword (2hs).

You don't want something with a huge delay like the Throneblade. Sure it's got a pretty proc, but you're there to cause damage and you want to be able to cause it as fast as possible.

My advice - go with a low delay weapon. MoSS would be my choice. It does incredible damage.
