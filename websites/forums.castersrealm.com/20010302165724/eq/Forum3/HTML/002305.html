This is a very serious question, but like most questions, comes from the wrong angle so the answer will not get a chance to be right. Of course folks want a changing environment and having to learn new tactics is a way to grow our chars. Dynamism is only one piece, and ultimately it comes back to game balance. And asking if players want things altered in an MMORPG or if they should be fixed in concrete after beta, misses the mark that sparked this which seems to come down to: Wow, clerics post 50 seem to be playing a whole different game than what we learned over the last year-and-a-half; what gives?

VI's asserted that players are clever and manage to figure out an optimal strategy. This causes VI to want to update the environment to keep things interesting. This change is a part of that, but since there are really three different envrironments in EQ, tuning is tricky (Olde Wolrd, Kunark, Velious represent almost three different games from a strategic view. Arguably planes can be included but those were conceived to behave very different from conception).
- - - - - -
When we set up something we think is tricky, one or some of the 300,000 + players finds it easy to do. You guys are always able to come up with a way to make things work. You're tricky people. And once the 'trick' is known, soon enough everyone is doing it (or trying to do it). - Alan http://eq.castersrealm.com/viewarticle.asp?Article=3242
- - - - - -
This is a fundamental flaw in game design in the realm of game balance. In VI's defense, this is incredibly hard to achieve normally and is compounded by the number of choices available (race/class/spell/armour/weapon combinatorial). However, for game balance to have any meaning, there can be no 'trick' to the solution, i.e., no prefered strategy or you see this behavior where one person discovers a way around a 'tricky' area and passes that knowledge on. There need to be multiple successful strategies where there is some downside to picking different ones as well as differing beneficial results (see for instance the discussion where enchanters and clerics are considered essential or you can't venture into a high level dungeon...a way to alter the dynamic is to make enchanter less valuable ala velious mobs or clerics less reliable via quicker death.) The downside to this trick-type of strategy being balanced by altered class conceptions is that we players need to learn the 'trick' in these situations: As mentioned for clerics, multiple clerics all chaining CH.

Second point, dynamic equals unpredictable: Bad idea. Randomly and arbitrarily changing MObs' strategies only results in having players try random strategies until the 'trick' is rediscovered. (See for instance Myst where some puzzles may not be able to be solved by evidence so a brute force approach to flipping all the levers can be taken).

If MObs behavior adapted over time, even if in quantum steps at patch time, that is one way to stimulate creative sulotions and a dynamic environment. Altering the pathing because players found an exploit is a way to keep the game dynamic (MObs learning about the unguarded flank over time is GOOD), but doing it simply to stop/block players from easily entereing an area is bad (sending in a rogue to scout then bringing in the rest of the group(s) behind the patrol is too successful, we need a static MOb to defeat the Rogue because that is too easy a strategy for players is BAD).

Hope this is coherent, but I'm in hurry and RL intrudes. Once over for consistency is all it gets, edits be damned! :-)
