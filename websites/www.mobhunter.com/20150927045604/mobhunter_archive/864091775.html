There is, at this time, a bug in the messages that the server is sending to the client when a character's faction is affected. The server is sending the client the wrong faction name. Also, if a faction change would affect three factions, it's often displaying all three faction names as the same.
Faction updates on characters are working correctly, this is only affecting the displayed faction name that you see.

Because this isn't causing any issues with character faction, we don't plan to patch to fix this until our regularly scheduled patch. A patch to fix this would require taking all servers down, and especially considering yesterday, we'd rather leave them up and let you all play. I just wanted to reassure you that it's just a goofy message error, not a problem with the actual faction of characters.
