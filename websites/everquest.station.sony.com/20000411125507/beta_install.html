EverQuest Installation

Before installing EverQuest, make sure that your PC specifications meet or exceed the requirements listed above.

Insert the EverQuest game CD into your CD-ROM drive. If your CD-ROM drive has the AutoPlay function enabled, the installation will begin automatically. If AutoPlay is not enabled, Open the "My Computer" icon on your desk top, double-click on the CD-ROM icon, then double-left-click on "setup.exe". The installation process will then commence.

  1. Choose an installation size. The "full" installation will install all of the required game files, plus the intro movie, whereas the "typical" installation will only install the required game files. To view the intro movie with a Typical install, the EverQuest CD will need to be inserted into the CD-ROM drive. (Note: The movie is not available on the Phase IV BETA CD).

    2. Typical requires 500MB* of free disk space.

    Full requires 600MB* of free disk space.

  2. Select a directory in which you would like to install EverQuest to. The default directory is C:/Program Files/EverQuest. If you would like to install the files into a different directory, change the path and proceed. If the directory does not exist, select a directory name(Example: C:/Program Files/Eqrules), then proceed.
  3. Installing the DirectX 6 drivers.

    4. During the installation of EverQuest, and option to install DirectX 6 to your hard drive will be given. If you already have DirectX 6 installed, select "No" when prompted. If you are unsure that DirectX 6 is installed on your machine, you may check by running the "DXSETUP.exe", which is located in the Program Files/DirectX directory. To proceed with the installation of the DirectX 6 drivers, select "Yes" to proceed, then follow the on-screen instructions.

  4. After completing the installation, you will be prompted to restart your computer, select "Yes" and allow your system to reboot.
