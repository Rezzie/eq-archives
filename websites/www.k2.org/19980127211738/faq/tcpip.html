TCP/IP Setup
  1. Go to your Control Panels and open TCP/IP.
  2. You will see the following screen:

  3. Choose to Connect via MacPPP or FreePPP depending on which you are using.
  4. Configure should be set to Using PPP server.
  5. For the Name server addresses, please enter 207.229.143.1 and 207.229.143.2.
  6. Enter enteract.com under Search domains.
  7. Close TCP/IP.
