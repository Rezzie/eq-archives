Before You Begin

  1. It is a good idea to make a separate directory for Trumpet. To do so, simply go to your File Manager.
  2. Click on File and then Create Directory. This will let you create a directory called trumpet.
  3. Move Trumpet into this directory. This can easily be done by dragging the file on the right side of your File Manager over to the trumpet directory you already created on the left side.
  4. If you have a zipped copy of Trumpet, please unzip it. If you have a self-extracting copy of the program (it will have an .exe extension), please simply double click the file.

Creating an icon for Trumpet Winsock
  1. In your Program Manager, go to File and then New.
  2. Choose Program Group.
  3. Type in whatever you would like to call this group. Trumpet Winsock or Internet is fine.
  4. Click OK.
  5. Go to File and then New again.
  6. This time, choose Program Item.
  7. In the Description text box, type whatever you would like to call this application. Trumpet Dialer or EnterAct would be fine.
  8. In the Command Line text box, type in the path of where your Trumpet is located. It would likely be c:\trumpet\tcpman.exe, although depending on where you uncompressed Trumpet Winsock, this may not be the correct path. Tcpman.exe is the name of the executable program that runs your dialer.
  9. Cilck OK.

Setting Up Trumpet
  1. Double click your newly created Trumpet icon.
  2. You will see the main screen for Trumpet. Go to File and then Setup.

  3. Fill in the following values:
    • IP Address: 0.0.0.0. Even if you have a static IP address, the server will assign that IP address to you whether you enter it here or not.
    • Netmask: 255.255.255.0 (This may be grayed out. If so, you will need to de-select Internal PPP to change it. When finished, select Internal PPP again.)
    • Name Server: 206.54.253.10
    • Domain Suffix: enteract.com
    • Default Gateway: 0.0.0.0
    • Time Server: Leave this field blank
    • Packet Vector: 00
    • MTU: 1500 for PPP or 1006 for SLIP
    • TCP RWIN: 2048
    • TCP MSS: 1024
    • TCP RTO MAX: 60
    • Demand load timeout [secs]: 500
    • Check either Internal SLIP or Internal PPP depending on which protocol you will be using. Keep in mind that if you wish to use SLIP you will have to enter an S in front of your user name when you log in. (i.e. a username of fred becomes Sfred.) PPP does not require an extra character in front of the user name.
    • The SLIP port should be the same as the com port that your modem uses. Normally this will be 1 or 2.
    • Check Hardware handshaking. If you will be using PPP, you do not have to worry about Van Jacobson CSLIP compression. If you wish to make use of this option, type a C in front of your login instead of an S. (i.e. the login or fred becomes Cfred.)
    • Select None for online status detection.
    • Click OK.
    • You will need to exit Trumpet and go back in for these settings to work

Automatic Login
  1. Go to your Dialer pull-down menu from your main trumpet screen.
  2. Select Options.

  3. Select Use standard SLIP settings for parity & word size.
  4. Select automatic login and logout on demand or automatic login on start up only.

Note: If you choose to login automatically on start up only, every time you open up trumpet winsock, your modem will dial automatically. Otherwise, you will have to go to the Dialer pull-down menu and select Login.

Trumpet comes with a file called login.cmd. This is the file that automatically logs you into Enteract. When you first open Trumpet after you have filled in your settings, you will be asked for your login (remember to use Slogin for a SLIP login or Clogin for a CSLIP login) and your password as well as the phone number you will be using. If you find that this login.cmd is not functioning properly (i.e. you are not able to make a successful connection with us even though all of the previous settings are correct) you will need to dial in manually in order to obtain the login.cmd file that we have located at our ftp site in /software/windows. Replace the original file with this one. If you are unsure of how to obtain access to our ftp site, give us a call and we can help you.

After you have given Trumpet the information it needs to log you in automatically, you will never have to supply that information again.

You will know you are connected when Trumpet returns the statement: PPP Enabled or SLIP Enabled. At this point you can simply minimize your window and open up any internet application you wish.

Manual Login

  1. Click the Dialer pull-down menu located in your main trumpet screen.
  2. Select Options.

  3. Select Use standard SLIP settings for parity & word size.
  4. Select No Automatic Login.
  5. Click OK.
  6. From the Dialer pull-down menu, click Manual Login.
  7. You will be left with a flashing cursor at the top left corner of your screen. Type ATDT and then the local phone number you will be dialing to connect to us. You need not put a space between the ATDT command and the phone number.
  8. When your modem finishes dialing, you will be asked to enter in your login and password. (Remember to use Slogin for a SLIP connection or Clogin for a CSLIP connection.) Please enter them.

When you are connected, you will see a bunch of garbage characters stream across your screen. Hit the Esc key in the upper left hand corner of your keyboard to finish the connection and stop the stream of characters.

That should be it. At this point you can minimize your Trumpet window and open up any Internet applications you wish!
