MY HOMEPAGE
 You should obviously be able to recognize this page, unless I haven't updated the graphic along with the page. This was my first web site, but it definetly didn't look this great then. It was very plain with cheap graphics and crappy content. It has gone through several changes over time, each one better than the previous one.

This one was typed up in TextPad, a really cool text editor. The graphics were drawn in Paint Shop Pro, including the screen shots of all the web pages. The windows that contain all the information were made with layers, a new addition to the HTML library, introduced by Netscape with their Communicator. The border is made by creating a table with one cell, the full size of the layer. The windows-like bar at the top is just and image, with the "X" stolen off a screen shot of a window. If you have any questions about this page, layers, etc., email me.
