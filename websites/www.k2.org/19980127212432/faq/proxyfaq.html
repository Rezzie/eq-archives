The EnterAct Proxy Server

What is a Proxy Server?

The EnterAct proxy server is a service we offer our users that allows them a greater degree of privacy when browsing web pages. Normally, when you surf the Web, certain information is logged by the webservers that you visit. Here is an example from the access logs kept by EnterAct's own web server:

ppp-r37.aracnet.com - - [31/Jan/1997:21:52:08 -0600] "GET /~joeblow HTTP/1.0" 302

This shows a request from a specific address, in this case a dialup user from aracnet.com, for the web page of user joeblow. Our proxy server allows you to browse while only revealing information about the proxy server. A comparable proxy server log entry would merely reveal that proxy.enteract.com visited the given site, but nothing else. You remain completely anonymous. However, keep in mind that, since web pages and requests have to go through the proxy server first, this will slow down your Web surfing to a small degree.

This is not to be confused with a "caching proxy," which is a server that stores copies of commonly-requested web pages for faster retrieval.

How do I use it?

All of the commonly used browsers give you the option of browsing via a proxy.

  • Netscape Communicator: Under the Edit menu, and select Preferences. Expand the Advanced category and select Proxies.

    Select Manual proxy configuration and click on View.

    Enter proxy.enteract.com in the HTTP: field and enter 8080 as the port. Click Ok until you are back to the Netscape browser window.

     

  • Netscape Navigator: Under the Options menu, select Network preferences... Bring the Proxies tab to the front.

    Select Manual Proxy Configuration and then click on View.

    Enter proxy.enteract.com in the HTTP: field and enter 8080 for the port. Click Ok until you are back to the Netscape browser window.

     

  • Internet Explorer 4.0: Under the View menu, select Internet Options. At this window, bring the Connection tab to the front.

    Check the Access the Internet using a proxy server option. Enter proxy.enteract.com in the Address field and 8080 for the port. Click Ok.
