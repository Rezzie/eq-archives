Windows 95 Dialup Installation

You too can be on-line in 10 minutes or less. Just follow these easy instructions and you will be successfully connected to the Internet!

  • A) Install Dial-up Adapter
  • B) Setup the Domain Name Server Information in the TCP/IP properties
  • C) Setup Dial-Up Networking
  • D) Time to connect!

A) Install Dial-up Adapter and TCP/IP
  1. Click your Start button.
  2. Click on Settings.
  3. Go to the Control Panel.
  4. Double click on the Network icon. You will see the following screen, although you may not have all of the items listed in the installed components window below. If you see Dial-Up Adapter and TCP/IP (or TCP/IP->Dial-Up Adapter) listed you can skip to step B. If not, please continue your journey through step A.

  5. Click the Add... button.
  6. Highlight Protocol by single clicking on it.
  7. Click the Add... button.
  8. Single click on Microsoft.
  9. Single click TCP/IP and click the OK button. You have now just installed TCP/IP. You will need to take similar steps to install the Dial-up Adapter.
  10. Highlight Adapter by single clicking on it.
  11. Click the Add... button
  12. Scroll down for Microsoft and single click on it.
  13. Single click Dial-Up Adapter and click the OK button.

When you have finished installing each of these two components, you will see them in the installed components window from your network dialog box. Win95 may install additional components on its own.

Note:If you completed step A), Win95 may ask for the installation disks or CD-ROM so that it can install certain files. At the end of step B), you will have to restart your machine. Before it restarts, please make sure that these disks are ava ilable.

B) Setup the Domain Name Server Information in the TCP/IP properties.

  1. From the network dialog box (pictured perviously), highlight TCP/IP by single clicking on it.
  2. Click Properties...
  3. You need not change anything in the IP Address tab. Leave the radio button clicked on "Obtain an IP Address Automatically". Even if you have a static IP address, the server will assign that IP address to you whether you enter it here or not.
  4. Click the DNS Configuration tab. You will see the following screen:

  5. Click the Enable DNS radio button.
  6. Enter in your username in the Host: box. Make sure you enter it in lower case.
  7. Enter enteract.com where it asks for Domain. Again, this should be entered in lower case.
  8. Enter 207.229.143.1 in the top box of the DNS Server Search Order section and click the Add... button.
  9. Now enter the secondary, 207.229.143.2, in the same box and click the Add... button.
  10. Type enteract.com in the domain suffix search order box and click the Add... button.
  11. Type com in the domain suffix search order box and click the Add... button.
  12. Click OK.
  13. You will be asked to restart your machine. Please do.

Note:If you had to previously install network components in step A), you may be asked to fill in identification information for your computer. This information can be entered in the Network dialog box when you click the Identification tab.

C) Setup Dial-Up Networking

  1. Double click on your My Computer icon.
  2. Double Click Dialup Networking.
  3. If you don't see Dialup Networking, please continue to step 4. If you do, skip down to step 10.
  4. Double click on your Control Panel icon.
  5. Double click on the Add/Remove Programs icon.
  6. Click on the Windows Setup tab.
  7. Double click on Communications.
  8. Make sure that Dial-Up Networking has a check next to it.
  9. Win 95 will need to access files from your Win95 CD or floppy disks. Once it has copied the necessary files, you can go back and try step 2 again.
  10. If you have a connection icon for Enteract already, skip down to step 15, otherwise, please continue with step 11.
  11. If you currently do not have a connection icon for Enteract, double click on the Make New Connection icon.
  12. Name your connection (EnterAct would be a perfect name).
  13. Enter the phone number that is closest to you.
  14. Click OK and the new Enteract icon should appear.
  15. Right click this icon and go to Properties. The following screen will appear:

  16. Fill out the area code and phone number for the area located nearest to you.
  17. Choose the modem you will be using. This will probably already be selected for you.
  18. Now, click the Server Type... button from your connection dialog box. You will see the following screen:

  19. In the Type of Dialup Server pull-down menu, make sure PPP is selected.
  20. Under Advanced options: select Enable Software Compression only.
  21. Under Allowed network protocols: select TCP/IP.
  22. Click the TCP/IP settings button. You will see the following screen:

  23. Make sure that Server Assigned IP address is selected. Even if you have a static IP address, the server will assign that IP address to you whether you enter it here or not.
  24. Click Specify Name Server Addresses.
  25. Enter 207.229.143.1 as the primary and, 207.229.143.2 as the secondary.
  26. Select Use IP Header Compression and Use Default Gateway.
  27. Click OK until you are back at your Enteract connection icon.

D) Time to Connect!
  1. Double click on your Enteract connection icon.
  2. Fill in your login and password in the dialog box that appears on your screen. Optionally, you can choose to save the password. This will eliminate the necessity to type in your password in future.
  3. Click the Connect button.

That should be it. When your modem finishes connecting, you can minimize your connection dialog box and open up any internet applications you wish! If you have any trouble at all, please do not hesitate to call us at (773)248-8511.
