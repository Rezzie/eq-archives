How do I put CGI scripts on my web page?


Assuming you already have a completed CGI program you wish to install in your account, here's how to do it:
  1. Make sure the file is executable by everyone:
    chmod a+x mycgiprogram

  2. Make sure the file ends with ".pl" or ".cgi" -- this is how the server knows that the file is a CGI program to be executed and not displayed:
    mv mycgiprogram mycgiprogram.cgi

  3. Make sure the file appears in your public_html directory. This is really a given. Note that there is no requirement for a cgi-bin directory. You may choose to create one, but the server will still only recognize files as CGI programs by their file extension (.pl or .cgi) and not their presence in a cgi-bin directory.
    mv mycgiprogram.cgi ~/public_html

For those venturing to write their own CGI programs, see this excellent tutorial.

We have the Perl CGI library module CGI.pm installed at EnterAct. If you are an avid Perl CGI coder, you ought to look into this package.
