Setting up your professional site listing Setting up your professional site listing

Want to add your CompleteAct site to our listing?

Our index system will allow you to customize your entry on the "CompleteAct Accounts" page by specifying a description of your site, as well as determining the way you'd like your name to appear in the list, and the desired page to jump to in your site. Additionally, you can set up a small "icon" image that will be shown along with your name and site information. If you'd rather your site didn't show up in our index of pages, simply don't follow these directions; by default, you won't be listed, by default. If you've added an entry and wish to remove it, remove the ".info" file you had created.

Information about your site is stored in a file that you create in your 'vhost_yourdomain.com/http-docs' directory. The file should be named ".info" . Most of the time, files beginning with the "." character don't show up in directory listings on Unix systems, so don't be alarmed if you create the file and can't find it. It's probably there. The file needs to be flagged readable... you can accomplish this by typing at the Unix command prompt:

chmod a+r .info

The info file is formatted much like an HTML document. Each feature is configured by a "tag", which is specified between angle brackets ("<" and ">"). For example, the description of your web site should be indicated with the tag; the description should be terminated with a tag. Everything in between those two tags is the description of your home page, and will appear in the home page index.

Note: please do not use HTML within the description; making mistakes in the HTML can screw up the entire page.

Here's a sample entry:


EnterAct
EnterAct, an Innovative Internet Service Provider
webmaster@www.enteract.com
The .info file which created this entry is: 
<icon>SBEA2.gif</icon>
<name>EnterAct</name>
<description>EnterAct, an Innovative Internet Service Provider</description>
There are 5 possible tags, none of which are necessary. A default entry will be generated if a .info file merely exists, and is completely empty. These tags override certain default values.
<icon></icon>
This tag selects an icon to display. Refer to the icon as a URL relative to your homepage (i.e., if you want to display http://www.yourdomain.com/GIF/icon.gif, enter GIF/icon.gif in this tag. If this tag is not present, no icon is displayed. In addition, your icon must be 100X100 pixels or less. If it is larger, it will not display.

<name></name>
This tag provides the title for your page. If this tag is not present, the title is your sitename, e.g., www.yourdomain.com.

<welcome></welcome>
This tag is usually not necessary. It makes it possible to jump to a page other than the default home page. For example, if you would rather people jump to http://www.yourdomain.com/startherefirst.html, you would enter startherefirst.html in this field. The default is for people to jump to http://www.yourdomain.com.

<description></description>
This tag provides a description of your site. If this tag is not present, no description is included.

<contact></contact>
This tag provides an email address point of contact. The default value is webmaster@www.yourdomain.com.
