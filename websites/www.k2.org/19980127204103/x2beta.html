Since they are still in testing, these numbers are subject to change and we cannot guarantee their reliability.

We ask that any users that try out the service, and let us know how it appears to be working -- or not working. We are especially interested in any trouble reports. For these purposes we've set up a special mailbox x2@enteract.com, any general questions about the technology or future plans should still be directed to the usual support@enteract.com.

Preventing Disconnects with X2
So far in our testing, a few customers have reported being disconnected 3 to 5 minutes into an otherwise stable 50Kbps or higher connection.If you are experiencing this problem, try increasing the value of the S10 register (found in the init string for your modem). It should be set to at least 14 and maybe as high as 20.
Also make sure that you are running the latest code. x2 is still a protocol in the process of development. To make sure that you have the latest code, go to USR's web site and get their UPDATE WIZARD; this will inform you as to what version you have and allow you to upgrade if necessary.
