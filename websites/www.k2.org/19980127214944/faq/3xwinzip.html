WinZip

Installing WinZip
  1. Download WinZip from either http://www.tucows.com or http://www.winzip.com.
  2. Copy the file you just downloaded to a unique directory. Call it winzip if you like.
  3. Run the program by double clicking the file you just saved to the winzip directory or folder.
  4. This will take you through WinZip's setup wizard.
  5. The setup is pretty self-explanatory, but at one point you will be asked which directory you want to install WinZip into.
  6. Click OK to keep the default.
  7. You will also be asked if you want to do an express setup or a custom setup. Choose express.
  8. This will finish the installation and create all of your icons for you.
  9. Open Winzip by double clicking the icon that was just created.

Unzipping Files
  1. To unzip a file, click Open.
  2. Choose the file you wish to unzip and double click it.

  3. If you know the directory you want to put these files in, click the Extract icon and choose the appropriate directory. WinZip will take these files and put them in the directory of your choosing.
  4. If you wish to have WinZip not only unzip the files but also install the program, click Check Out.
  5. You can change the folder if you don't want the program to be installed in the default folder.
  6. When you've chosen the folder, click OK.
  7. If there is an install.exe or setup.exe to be run in the program you are unzipping, WinZip will run it.
  8. When you finish, and wish to close out WinZip, you will be asked if you want WinZip to delete the folder and icons it just created. Unless you want these things to be deleted, click No.
