Microsoft FrontPage 97

Personal Web Server

 

Follow these steps to replace this page with your own homepage

  1. In the Microsoft FrontPage Explorer, choose File: Open FrontPage Web.
  2. Type the name of your web server and click List Webs.
  3. Select the entry for <Root Web> and click OK.

This file is named index.htm and is the page you will see if you type only the name of your web server into a browser.

  1. Edit index.htm and add a link to another page or replace index.htm with a new homepage.
