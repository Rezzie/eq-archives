PkUnzip

Preparing PkUnzip
  1. Download pkz204g.exe from http://www.pkware.com/download.html.
  2. Copy the file you just downloaded to your DOS or Windows directory on your hard drive.
  3. If you are using Win 95, simply double click the pkz204g icon in your Windows folder. This will extract all of the files that are compressed in the file you downloaded.
  4. If you are using Win 3.X, double click your DOS Prompt icon found in your Main program group.
  5. You should already be in your windows directory, but if you want to change to your DOS directory, type cd \dos at your c:\windows> prompt.
  6. Run the file you downloaded by typing pkz204g. This will extract a number of files like pkzip.exe and pkunzip.exe.

Using PkUnzip

  1. Now, to unzip a file, you will want to change to the directory where the zipped file currently resides. To do this, type cd \<directory name> where <directory name> is the name of the directory you wish to go to.
  2. Type pkunzip <filename> where <filename> is the name of the zipped file you want to unzip.
  3. At this point, pkunzip will uncompress all of the files that make your zipped program or file work.
  4. When it finishes, type exit to return to Windows.
