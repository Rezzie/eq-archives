NewsWatcher

  1. Download NewsWatcher.
  2. Your Mac should automatically unstuff the program once you've downloaded it if you have an unstuffing program on your hard disk.
  3. Once the program is installed, run it by double clicking the NewsWatcher application.
  4. Go to File/Preferences.

  5. You will see the following screen:

  6. Choose Server Addresses. You'll see the following screen:

  7. The News Server should be news.enteract.com.
  8. The Mail Server should be smtp.enteract.com.
  9. You may change any of the other preferences to meet your own specifications. Click OK when done.
  10. Go to the Special pull-down menu and select Get Group List From Host....
  11. This should show you the full group listing. Now, because there are over 18,000 newsgroups, you want to be able to find the one you want to subscribe to quickly. To do this, go to Edit/Find.

  12. You will see the following screen:

  13. Type in whatever you want to search for. Let's look for all the EnterAct groups. Type enteract and then click Find.

  14. Let's say you want to subscribe to enteract.news (a good idea, really).
  15. Go to File/New Group Window.
  16. Drag enteract.news to this New Group Window that you've just opened up.
  17. Repeat this process until you've added all of the groups you want to be subscribed to.
  18. When finished, go to File/Save to save your selections.
  19. When you want to read an article in a newsgroup, simply double click the group's name.

  20. Double click the article you wish to read from within that group.
