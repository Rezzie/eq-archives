QVT/Term
  1. Download QVT/Term from either www.tucows.com or www.frontiernet.net/~qpcsoft/.
  2. Create a unique directory or folder for the program and save the file into that directory or folder.
  3. Run the self-extracting file you just downloaded.
  4. The installation wizard will do most of the work for you, but will ask you to choose where you want the program to be installed. Choose the directory you created in step 2 or just leave the default value if you prefer.
  5. When finished, you will be left with a program group for QVT/Term as well as an icon for the program. Double click the icon to start the program. You will see the following screen:

  6. While there are many settings you can play with, the first option you want to choose is Setup/Default Settings from the pull-down menus.

  7. You will see the following:

  8. Choose VT102 for your emulation.
  9. Feel free to play with the other settings, but all of the others are optional.
  10. To connect, simply go to File/Open.

  11. If you created an EnterAct connection, choose that one. If you just saved your changes to default, select the default connection.

  12. Login with your username (all lower case!) and your password.

  13. You will then be left with your UNIX prompt. Feel free to explore your account using telnet!
