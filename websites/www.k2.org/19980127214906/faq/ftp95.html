Windows95 FTP
  1. Click your Start button and go to Programs/MS-DOS Prompt. You will see the following screen:

  2. Type ftp shell.enteract.com. This assumes that you want to connect to your own account at EnterAct. You can connect to any ftp site you want though by just replacing shell.enteract.com with the host site that you desire.

  3. If you were going to log into your own account, you would enter your username and password at the respective prompts.
  4. If you wanted to log in anonymously into another ftp site, you would type anonymous at the login prompt and your e-mail address (username@enteract.com) at the password prompt.
  5. At this point you will be logged in and are able to use the ftp commands to get the desired files.
  6. To see the files, type dir. This will list the files available for downloading.
  7. To change to a particular directory, type cd <directory name> (cd pub for example).

Downloading
  1. To download a file, type get <filename>.
  2. If you want to get multiple files, you can type mget *. This will prompt you with each filename to see if you want to download it. You can reply with a y or n.
  3. NOTE: If you are downloading binary files, you must type bin at the ftp prompt before you begin the download.
  4. The files will be downloaded to whatever directory you were in when you started the ftp program.

Uploading
  1. To upload a file, make sure you changed to the DOS directory that held the files you want to upload before starting ftp.
  2. Type put <filename>.
  3. If you want to upload multiple files, you can type mput *. This will prompt you with each filename in your current directory to see if you want to upload it. You can reply with a y or n.
  4. NOTE: If you are downloading binary files, you must type bin at the ftp prompt before you begin the download.
  5. The files will be downloaded to whatever directory you were in when you started the ftp program.
