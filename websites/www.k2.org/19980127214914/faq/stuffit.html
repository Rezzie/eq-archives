StuffIt
  1. Download StuffIt
  2. Once you've downloaded and installed StuffIt, run the program by double clicking the application.
  3. Go to File/Expand....

  4. You will see the following screen:

  5. Choose the directory where your compressed file resides.
  6. Highlight the file you want to unstuff and click Open. This will decompress your file so that you can use it.
  7. Most often, when you download a stuffed program, StuffIt will automatically decompress the program for you. If it doesn't, though, you can always decompress it manually as explained in the above steps.
