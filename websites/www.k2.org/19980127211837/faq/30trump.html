Preparing for Installation

The first thing you will need to do is download trumpet 3.0 from Trumpet's web site.

Now you will want to create a directory for Trumpet Winsock in your File Manager.

  1. Open your File Manager.
  2. Highlight the C:\ directory found at the top of your directory tree.
  3. Go to File/Create Directory from the pull down menus located at the top of the screen.
  4. Call your directory trumpet.
  5. Drag the file you downloaded to the trumpet directory you just created.

Installation
  1. From the trumpet directory, please double click the file you just copied. This will extract a number of files and put them in the trumpet directory.
  2. Find the install.exe file and double click it. This will begin the setup process for Trumpet 3.0.
  3. After installation, you will be left with several icons, one of which will be Trumpet Winsock. Please double click it. You will see the following screen:

  4. Leave the IP address at 0.0.0.0. Even if you have a static IP address, the server will assign that IP address to you whether you enter it here or not.
  5. Enter 207.229.143.1 and 207.229.143.2 as the DNS servers. Make sure to separate these by a single space.
  6. Enter enteract.com as the Domain suffix.
  7. Change the Netmask to 255.255.255.0. To change this field, you will need to click on Packet Driver.
  8. Click PPP for the Driver.
  9. The MTU should read 1500.
  10. TCP RWIN should be 2048.
  11. Change TCP MSS to 1024.
  12. Make TCP RTOMAX equal to 60.
  13. Demand load timeout can remain 5 seconds.
  14. Click OK.
  15. Click Dialer Settings. You will see the following screen:

  16. Make sure that the Comm port is set correctly. If you aren't sure which comm port you use, try com2 first and if that doesn't work, try each port until you select the correct one.
  17. Choose the correct Baud rate. 57600 is fine for 28,800 modems. 38,400 should work well for 14,400 modems.
  18. The rest of the settings are up to you.

Modifications

When you finish the installation of Trumpet, there will still be a couple of things we will need to change.

  1. In Trumpet go to File/PPP Options.

  2. Choose Password Authentication Protocol (PAP) and enter your username and password in the fields (lower case). Click OK.

  3. Now go to Dialer/Profile.

  4. Enter your username, password, and the phone number you will be dialing to connect to us.

  5. Click Server Settings.

  6. Make sure every field is blank. If something is entered in a field, please delete it.
  7. Make sure the bottom checkbox is left unchecked.
  8. Click OK when done.

Dialing
  1. Before you go ahead and connect to us, you will need to exit trumpet and go back into it.
  2. Once you've done that, go to the Dialer pull-down menu again and click login.

This will dial your modem and allow you get connected to the Internet. When it is finished dialing, minimize the screen and open up whatever internet application you choose (i.e. Netscape, Eudora, WS_FTP, etc.).

To disconnect your connection, maximize Trumpet and go to the Dialer pull-down menu again. Click Bye. Trumpet will then disconnect your modem.
