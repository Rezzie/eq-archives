Free Software

Here are a few freeware programs I've written for the Macintosh. They require system 7, and have been tested very little, so use at your own risk. Share and enjoy.
  • CivCheat: This program modifies saved Cilivilzation games, giving you additional gold, settlers, or advances. Download CivCheat 2.0

  • SimTower Cheat: This program modifies saved SimTower games, giving you additional money. Download SimTower Cheat 1.0

  • ColPatcher: This program patches Microprose's Colonization to run a little faster on 680x0 Macintoshes. Disclaimer: I do not work for Microprose, and this patch was done without their knowledge. Download ColPatcher 1.0

  • HyperTyper: Need to change file types and creators? Then this is just the program you need. OK, so it violates every Macintosh Human Interface Guideline, but it still is "the coolest 4K Macintosh program ever written". Download HyperTyper

  • FatPigBits Gold: Yes, the "Ultimate Visiual Interface Inspection Tool" is now available on the web. Download FatPigBits Gold
