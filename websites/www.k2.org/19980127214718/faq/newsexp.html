News Express

  1. If you haven't already, download News Express from www.tucows.com.
  2. Unzip the program into its own directory on your hard drive.
  3. Run the nx.exe file that is created after you have unzipped the downloaded file.
  4. The first time you run the program, you will see the following screen:

  5. You want to select Retrieve new groups only [if any]. This will make News Express bring down only new news groups as opposed to every single news group (there are over 16,000 of them) every time you connect to the news server.
  6. Click the Servers and Directory tab. You will see the following screen:

  7. The News [NNTP] server should be news.enteract.com.
  8. IMPORTANT: Leave the username and password fields blank. If you fill these fields in, you will not be able to connect to the news server.
  9. The Mail [SMTP] server is smtp.enteract.com.
  10. Leave the Home directory as the default value that News Express already entered.
  11. Click on the Personal Information tab. You will see the following screen:

  12. Fill in your name and e-mail address (username@enteract.com). You can leave the Organization and Default signature file blank if you wish.
  13. The rest of the options are up to you. Feel free to look through all the tabs and change whatever settings you desire. When done, click OK. NOTE: You can go back to these settings at any time by going to File/Options in the main News Express window (as seen below).

  14. Now, because you don't want to be subscribed to all 17,000+ groups (who has time to read them all anyway??), you want to subscribe to only those that interest you. News Express allows you to search for the groups you want. Let's say you want to see all of the music groups. You would then just type music in the Search field located in the lower right hand corner of the screen. You would see the following screen:

  15. As you can see, only the groups with the string music are listed. To subscribe to one of them, just click the little round ball located to the left of the group name. Once you are finished subscribing to groups, these groups will be found again by clicking on the Subscribed groups tab so that you don't have to wade through all of the groups every time you go into News Express.
  16. To read the articles in a group, simply double click the group you want to go into and then double click the article you want to read.
  17. That should get you started. For more information about News Express, check out their on-line Help menu from the main screen.
