Make a Web Page Counter



EnterAct subscribers can create web counters from this web page. You must know your username and current password to create a counter. Your username is the name you input when logging onto the system. It's equivalent to the first part of your email address... i.e., yourname in "yourname@enteract.com".

Username:

Password:

Counter Name: (no spaces permitted in the counter name)

Your new web counter will be named: yourname-countername.dat where yourname is your account name and countername is the name you gave for your counter above.

Reference it like so in your web page: 

<img src = "http://www.enteract.com/cgi-bin/Count.cgi?dd=A|df=yourname-countername.dat">

The dd=A part specifies the "A" style counter. Other styles are available, like "childbook", as used at the top of this page. See our counter help page for other counter styles and options.

Once you've created your counter, if you would like to initialize it to some other value than 0 (for example, if you want to restart the count at the same value your page had when hosted at a different ISP), you can use our change counter value page!
