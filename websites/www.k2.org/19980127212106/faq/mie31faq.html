Entering windows automatically and autoexec.bat

This following section cures a great many problems and errors with this software and should be checked first if you get any strange errors or cannot connect with us. If you go into windows or some menu right after booting up, you will need to edit your autoexec.bat file after installing this software. There are 4 lines that are appended to the end of this file that will not be loaded if windows or a menu are loaded first.

Here are the four lines:

SET PATH=%PATH%;C:\IEXPLORE\mail
SET PATH=%PATH%;C:\IEXPLORE
set ieppp=c:\IEXPLORE
set pctcp=c:\IEXPLORE\pctcp.shv

Move these above the c:\windows statement in the autoexec.bat file or move the c:\windows statement below these lines. You should then be able to reboot and double click on the Get me on the Internet icon to obtain your EnterAct dialer icon.

Winsock.dll

If you have several applications which use your modem (e.g. AOL or Trumpet Winsock), chances are very good that you will have more than one instance of winsock.dll on your hard drive. If this is the case, it could cause problems when you try to connect to EnterAct. If you find that it looks like you connect, but then cannot go to any web page, retrieve mail, or read news, it is possible that you have more than one of these files and they are conflicting.

To solve this, do a search for winsock.dll from your file manager or from within DOS with the C:\>dir /s winsock.dll command. You should have one in your iexplore or windows directory. Any others should be renamed to winsock.old.

Disk 1

Sometimes after installing Disk 1 of the 3 disk set, the disk becomes useless because all of the filenames are changed and all of the sizes of the files are set to be about 4 gig! To prevent this, copy this disk to your hard drive in a separate directory before installation.

Baud Rate

Try lowering your baud rate. With some modems, if your baud rate is too high, you may not connect well, or even at all. If you have a 28,800 modem, try setting your baud rate to 57,600 bps. And if you have a 14,400 modem, try setting it to 38,400. You can always try raising it up to 115,200 for 28.8 modems or 57,600 for 14,400 modems to see if you can achieve faster speeds.

The baud rate setting can be found by double clicking on the EnterAct dialer icon and clicking Properties. There is a modem tab that will show the maximum speed for the connection.

Init String

There is no obvious setting where you can change your initialization string. What you will have to do is edit the modems2.ini file. This file should have two lines; one that lists your modem and one that lists the init string. If you are having trouble being able to dial out, change it to AT&F, AT&F1, or simply ATZ.

Starting Over

If you need to completely re-install this software, there are a few things you'll have to delete before you do so.

  1. In the system.ini file there are two lines that begin with device=c:\iexplore\… These should be deleted.
  2. Delete the entirety of the iexplore directory.
  3. In the Windows directory, delete iexplore.*.
  4. In the Windows directory there is also a signup directory. Delete it.
  5. Delete the 4 lines in the autoexec.bat file that mention iexplore.

With all of these things deleted, you are ready to re-install the software.
