Setting up Pegasus Mail

NOTE: Although Pegasus can be used for multiple mail accounts and network purposes, these instructions will only explain how to configure the program for a single user. For more information about other options, consult the Pegasus on-line help area or their web site at www.pegasus.usa.com.

  1. If you don't have it already, download the latest version of Pegasus from either www.tucows.com or www.pegasus.usa.com.
  2. Create a directory or folder for Pegasus and move the downloaded file to that directory.
  3. Run the self-extracting executable by double clicking it. This will extract all of the files necessary to set up Pegasus.
  4. To run Pegasus, double click the Pegasus Mail icon.
  5. You will see the following screen:

  6. Click the first icon to configure for a single user.
  7. You will be asked where you would like Pegasus to store mail-related files. Keep the default path and click OK.
  8. At the next window, simply click OK.
  9. You will then see this screen:

  10. Click Yes.
  11. You will then have to configure Pegasus for EnterAct's mail servers. You will see the following screen. Fill it out as you see here:

  12. The POP3 host is pop.enteract.com.
  13. The User name should be your username and in lower case (as opposed to fred, unless of course your username is in fact fred).
  14. The Password is your password that you use to log in.
  15. The SMTP host should be smtp.enteract.com.
  16. Click Advanced configuration options. You will see the following screen:

  17. While most of these options are completely up to you, you should select the Delete mail from host once successfully retrieved option.
  18. Also, unless you want outgoing mail being stored in a queue as opposed to being sent immediately, you'll want to select Send mail at once without placing in queue as well.
  19. Make sure you fill in your e-mail address in the From field.
  20. Click OK until you are back at the main Pegasus screen.

Checking Mail

In the main screen of Pegasus, click on the File pull-down menu and select Check host for new mail. Once all of your messages are downloaded, you can simply double click a message to read it.

Sending Mail

  1. To send a message, simply go to the File pull-down menu and select New Message.
  2. You can then enter the recipient's e-mail address and the subject of your message, along with the message itself.

  3. When done, click the Send button. Your message will be sent automatically unless you configured Pegasus to queue your mail.

Winsock.dll woes with Pegasus

If you go to your File pull down menu and do not see such items as Network Configuration or Check Host for New Mail, you will need to do one additional thing to get Pegasus to work properly. Copy your winsock.dll file from wherever it currently is to your Pegasus directory. Then, once you go back into Pegasus you should be able to check for new mail as well as enjoy other options previously unavailable to you.
