NetTerm
  1. Download NetTerm from http://starbase.neosoft.com/~zkrr01/ or from http://www.tucows.com.
  2. Copy the file you download into its own directory. You can call it netterm if you like.
  3. Run the self-extracting executable by double clicking the file in your file manager in Win 3.1 or from the folder you created in Win 95.
  4. A file called setup.exe will be created. Run it by double clicking it.
  5. You will be asked what directory or folder you want to install NetTerm into. Choose the netterm directory/folder you created earlier in step 2.
  6. When the setup program finishes installing the program, you'll need to go to the program group where it was installed and double click the NetTerm icon.
  7. You will now need to configure NetTerm to connect with EnterAct. To do this, choose File/Phone Directory from the pull-down menus.

  8. You will see the following screen:

  9. To add an entry (i.e. EnterAct), click the Add button located at the bottom of the window.
  10. In the space provided, change the host name to shell.enteract.com.
  11. Make sure that the emulation is set to VT100.
  12. If you are dialed up to EnterAct already, simply click Connect and you will see the following screen:

  13. Login with your username (all lower case!) and your password.
  14. You will then be left with your UNIX prompt. Feel free to explore your account using telnet!
