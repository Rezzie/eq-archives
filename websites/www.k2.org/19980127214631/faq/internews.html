InterNews

Setting Up InterNews
  1. If you don't already have a copy of InterNews, go ahead and download one.
  2. The file that you download is a self-extracting archive. Just double-click its icon and it will install itself.
  3. Double-click the InterNews icon. You will be prompted with the following screen:

  4. For News Host enter news.enteract.com. All other fields can be left as is.
  5. Click Save.
  6. InterNews will now begin to download a list of availiable newsgroups. EnterAct offers over 18,000 newsgroups, so this may take a few minutes.

  7. Once InterNews is finished downloading, it will create a folder with the following icons:

  8. Double-click the All Newsgroups icon. You will see the following screen:

  9. In the upper-left window is a list of all the newsgroups availiable on EnterAct's news server. To see a display of messages within a newsgroup, just double-click the name of the newsgroup. Message headers will then be displayed in the upper-right window. To view the body of a message, double-click its header. The body of the message will then be displayed in the bottom window.
  10. To look for a specific newsgroup, go to the Reading menu and click Find... In the window that appears, enter the name of a newsgroup or a keyword and then click Find. For example, if you entered "mac," one of the entries you would find would be enteract.mac, the EnterAct Macintosh newsgroup:

  11. Since you may not wish to search through all 18,000 or more newsgroups every time you use InterNews, you can create a subscription file that contains only your favorite groups. Go to the Subscriptions menu and click New Subscription. InterNews will create a subscription file and place that in your news folder.

  12. To add newsgroups to this file, simply click and drag the names of the newsgroups you want to save to the new subscription icon. Once you have done that, double-click the subscription icon to view the news groups you have selected.

Posting to Newsgroups
  1. To post a message to a newsgroup, simply select the newsgroup to which you would like to post a message in the upper-left window. Then, go to the Compose menu and select New Composition. You will see the following screen:

  2. In the Newsgroups field you will see the name of the newsgroup to which the message will be posted. Type a subject for your message in the Subject field and then type your message in the bottom half of the window. When you are ready to send the message, click the Post Article button.
  3. The first time you post a message with InterNews, you will be asked for the name and return e-mail address you wish to appear in the header field of the message. What you enter at this point will be the default settings for InterNews. To change these settings later, go to the Edit menu and click on Preferences, and then, in the Preferences window, select Personal.
