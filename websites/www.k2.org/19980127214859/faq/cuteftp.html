Cute FTP
  1. Download CuteFTP from either http://www.tucows.com or http://www.cuteftp.com.
  2. Copy the file you just downloaded to a unique folder or directory. Call it cuteftp if you like.
  3. Run the program by double clicking the file you just saved to the cuteftp directory or folder.
  4. You will see a program icon for CuteFTP. Run it by double clicking on it. You will see the following screen:

  5. To add a site to the list (EnterAct's for example), click the Add site button. You will see the following screen:

  6. Fill in the Site Label with the name of the site to which you want to connect (i.e. EnterAct).
  7. Fill in the Host Address with the address to which you want to connect (i.e. shell.enteract.com)
  8. If you are adding an entry for your personal account, fill in the User ID and Password with your own username and password.
  9. If you are adding an anonymous FTP site, click the Anonymous radio button in the Login type section of the screen.
  10. Click OK when done.
  11. To connect, simply make sure the entry is highlighted and click the Connect button located at the bottom right hand corner of the screen (see screen shot after instruction #4). You will see the following screen:

  12. On the left side of the screen you will see the files on your hard drive. On the right side are the files on the server you connected to. If you connected to shell.enteract.com and logged in as yourself, these files and directories will be what is in your personal account.
  13. To download files from the host to your hard drive, simply double click the file on the right side that you want. The file will be transferred to your hard drive in the directory you have selected on the left.
  14. To upload files from your hard drive to the host (if you want to move your web pages to the server for instance), simply double click the files you want to transfer. Your files will be transferred to the host in the directory you have selected on the right.
  15. You can always connect to any site (not just EnterAct's) to download files. All you need to know is the ftp site.
