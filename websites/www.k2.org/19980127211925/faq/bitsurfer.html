Motorola Bitsurfer

The following instructions are intended to be used by any customer with a Motorola Bitsurfer even though they specifically address configuration for Windows 95. While much of the information here is applicable to other operating systems, it does not currently explain specifically how to set up other software.

  • Install Your Terminal Adapter
  • Configure Your Terminal Adapter
  • Configure Windows 95

Install Your Terminal Adapter
  1. Go to your Modems icon from your Control Panel.
  2. Select the "Motorola BitSURFER PRO Plug & Play" modem. If you do not have this modem as an option, you will need to dial into the Motorola BBS (508-261-1058) and go into conferences\products\modem\bitsurfer\bsfaq\#14. Alternatively, you can obtain this, as well as the latest flash ROM updates at http://www.mot.com/MIMS/ISG/Products/bitsurfr_pro/bitsurfr_pro_software.html.
  3. Add this to the list of installed modems.

Configure Your Terminal Adapter

  1. Click Properties from your Modems control panel.
  2. Specify the correct com port for your ISDN device.
  3. Maximum speed should be 115200.
  4. Click the Connection tab.
  5. Click the Advanced button.
  6. In the Extra Settings field, type one of the following initialization strings; either should work: AT&F&C1&D2%A2=95%A4=0@B0=2 or AT&F&C1&D2\Q3%A2=3D95@B0=3D2%A4=3D1.
  7. Click OK.

Configure Windows 95
  1. Configure Windows 95 as you would for a connection with an analog modem. Click here for those instructions.
  2. Open your My Computer icon.
  3. Double click Dial Up Networking.
  4. Right click your EnterAct icon and select Properties.
  5. Make sure the Use country code and area code checkbox is not checked. If the local number you dial requires you to dial an area code, enter it as part of the phone number.
  6. If you are trying to connect using 2 channels, the phone number should be typed like this: 13127050096&6 (note no dashes). The entire number does not need to be typed twice. Only the last digit needs to be specified after the &.
  7. If you only want to connect using one channel, just enter the phone number in once without the &6 at the end.
  8. If you connect successfully with 2 channels, your DTE light will be a solid green light. If you are only connecting at one channel, you will see a flashing DTE light.
