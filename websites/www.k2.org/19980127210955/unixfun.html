Navigation

A short introduction to the filesystem.

Alright, so when you first come in to our system, it's pretty dull. Most likely you get a prompt like this;

$

One of the most useful commands in Unix;

$ ls 

Mail            News            mail            public_ftp      public_html

This shows you what files are in your directory. You can get more information by telling ls that you want a long listing;

$ ls -l 

total 16
drwxr-xr-x  2 djhoward  dialin  512 Jun  2 23:36 Mail
drwxr-xr-x  2 djhoward  dialin  512 Jun  2 23:36 News
drwx------  2 djhoward  dialin  512 Jul 11 19:48 mail
lrwxr-xr-x  1 djhoward  dialin   29 Jul 11 19:53 public_ftp -> /usr/local/ftp/users/djhoward
drwxr-xr-x  2 djhoward  dialin  512 Jun 28 15:45 public_html

Ahhh, very informative! The d on the first line of most of those entries indicates that the item listed on that line is a directory. You can find out what's inside of a directory by asking ls;

$ ls mail 

developer-support       info                    sales                   support
help                    log                     saved-messages          webmaster
hightech                modem                   sent-mail

Let's write a simple simple web page. We can use the pico full-page editor for this;

$ pico index.html

In pico you can manage about with the cursor keys, type in what you want, save, cut, copy paste, etc. Let's just type up something simple like; 

   UW PICO(tm) 2.5                  File: index.html                  Modified

<html><head>
<title>An Awesome Homepage!</title>
</head>

<body>
<h1>Hey!</h1>
<p>You know, <a href="http://www.enteract.com">EnterAct</a> is a really 
cool ISP!</p>
</body>
</html>










^G Get Help  ^O WriteOut  ^R Read File ^Y Prev Pg   ^K Cut Text  ^C Cur Pos
^X Exit      ^J Justify   ^W Where is  ^V Next Pg   ^U UnCut Text^T To Spell

Quick pico tutorial; those ^ characters at the bottom stand for the control key on your computer. For example, pressing the control key and x at the same time will allow you to save your file and exit. After this, your file is accessable;

$ ls 

index.html

That index.html file is available on the world wide web now. Fire up Netscape and take a look! (For example, the one Dannyman just made is under http://www.enteract.com/~djhoward/ !)

findnewsgroups

This program lists all USENET newsgroups with their accompaning descriptions or just those matching a string which you provide.

It is useful for finding newsgroups to which you might want to subscribe. Some newsreaders have the functionality of findnewsgroups built-in.

Run as findnewsgroups it will return a list of all USENET newsgroups and their descriptions carried by news.enteract.com. (Best to run this as findnewsgroups | more, because the list numbers over 15,000!)

Run as findnewsgroups string it will return a list of all newsgroups or descriptions matching that string.

For example, findnewsgroups beatles returns: 



rec.music.beatles       Postings about the Fab Four & their music.

For help and usage information, run findnewsgroups -h.

fortune

Per user request, the Unix program "fortune" has been installed on adam as /usr/local/bin/fortune.

To run fortune, type fortune at the shell prompt. Fortune returns a random fortune.

You can select from particular fortune databases. Available fortune databases include limerick, startrek and zippy.

To get a startrek fortune type: fortune startrek

There are a few other options available... Check the man page for details (man fortune).

If you decide that you want a random fortune when you log in, you can edit your .login file. Change the following line (it is probably the last line in the file unless you've changed your .login file from the default) from:

if (-x /usr/games/fortune) /usr/games/fortune
to
if (-x /usr/local/bin/fortune) /usr/local/bin/fortune

Hint: to edit your login file with pico, just type pico .login.

Enjoy!
