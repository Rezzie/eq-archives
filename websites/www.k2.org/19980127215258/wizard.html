Internet Setup Wizard - Intro

Getting Started with the Plus! Pack

Having shelled out $40 plus bucks, one would hope to get something worthwhile out of the Plus! Pack. The Internet Setup Wizard is one of these tools that makes owning the pack worth it.

Using the Internet Setup Wizard

Microsoft's Plus! Pack comes with a utility called the Internet Setup Wizard. This utility provides a quick and easy way to configure Windows95 to connect to your Internet Access Provider using DialUp Networking.

Prerequisites for Getting Started

  1. Internet Utilities have been installed from the Plus! Pack distribution CDROM
  2. Dial-Up Networking is installed and a "New Connection" has been created for your Internet Service Provider
  3. TCP/IP protocol is installed
