Chameleon Dialup Installation
  1. To begin, please double click your Custom Connect icon, found in your Netmanage Chameleon program group. You will see the following screen (note: the screen will be blank the first time you run Custom Connect):

  2. To setup your connection for EnterAct, click on Setup and choose IP Address.

  3. If you will be connecting with PPP, leave this dialog box alone. If you will be connecting with SLIP, change the value to 1.1.1.1.
  4. If you have a static IP address, you can enter your IP address here, although even if you use 0.0.0.0 (this lets the server assign the IP address), you will still connect to us with your static IP address.
  5. Now choose the Host Name option from the Setup pull-down menu.

  6. Enter your username in lower case and click OK.
  7. Now, from the Setup menu, choose Domain Name.

  8. Enter enteract.com and click OK.
  9. Choose Port from the Setup pull-down menu.

  10. Chameleon will not connect faster than 19200, so set your Baud Rate to 19,200.
  11. Data Bits should be 8.
  12. Stop Bits should be 1.
  13. Parity should be None.
  14. Choose Hardware Flow Control.
  15. Select the Connector you use. If you aren't sure, try Com1 first. If your modem does not respond, try Com2. Go down the list until you find the correct Com port.
  16. Click OK.
  17. Next, select Modem from the Setup pull-down menu.

  18. The only thing you may ever want to change in this dialog box is the initialization string. ATZ is fine, but you might want to try AT&F or AT&F1 to see if your connection improves at all. Note that these two strings might in fact not work at all for your modem.
  19. Click OK.
  20. Choose Dial from the Setup pull-down menu.

  21. Enter the phone number you wish to dial. If you aren't dialing within your same area code, make sure you enter the area code before the phone number.
  22. Click OK.
  23. Choose Login from the Setup pull-down menu.

  24. Enter your username (all lower case) and your password (case sensitive).
  25. For Startup Command, enter either ppp or slip depending on which protocol you wish to use.
  26. Click OK.
  27. Go to the Services pull-down menu.

  28. Choose Domain Servers

  29. Enter 207.229.143.1 in the first set of boxes.
  30. Enter 207.229.143.2 in the second set of boxes.
  31. Leave the third set of boxes set to zeros.
  32. Click OK.

You Are Now Ready to Connect!!

When you are ready to connect, simply click the Connect option on your task bar (it is located next to Services). Chameleon will automatically log you in. When it is finished you will be able to start up Chameleon's web broswer. It will be the WebSurfr icon in your Chameleon program group.

From here feel free to visit any sites you wish or download any software.
