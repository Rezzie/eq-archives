Entisa president, Don Beach and vice president Terry Smith agree that Obsydian, a model based object oriented CASE tool, is a robust development environment. That is why Entisa is committed to providing tools, software and consulting services supporting Obsydian.

Obsydian is a model based repository which seamlessly embraces the sometimes complicated and confusing object oriented techniques. It provides multi-platform code generation which frees developers from the tedious task of learning various platforms, coding interfaces and learning communications software. The benefits are endless. For further information about Obsydian visit the Synon web site.

Have you heard about the ODJ? (Obsydian Developers Journal). Finally, a publication specifically dedicated to the Obsydian community. It is packed full of useful techniques and vendors who provide a wide range of products and services. We at Entisa are contributors to the publication...our very own president Don Beach aka Mr. Meta authors tips on using Meta-code.
