Anarchie
  1. Download Anarchie
  2. Use an unstuffing program like StuffIt to uncompress the program.
  3. Run Anarchie by double clicking on the application.
  4. To begin an FTP session, go to the FTP/FTP... menu.

  5. You will see the following screen:

  6. Enter the host you want to connect to into the host field. If you are connecting to your own account, you'll want to enter shell.enteract.com.
  7. If you are connecting to an anonymous ftp site leave the username and password fields blank.
  8. If you are connecting to your own account, enter your username and password in the appropriate fields.
  9. You can leave Path blank.
  10. Click List.
  11. You will see the following screen:

Downloading
  1. Double click the directory that holds the file you wish to download. If the file is not found in the first directory you click on, keep double clicking on directories until you find it.
  2. Highlight that file and go to the FTP/Get Selection pull-down menu or simply double click the file and it will download to your hard disk.

Uploading
  1. Double click the directory that you want to upload your file to. You may want to simply upload to your root directory.
  2. Go to the FTP/Put... directory. You will see the following screen:

  3. You will need to find the directory that holds the file you are uploading to the server. Double click the appropriate directories until you find your file.
  4. When you find it, double click the file. Your file will then be uploaded from your hard drive to the server you connected to originally.
