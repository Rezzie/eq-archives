Star Fox 64

I just got this game on July 2, 1997!!! I think that it is a really cool game. There are two ways to accomplish a mission which makes it exciting. If you really need help, check out Starfox 64. The game is hard in some levels but easy in others. The rumble pack doesn't really make a difference but it's still cool.
