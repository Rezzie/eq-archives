How to Add a Counter to Your Page

Counting your hits in six easy steps

  1. Find the counter style below that you like.

  2. Copy the text below it that starts with <img and ends with dat">, and paste it directly into the place you want it to appear on your web page.
    For more details on how to edit your web page, see how to do a web page, then come back here.

  3. Change the word example to something unique, like your site or user name.

  4. Create the .dat file for your counter. This files keeps track of your current counter count. It resides in the directory /usr/local/www/Counter/data/ on our Unix system. This file can be created in one of two ways:

    • Simple way: Use this handy-dandy web page.
    • Somewhat harder way: Run the program addcounter on our Unix system. If you wish for the counter to have a name different from yourusername.dat then specify which name you wish the counter to have. i.e. addcounter lugnuts will create a counter named yourusername-lugnuts.dat for you to use. If you don't specify a name, addcounter will use your username for the new counter's name.
      Here, yourusername refers to the name you use to connect to EnterAct. The first part of your email address.

    You can create as many independent counter .dat files as you want. Each will be named yourusername-something.dat. Create one for each web page you want to keep track of!

  5. Save your work on your web page and if necessary upload your new page to the server.
    Once again, you can refer to how to do a web page if you require assistance.

  6. If you'd like to initialize your newly created counter to something other than "0", you can use our change counter value page.

    More detailed documentation (like, really technical) is also available.*
