Features

Windfall's clients will have the benefit of communicating with the same person for all aspects of their page, from graphic design to connectivity issues. No aspect of the design process is delegated to outside sources.

Included in the above pricing information are all aspects of Web site hosting, including computer equipment rental, hard drive space, T1 connection bandwidth, and web server management. As a client, you will not have to deal with any technical aspect of "owning a Web site" unless you choose to.

Flexibility is a key component of Windfall's philosophy. If your vision for a Web site changes over time, Windfall is willing and able to adapt.

Hourly billing is straightforward, with descriptions of services rendered charted next to accurate time entries. This applies to work done after initial contract is completed.

If a client wishes to move their Web site from Windfall's server to their own at any point, Windfall is willing to aid in this simple process.
