Fetch
  1. Download Fetch
  2. Open Fetch.
  3. You will either see this screen appear:

  4. Or you will need to select New Connection from the File menu.

  5. Enter the host you want to connect to into the host field. If you are connecting to your own account, you'll want to enter shell.enteract.com.
  6. If you are connecting to an anonymous ftp site enter anonymous as your user ID and your e-mail address as the password, username@enteract.com.
  7. If you are connecting to your own account, enter your username and password in the appropriate fields.
  8. You can leave Directory blank.
  9. Click OK.

Downloading
  1. Double click the directory that holds the file you wish to download. If the file is not found in the first directory you click on, keep double clicking on directories until you find it.
  2. Highlight that file and click the Get File... button or just double click the file.
  3. You will be asked where you want to put the file you are downloading. You will see the following screen:

  4. Choose where you want to put the file by selecting the down-arrow located at the top of the screen or by double clicking the appropriate directory.
  5. Click the Open button.
  6. The transfer of data will begin. When it finishes you can close out of Fetch.

Uploading
  1. Double click the directory that you want to upload your file to. You may want to simply upload to your root directory. That is fine.
  2. Click the Put File... button. You will see the following screen:

  3. You will need to find the directory that holds the file you are uploading to the server. Double click the appropriate directories until you find your file.
  4. When you find it, highlight it and click the Open button or simply double click the file. Your file will then be uploaded from your hard drive to the server you connected to originally.
