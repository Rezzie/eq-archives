ConfigPPP Setup
  1. Open ConfigPPP, which can be found in your Control Panels.
  2. You will see the following screen:

  3. Click Config...
  4. You will see the following screen:

  5. Fill in a name for the PPP Server. “EnterAct” is fine.
  6. The port speed can be set at 57600 if you have a 28.8 modem. If you have a 14.4 modem, set the port speed to 38400.
  7. Flow Control should be CTS&RTS (DTR).
  8. Make sure the Tone Dial radio button is selected.
  9. Enter the local phone number that you will be using to dial into EnterAct where it asks for Phone num. If you need to disable call waiting, put 1170 or *70, in front of the phone number.
  10. Most modems don't require a modem initialization string for the Mac, but there are two exceptions that you might want to be aware of. The first is for Global Village modems. If you have this kind, please make the modem string AT&F1&K0&K3. If you have a Supra, enter AT&F1 in the modem init area. If neither of these options work for you, you can always try ATZ.
  11. Click Authentication...
  12. You will see the following screen:

  13. Enter your username in all lower case where it asks for Auth. ID and your password where it asks for Password. The password will appear as dots. Make sure you type it correctly. If your password has upper case characters, make sure that you type it with upper case characters.
  14. Click OK.
  15. Click the LCP Options button.
  16. You will see the following screen:

  17. Make the Timeout value 15 seconds.
  18. Click OK.
  19. Click the IPCP Options button.
  20. You will see the following screen:

  21. Make the Timeout value 15 seconds.
  22. Click Done to finish.

Dialing

From here you can simply click the Open button. Your modem will dial for you and log you into EnterAct. Once you are connected, the faces in the upper left hand corner of your screen will smile and it will say PPP Up. At this point you can open up any internet software you have. Enjoy!
