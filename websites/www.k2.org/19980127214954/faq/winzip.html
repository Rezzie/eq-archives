WinZip

Installing WinZip
  1. Download WinZip from either http://www.tucows.com or http://www.winzip.com.
  2. Copy the file you just downloaded to a unique folder or directory. Call it winzip if you like.
  3. Run the program by double clicking the file you just saved to the winzip directory or folder.
  4. You will see a program icon for winzip. Run the program by double clicking on the icon. This will take you through WinZip's setup wizard. You will see the following screen:

  5. Click Setup to begin.
  6. You will be asked which directory you want to install WinZip into. Click OK to keep the default.
  7. Click Next until you see the following screen:

  8. Choose to Start with WinZip Classic and click Next.
  9. The next screen will ask if you want to do an express setup or a custom setup. Choose Express.

  10. This will finish the installation and create all of your icons for you. Click Finish when it is done.

Unzipping Files
  1. To unzip a file, click Open.

  2. Select the file you wish to unzip by double clicking it. You will see the following screen:

  3. If you know the directory you want to put these files in, click the Extract icon and choose the appropriate directory. WinZip will take these files and put them in the directory of your choosing.
  4. If you wish to have WinZip not only unzip the files but also install the program, click Check Out. You will see the following screen:

  5. You can change the folder if you don't want the program to be installed in the default folder. When you've chosen the folder, click OK.
  6. If there is an install.exe or setup.exe to be run in the program you are unzipping, WinZip will also run that.
  7. When you finish, and wish to close out WinZip, you will be asked if you want WinZip to delete the folder and icons it just created. Unless you want these things to be deleted, click No.
