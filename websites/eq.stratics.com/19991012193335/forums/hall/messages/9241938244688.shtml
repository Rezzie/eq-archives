You start with all the trade skills, plus a lot of other skills. If you go to your guildmaster, some skills are shown that have a level by them, which means you can't train them until that level. For example meditate for magic users can't be trained until level 4. You can not use that skill until you train one point at least, even if you are level 4. The skills you start with can be seen by clicking the skills button on the inventory screen. Any skill listed there can be used, except for taunt, like I said.
