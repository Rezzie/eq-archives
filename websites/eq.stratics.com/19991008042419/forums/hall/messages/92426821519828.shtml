In an economic sense, it doesn't make sense as you've said. At the very least, items required for a trade skill which are found in cities where the complimentary items cannot be found should be at greatlly reduced cost, i.e. if a cities has groove nocks to fletch but not other components, local npcs should regard groove nocks as nearly worthless, and other components as quite valuable. This is appropriate; groove nocks ARE worthless without the other items. (In economics, the classic example of the value of "perfect compliments" such as this is shoes; how many left shoes are you willing to buy if no right shoes are available? Very, very few. Low demand equals low price.

If Verant wanted to make me see the world, they should have made it worthwhile to travel between places carrying goods that are common in one locale and rare in another, not by making an item totally unavailable in some places and available in others. The fixed price structure in the game is annoying.

While Im on the subject, I might as well rant: it is a little irritating how Verant made all these game aspects in order to "encourage" us to do this or this. Inavailibility of items doesn't encourage me to travel, it MAKES me travel. Having no npc healers doesnt encourage me to find player clerics, it MAKES me find player clerics (which can be tough at 2:00 am PST). The only one that is encouragement instead of making me do something is the whole soloing issue, which I think is as it should be. But really, whose game is this anyway? If a person wanted to be a misanthropic hermit type and never interact with other players, that should be their option. It's one thing to encourage me to find a player cleric by having npcs charge 1pp for a cure disease; its another not to have any npcs that will cure disease at all.

All these issues can be summarized as follows: Verant did not select game elements based upon how well they fit a realistic model of how Norrath might work, they selected them in order to make the game hard and to force certain kinds of player interaction. Thats artificial. An immersive world needs to start with a realistic model and then add game features based upon what would make sense in that model, not start with a game concept and then stick with it. The more it feels like a game, the less enjoyable the role-playing experience is. EQ is good, but could be so much more if the damn world just felt at all real.
