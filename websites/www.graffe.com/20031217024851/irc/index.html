::: Wizard Chat :::

You can chat will fellow wizards by using one of two methods. You can either use the java chat or you can use the traditional way of using a stand alone IRC client. Two client suggestions are mIRC or Bersirc.

Java Chat:
If you wish to use the Java Chat, it has been set it up two different ways. Please be sure to input your nickname (use your wizard's first name) in the field provided before you join. If you do not, your name in chat will be 'Guest' with some huge number after it and you will look like a full fledged newbie asking to be heckled as well as kicked from the channel after a 5 minute time period. Here are your two java chat choices -

  • Java Chat Standard
  • Java Chat in a draggable window (recommended)

IRC Clients:
If you are going to use a client, click here or use these settings -

  • Server: irc.stratics.com
  • Port: 6667
  • Channel: #graffe

Suggestions:

  1. Register your nickname with the server by typing /msg nickserv register <yourpasswordhere>
  2. After you have registered your nickname, indentify yourself to nickserve by typing /msg nickserve identify <yourpasswordhere>

Rules:

  1. Keep the profanity to a minimum.
  2. No ASCII pictures will be allowed.
  3. Over use of chat colors is very annoying, try not to use them.
