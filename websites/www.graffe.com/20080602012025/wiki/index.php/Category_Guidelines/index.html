Existing Categories

Game Specific

At minimum all articles should have their GameCode Category attached to the article. To do this, add [[Category:GameCode|TheArticleName]] to the article, replacing "TheArticleName" with the last part of your article name and "GameCode" with the appropriate code from the table below:

Game Game Code
EverQuest EQ
EverQuest2 EQII
EVE Online EVE
Lord of the Rings: Online LOTR
Vanguard: Saga of Heroes VG
World of Warcraft WoW



Expansion Specific

If the article pertains to an aspect of the game that is unique to a certain expansion, add the appropriate Category tag, using the GameCode and Expansion Code. Ex: [[Category:EQ: OoW|TheArticleName]]. The follow Expansion Codes exist:

Others

If there are any other existing categories that you feel the article should be part of, please add them accordingly. For a listing of all categories, see: Special:Categories.

Non-Existent Categories

If you feel that a certain article should belong to a category that doesn't exist, use the following instructions:

  1. Add [[Category:GameCode: NewCategoryName|TheArticleName]] to the article.
  2. Save the article and return back to viewing the article.
  3. The new category link will be displayed at the bottom of the article in red. Click this to go to the category page.
  4. On the new category page, add a brief description describing the category and save.
  5. Your category is now created.
