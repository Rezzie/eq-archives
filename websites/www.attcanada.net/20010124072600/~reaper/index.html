What's New

If any of you have Bard related news or information that you think would be useful, please send it to me.  Thank you.

January 22th, 2001

  • Changed the Melody of Ervaj song to be level 50, instead of level 49, it apparently is listed as a level 50 song on the actual song sheet.
  • Deleted a few dead links on the Links page.

January 14th, 2001

  • Made a few minor changes to drop info on Velious songs, and changed the target of Angstlich's Assonance (level 60 song) from Party to Target.
  • Changed the times on a few of the disciplines on the Skills page to reflect the changes made in the recent patch.

January 12th, 2001

  • I'd like to thank everyone for their comments about the new colour scheme, haven't had any bad remarks about it, so I'll see about putting it on the message boards in the next couple of days.
  • The Composition of Ervaj (level 60 SoV song) has been confirmed as dropping off that big turtle in the Iceclad Ocean, so you can bet it's an NPC drop spell off level 60ish mobs.

January 11th, 2001

  • I felt like spicing up the site a little, so I went for a new graphics theme, send me an email and lemme know what you think!  If you all approve, I'll probably change over the message boards too.
  • I've added the Flute of the Sacred Glade to the Music Room, that bad boy drops in the Plane of Growth, so it's gotta be good :)
  • I know this has been posted to death on the message boards, but I thought I should warn people here.  The latest patch seems to have killed the bard's ability to run Selo's with a drum, making lower level bards slower than SoW.  This means that we'll be slowpokes, and some mobs can catch us (or outrun us).  So everyone, please, be careful.  If you have an opinion on this matter, Solos has posted a thread on Backstage addressing this issue, perhaps you should go voice your opinions so Absor can hear us!  It's time Verant stops treating us like we're unimportant.  Here's the link to the thread, tell us what you think!

January 10th, 2001

  • In light of some new submitted information, I have changed the effect on a couple of SoV songs, as well as added their drop locations, to the Songs page.
