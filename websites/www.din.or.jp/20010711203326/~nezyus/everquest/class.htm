職業　Classes
　ここでは、EverQuestで使用可能な14種の職業を紹介します。

　スキル習得レベル表は、職業に関わらない生産系や戦闘の基本スキル（Defense・Offence・Hand-to-Hand）、及び魔法使い系の職業のすべてが持っている魔法の基本5スキルは表記してありません。初期段階で自動的に習得しているものだと思ってください。

（ただし魔法基本5スキルは複合上級職はレベル9で習得します。Rank1 Spell習得と同時ってことですね）

    - Bard　/　吟遊詩人 -
    コメント Lv. 習得スキル名
    　このゲームでは、吟遊詩人といってもプレートの鎧まで装備できたりして、そこそこの白兵戦能力を持つ、魔法戦士的な存在のようです。
    　Bardの特徴はやはりその唇から紡がれる魔法の歌で、パーティー全体をサポートできるような付与・回復魔法が多く、どんなパーティーでもBardは歓迎されるでしょう。
    　Bardの魔法の歌はマナを必要としないと言うことも大きな特徴です。
    　また、彼らは盗賊の技術もたしなみます。

    　やはり基本通り、武器は鈍器中心。プレートを装備すれば比較的白兵戦もこなすようです。

    - Druid　/　ドルイド -
    コメント Lv. 習得スキル名
    　自然を愛する魔法使い、ドルイドは回復魔法や攻撃魔法を使いこなし、防御魔法や移動時の魔法をもこなす多彩な魔術師です。
    　決して強くはありませんが、それなりに白兵戦も可能で、なかなか楽しそうなクラスです。

    - Enchanter　/　付与魔術師 -
    コメント Lv. 習得スキル名
    　付与魔術師。といってもよくわかりにくいかも知れませんね。
    　Enchanterは対象の能力を上げたり下げたりといった魔法を得意とする魔術師で、パーティープレイでこそその真価を発揮します。
    　その反面、直接攻撃魔法のバリエーションが乏しかったり、単独行動は少々リスクがあるかも知れません。
    　パーティープレイができる人向けといえるでしょう。

    - Magician　/　召還術師 -
    コメント Lv. 習得スキル名
    　食料や武器、しもべとなるモンスターを呼び出し、使役させる魔法を得意とする魔法使いタイプのクラス。
    　召還魔法だけでなく、攻撃魔法も使いこなし、呼び出したしもべを盾にしながら、魔法で援護射撃、という効率のいい戦闘が行えます。
    　強力なクラスの一つであるといえるでしょう。

    - Monk　/　モンク -
    コメント Lv. 習得スキル名
    　軽装で、自分の肉体を武器防具として格闘する、いわゆる武闘家、格闘家ってやつです。
    　この手のクラスって何故かMonk（修道士）ってあっちでは言うんですが、少林寺拳法か何かのイメージなんですかね。
    　Monkは徒手空拳で戦うため、装備品に金がかからないと言うのが長所でもあり、金を使う楽しみが乏しいという短所でもあります。
    　また、Monkは多彩な独自のスキルを持ち、他のクラスとはひと味違った楽しみがあるでしょう。

    - Necromancer　/　死霊術師 -
    コメント Lv. 習得スキル名
    　NecromancerはMagicianと同様、モンスターを呼びだし、使役する戦い方に長けたクラスです。
    　Magicianと異なり、呼び出すモンスターがアンデッドという忌まわしいものであるため、このクラスに就くとNPCの対応が悪くなります。
    　しかしながら、召還魔法の他にも、毒系の魔法、さらには対象のHPを自分のHPに転換する魔法など、特異なものを使いこなす彼らは非常に有能なクラスであるといえるでしょう。

    - Paladin　/　聖戦士 -
    コメント Lv. 習得スキル名
    　WarriorとClericの能力を7:3の割合で併せ持つクラスです。
    　戦士でありながら回復魔法を使い、しかもパーティーの最前列にいられる戦闘力はとても優秀です。
    　しかし、その反面、成長速度が犠牲になっています。
    　また、PaladinはLay on Handという、HPが完全回復する奇跡を一日に一回起こすことができます。

    - Ranger　/　レンジャー -
    コメント Lv. 習得スキル名
    　WarriorとDruidの能力を8:2の割合で併せ持つクラスです。
    　やはりPaladinと同様、複合職の定めである成長の遅さという特徴を持ちます。
    　白兵戦の能力はWarriorに次ぎ、Dual Wield（二刀流）などのスキルを修得することができます。
    　また、野外活動に便利なスキルを覚えることも特徴といえるでしょう。

    - Rogue　/　盗賊 -
    コメント Lv. 習得スキル名
    　鍵開け、罠の解除、盗みといった盗賊的なスキルに加え、毒塗りや不意打ちといった暗殺者的なスキルをも使いこなすのがRogueです。
    　多くの武器を使うことができ、チェインまでの鎧を装備できるRogueは戦闘員としても高い性能を示します。

    - Warrior　/　戦士 -
    コメント Lv. 習得スキル名
    　装備で身を固め、敵に向かって直進し、ひたすら殴り続けるのがWarriorのプレイスタイルです。
    　武器を使った戦闘ではWarriorの右に出る者はいません。
    　最もプレイするのが容易なクラスですが、回復能力や決定的な決め手に欠けるので、やはり魔法の使い手とともに行動するのがいいでしょう。

    - Wizard　/　魔術師 -
    コメント Lv. 習得スキル名
    　純粋な破壊用の攻撃魔法の扱いに欠けてはWizardの右に出る者はいません。
    　しかし、その反面、補助的な魔法に乏しく、完全な戦闘要員であり、誰かのサポートをするタイプではないといえます。
    　逆にその貧弱な体を守り、詠唱時間を稼いでくれる盾となるクラスとのプレイがいいでしょう。
