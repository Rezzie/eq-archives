Bazaar Commands

 

General Commands    Emotes    Pet Commands    Guild Commands    Bazaar Commands

 

 

There are two commands used in the Bazaar /bazaar and /trader :

 

/bazaar (WELCOME, SEARCH, LIST and HELP) - For additional information about commands see below.

 

/bazaar welcome - Returns the following text :


Welcome to The Bazaar!
There are currently X traders and Y items for trade.
For a list of commands, type /bazaar help or /trader help.

 

/bazaar search – Returns all items for sale in the bazaar at that moment, the seller's name, the item name, the price and any comments about the items. This list will often truncated due to the length.

/bazaar search <player name> - Returns the name of the character, the items he has for sale, the price and any comments about the items.

/bazaar search <item name> - Returns all instances of that item for sale, the character selling them, the price and any comments about that item.


Note : That the item name can be a partial match.


/bazaar search <race> <class> <slot> <statistic>

 

/bazaar search race - Returns a list of the possible race options for use with the search command.


 /bazaar search class - Returns a list of the possible class options for use with the search command.


/bazaar search slot - Returns a list of the possible slot options for use with the search command.


 /bazaar search stat - Returns a list of the possible stat options for use with the search command.


All values are interchangeable and only one of the values are required.


For Example : 

 

/bazaar search BAR WAR NECK STR - Returns all instances of items that a Barbarian Warrior can wear in the Neck slot and has a bonus for Strength, the price and any comments about the items.


/bazaar list - Returns a list of all merchants in the bazaar and how many items they have for sale.

 

/bazaar help - Returns a list of commands for both /bazaar and /trader :

Format : /bazaar, Commands: welcome, search, list, help.
Format : /trader, Commands: edit, delete, new, quit, list, help, add, welcome.
For additional information about commands, please purchase the appropriate documentation located in the central building.

 /trader (WELCOME, EDIT, DELETE, NEW, QUIT, LIST, ADD and HELP) - For additional information about commands see below.

 

/trader welcome - Returns the following text :


Welcome to The Bazaar!
There are currently X traders and Y items for trade.
For a list of commands, type /bazaar help or /trader help.

 

/trader help - Returns a list of commands for both /bazaar and /trader :

Format : /bazaar, Commands: welcome, search, list, help.
Format : /trader, Commands: edit, delete, new, quit, list, help, add, welcome.
For additional information about commands, please purchase the appropriate documentation located in the central building.


/trader list - Returns a list of all the items that you have for sale, including the item's name, the price and any comments about the items.


 /trader new - Places you on the list of merchants in the zone.

 

Note : This command will not work if you do not have a Trader's Satchel containing items.

 

Note : This command will also remove all items, prices and comments from your wares list.


/trader quit - Removes you from the list of traders and removes all your items, prices and comments from your wares list.


 /trader add - Adds the item on your cursor to your list of items you have for sale.


 /trader edit <price> <comment> - Allows you to edit the price and comments for the item on your cursor.

 

Note : The item must already be on your list of wares to edit the price or comment.

 

Note : The <price> must entered in copper pieces.


For Example :

 

/trader edit 9000000 Rare Spell


This would set the price to 9000pp and add the description “Rare Spell”.

/trader delete - Removes the item on your cursor from your wares list.


Note : You must have an item on your cursor to add, delete, or edit your wares list. It is also a good idea to use the /trader list command and check your list of wares after making changes.
