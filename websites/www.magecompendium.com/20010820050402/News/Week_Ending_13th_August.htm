This came of the official EverQuest message boards.

I have read several times that there are hard coding limits that prevent:

1. Stacking more than 20 items in one slot
2. Containers with more than 10 items

It's a thought, we'll continue working on this issue.

Topic :  More New Features

 

This came of the official EverQuest message boards.

 

Hello all,

I thought I'd drop by and let you know of a new feature that we are trying out on Test. Last week it was suggested here on the boards that we implement a "Looking for Group" flag that players can set and view in order to help people get together quickly, allowing more time for adventuring. We've decided to try it out and it's on Test now.

The /lfg [on|off] command will set or unset your "LFG" flag. Players can then use all of the "/who" and "/who all" filters to find each others. For instance, "/who all rogue 45 50 lfg" will return all rogues between 45 and 50 who are looking for a group. "/who lfg" will
return all players in the current zone who are looking for a group.

Your LFG flag will automatically unset when you enter a group. One final note, though anonymous and roleplaying characters can set their LFG flag, they will not show up in filtered /who lists. In other words, the first example above wouldn't return any anonymous rogues, however "/who all lfg" would return everyone in the game who is LFG
(subject to the 20 person limit).

Assuming all goes well in testing, this change will most likely be in the next patch.

- Gordon

Topic :  Religion Change

 

This came of the official EverQuest message boards.

 

Ok is it possible to change your deity ?


If it isn't... WHY NOT ?

No.

Deity selections are designed to have positive and negative effects upon what a character can do in game and how the NPCs react to them. We want that change to be permanent, as we don't want to see a situation where people switch deities based upon what is convenient at the time. For instance, if a really cool deity-specific quest or item comes in game, we don't want to see people flocking to a new religion.

As it was established that a change in religion would not be allowed, the game architecture was built in a way that now makes such a change impossible. Your faction (which comes from your race, class, and deity) is determined when you first start your character and can't be changed on the fly.

 

- Gordon

Topic :  Activating Hot Keyed Items

 

This came of the official EverQuest message boards.

 

Pressing a key for something in a hotbox is essentially the same as left-clicking on it. That's why it works for spell gems. Activatable items require a right-click in order to be activated. If a right-click were applied to a spell gem, you would forget the spell.

That said, the idea for the key to operate as a right-click for activatable items is a good one, and I'll forward it to the programmer who's handling UI changes.

 

- Gordon

August 9th, 7:00am

Guide Application

The Guide Application is now back up at http://guideapp.everquest.com. If you are over 18 years of age, have a good customer-oriented attitude, enjoy helping your fellow player, and are interested in joining the Guide Program, we will be pleased to review your application.

Kunark Users

A new file, emeraldjungle_chr.s3d, is included as part of today's patch. This patch adds additional displayable NPCs to the Emerald Jungle. Please run the "Kunark Patch" by pressing the "Kunark Patch" button on the bottom of the patch program UI prior to entering EverQuest.

Patch Day

Greetings everyone! Today's patch contains some exciting new changes. 

General Enhancements

- Text messages that wrap will be indented on subsequent lines. This means that if a person or NPC says something that is more than one line long, all subsequent lines will be indented a few spaces. This should make it a lot easier to pick out separate messages.


- The spells "United Resolve" and "Heroic Bond" had their levels transposed accidentally. Rather than putting them back as intended, United Resolve has been changed to Heroic Bond, which means you might have two copies of that spell in your spell book. Feel free to move one to the back. Clerics get a new group spell at 58 to compensate.


- The Wurmslayer is now a one-handed slashing weapon.


- The /help command will now list all /-commands available to you.


- Rangers, Paladins, and SKs now have mana equivalent to that of a pure caster with the same stats. This will give those hybrids a good deal more mana, regardless of their stats.


- NORENT items, items that disappear when you log out, will now show a "NORENT" tag when inspecting the item.


- Most summoned items now have the word "summoned" in their name.

Changes/Fixes

- Small Studded Tunic recipe now works


- Scaler recipe now works


- High Elves can now use the Moss Covered Twig and Branch


- Iksar can now wear the Crescent headband.


- The Shintl Lowbrew quest will now work.


- The Rusty Rapier now looks like a rapier.


- Elven Wine can now be brewed


- Some pelt-dropping animals were always dropping ruined pelts, rather than pelts of quality. This has been repaired.


- The Guild-invite toggle in display options will now work.


- The faction problems with the Charasis Key Quest have been resolved.
- Many pathing errors have been repaired.


- Repaired a bug that caused some anomalies after being charmed or feared.


- Repaired a bug that kept the Unswerving Hammer of Faith and Flaming Sword of Xuzl from disappearing.

Dragons ?

Rumor has it that some new dragons have been spotted in Kunark. It should be noted however that the dragons can be engaged from much further away than is intended. Rather than hold off implementing these dragons for another week, we've decided to implement them now and fix the problem in the next patch. Enjoy :)


New Command For Tracking

New option for those with tracking: /trackplayers. The command will work for all players with the tracking skill. The command has two settings:

- ON: Default setting, same as it is now. Players will show up in the tracking window.


- OFF: Players will not show up in the tracking window.


Client Customizations

Upon starting EverQuest, a new file (EQCLIENT.INI) will be created in your EverQuest directory. This file contains customizable settings for:

- Colors for text messages in the in-game chat window


- Default screen viewing mode


- Default travel speed (run or walk)

The documentation will be included in the online manual in your EverQuest directory once the design is finalized over the next several weeks, but the working document is posted below. We felt that people would rather have access to the customizations right away while we're in the process of building them. 

Modifying this file is optional.

EQCLIENT.INI documentation

The [TextColors] section contains entries for various text colors in the client. All colors are to be set with Red, Green, and Blue values, in decimal. Valid settings for each color range between 0 and 255. "Black" (0 red, 0 green, and 0 blue) will be white in overlay mode, and black in normal mode.
