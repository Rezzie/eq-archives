Week Ending 10th June

 

 

Date :   07/06/2001

 

Topic :  New Test Server

 

June 7th, 6:30 pm

Sullon Zek is now available as a Test server.

This is an experimental PvP server based on a deity teams concept. To see the rules for this server and for more details look on our Developer's Corner message board at http://www.everquest.com (look for a post titled: Deity Team Server Announcement) as well as on the web site itself. Please read these rules before playing on the new server. To access this server run testeverquest.exe (which can be found in your EverQuest directory).

- The EverQuest Team.
