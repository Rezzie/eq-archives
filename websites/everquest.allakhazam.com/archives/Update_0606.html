June 13, 2006

A large patch was implemented this morning. The patch message has been posted on our news board here (lots of updates, it is recommended reading).

With this patch comes the requirement of DirectX 9.0c. Be advised there are two versions of DirectX 9.0c: If you have an older version, you must upgrade to the April 2006 version in order to play EverQuest. This is the redistributable version found here (for use after you have installed the original DirectX 9.0c).

The file named "DirectXTest.exe" in your EverQuest directory can be run in order to determine whether or not you are running the proper version.

Also be advised that if you are running an illegal version of windows, you may encounter problems with said upgrade.
