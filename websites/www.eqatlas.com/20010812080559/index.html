ZoneAlarm Clarification...
Sunday, August 5th

There are some people who haven't figured out how to get EQ to work well with ZoneAlarm. First step that you need to do is to disable the popup feature of ZoneAlarm, under the Alerts tab. This makes a small window pop up alerting you of someone hacking into your computer, but with EQ running will cause it to minimize.

The other thing that you need to be aware of is that the new version of ZoneAlarm warns you if an updated version of the file is trying to access the internet. This is most noticeable on patch day, when your updated EQ (or other game) tries to access the internet and a popup happens. In EQ the popup will appear early enough that you can still click on it, but if for some reason it doesn't, check the list of applications and make sure the application you're using has internet access prior to starting it, if the ZoneAlarm checkbox appears beneath an always-on-top application, it may seem as if you've locked up, but you just need to click a button you can't reach...

SirCam Virus
Thursday, August 2nd

Please! Make sure that you don't have this virus on your system. On Tuesday I downloaded 300 MB worth of this virus, and I'm now just deleting everything with an attachment on the server.

A little education for all on how it works. It is downloaded in an email with the text of "Hello. I send this to you to get your advice." and then some other stuff. There is an attachment, which looks like a normal file, but has an additional extension on the end. I've seen .pif, .bat, .exe, .com and probably a few more different extensions. Opening this attachment will infect your computer. The virus then inserts itself so that it is run every time you run another executable, hiding itself in your recycling bin. Every time you run it, it emails a file from your My Documents folder to a recipient found in your cache, using its own email server to do so. Kind of fancy-like.

A few ways to make sure you don't have it. First, update your virus software. They've all had major updates in the past week, so if you haven't done it that recently, you're overdue.

Second, there is a utility that will remove the SirCam virus if you have it on your computer. It's located at http://www.symantec.com/avcenter/venc/data/w32.sircam.worm@mm.removal.tool.html. I recommend you go to this link and follow the instructions to get it to work. I had to use it when I accidentally hit Enter instead of Shift when I was deleting dozens of these things. And the virus will hop across your network in the blink of an eye. It is quite a pain in the butt to remove by hand, so I recommend using the utility.

Last, but not least, you should all install a program like ZoneAlarm (yes this is an advertisement, but I put it up because I love the product). Note that there is a free version for personal use (regular ZoneAlarm, not ZoneAlarm Pro). This is a handy personal firewall that everybody with a persistent online connection (like, for instance, if you play online games for hours on end) should have set up on their computer. Not only does this prevent hackers from using known exploits to get onto your computer (I get about 5 attacks a day, and that's on the safe side of a firewall), but it makes sure that programs on your computer don't access the internet without your approval (take that Microsoft!!). I found out SirCam had infected my other computer when a ZoneAlarm window popped up asking me if I gave SirCam permission to access the internet.

If anyone cares, I've been working on a large statement about what's going on with these sites, which hopefully I'll get up this weekend.
