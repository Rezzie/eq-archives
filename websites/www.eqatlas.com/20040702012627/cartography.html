Cartography Tool Maps (LoY Expansion)

 

Instructions for how to use these maps:

  1. You need the Legacy of Ykesha (or any later) expansion. You can either download this or purchase the CD version.
  2. Download one or more of the following zipped files.
  3. Unzip the contents into the /maps folder within your Everquest installation.
  4. In game, they will then come up inside the cartography window. Note that some require special parchments in order to be able to view.

Some basic information about the files:

  • They are named in the format "zonename_#.txt" with the # being 1, 2, or 3. There is also a base layer one without the number. These correspond to the layers that you see at the right of your cartography window.
  • Since they are text files, it doesn't really matter what they are named, you could change the names, as long as you follow the format, to make them show up in different layers if you wished.
  • You can also "merge" maps by copying the contents of one file into another. An example would be if you found a three-layered map and wanted to have all of that in one layer, you could just copy all the other files into one, named for the appropriate layer.
  • You can map into different layers if you are mapping. So if you have a layer 1 map, and map into layer 2, the layer 1 map doesn't get altered. Conversely, if you put a layer 1 map in and map onto layer 1, this will alter the map.
  • Currently there is no editing tool to remove things from these maps. Hopefully someone will make one soon, but just be careful of what you add.
