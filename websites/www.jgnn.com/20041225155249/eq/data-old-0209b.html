【EVERQUEST CASTERS REALM】 - New Alternate Advancement abilities
2002年09月25日 03時22分41秒

Skillの一部（Class不明で殆どがLv6y1からのもの)

- Planar Power : ItemかSpellのyour statisticsの最大値を5point毎に引き上げる事ができる。
- Planar Durability : HP最大値が5% up
- Advanced Innate STR : Status 2point up ( STA,INT,AGI,DEX,WIS,CHAもあり )
- Innate Enlightment : medi up
- Marr's Protection : Sv Magic up
- Warding of Solusek : Sv Fire up
- Blessing of E'ci : Sv Cold up
- Shroud of The Faceless : Sv Poison up
- Bertoxxulous' Gift : Sv Disease up
- Advanced Spell Casting Mastery : specialization checks chance 5% up
- Advanced Healing Adept : Heal effect 5% up
- Advanced Healing Gift : exceptional heal chance up
- Hastened ( any spell ) : 各種spellのrecast timeを10%短縮（？）

etc
