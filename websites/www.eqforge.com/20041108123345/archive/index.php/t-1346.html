What does an objective city or other objective thing do???
BeBro
Apr 06, 2001, 12:42 AM
Objective cities are the main targets in many scenarios. You get points for each captured objective city, and you need a certain number of these points (how much exactly can be specified by the scenario creator) to win the scenario. You can also specify if wonders should count as objectives...http://forums.civfanatics.com/ubb/wink.gif

BTW, under MGE only one-point objective cities work well, three-point objectives are not correctly counted, so should be avoided there. Three-point objectives work only under FW.
