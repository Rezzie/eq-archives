Not a very big poll this time since there have not been too many turns since the 100 BC polls. This one mainly revolves around improving terrain, especially near Dou and on building more settlers to expand since the knowledge we have of the AI civs through MPE shows that we will get Cope without difficulties and Mike as well unless something highly disastrous happens.

I think we should build a courthouse next in Dou. This improves science, trade and production by reducing corruption while a marketplace only increase trade. I think we should finish it before starting a wonder - we have enough caravans (and time because we are ahead of the AI civs in science) to finish Cope after we build the courthose. The courthose should probably be rush bought when we discover astronomy. To increase production (meaning we need less $$ to rush buy) I strongly recommend switching to silk in Dou. This also increase trade. Food production is slightly less but not significantly when taking into account roughly how many turns from now we will become a republic and grow Dou using WLTP.

The settler in option D would build the road between EF and LP and then probably head north to found C5, C6 or C7, possibly first building road from EF to FW.

Option E reflects the fact that we really need an attacking unit in the east in case enemies show up.

It's possible that we should try to find time/opportunity to build another elephant and/or diplomat in the west but I'm not including it in the poll this time, it probably wouldn't be finished before the next poll anyway.

The F options ask what to do with the settlers. I think we should improve the terrain a little before founding new cities so the cities can easily grow with WLTP once we become a republic. Once we are close to finishing building new settlers we can start founding cities.

I'm not recommending any changes to science/tax/luxuries and probably will not do so until we have discovered monotheism (and maybe republic as well).

If you disagree with something or have any suggestions please indicate so in this thread.
