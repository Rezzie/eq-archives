You are probably just replacing the graphic in the wrong place or the wrong file. Try putting a big red dot on the existing glacier graphic and see if it appears in the scenario first.

You cannot copy terrain.gif to paint and then use it (either as bmp or gif) because the palettes are wrong. If you want to use paint, get an existing bmp terrain file from somebody elses scenario and use it as a template in paint first.

If I am not mistaken, the Ultima scenario uses airfields to do the small hamlets.
