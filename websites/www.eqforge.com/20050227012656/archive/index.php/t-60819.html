It has been suggested that we should build a canal city in Hell. Two locations have been suggested, between Knossos/Corinth/Delphi (C1 and C2) and between Athens/Sparta (C3).

Also the Ministry of Domestic Affairs (MoDA) wants to start constructing a road/railroad from Hell to Barbaria ASAP. This is a project that takes a lot of time so the sooner we start the better. The MoDA suggests dedicating one engineer to this taks immediately and more once we have built some infrastructure (railroads, irrigation) in Hell. Also the engineer should be escorted by a diplomat and cavalry because we could run into hostile troops or even cities. The diplomat/cavalry could also explore this area to check for the presence of French cities. Starting by constructing a road seems to make sense and to build railroad later when we have more engineers for this task. Needless to say we would later found cities in this area unless there are French cities there.
http://www.civfanatics.net/uploads5/dc1752ad_hell.jpg

It has been pointed out that it would be useful to have a coastal city in southern Africa. To this end, the city of Bapedi could be relocated, it is of size 1 so it could immediately rush buy an engineer.
http://www.civfanatics.net/uploads5/dc1752ad_africa.jpg

Finally, we need a name for the area N/NW of China. Since in reality Siberia is N/NW of China we could use that name but I suggest replacing the first two letters and calling this province Cyberia. We could then found cities there and use names like Internet, World Wide Web, Dotcom and even Crash :D. More suggestions are welcome, both for naming this province and for naming future cities there.
