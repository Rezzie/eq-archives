Check out the reference material in the downloads section. There are some really good docs going into all of this.

Basically, you just use the ingame editor (with MGE or FW) or edit the things directly into your "rules.txt" file. It's all in there... Be sure to make some safe copies first, though... It's almost self-explanatory, but be sure to read some of the excellent docs first, to save you from too many headaches.....

Second, how do I add things in the civiliopedia? I don't want to change the existing things, but add new things.

If you have the classic version, you should remove "getinfo.exe" from your "pedia" folder. This removes the "bogus" pedia program, and the game will get pedia-information directly from your changed files. If you have another version of the game, there should be no problems viewing your changes in the pedia.

If you want to add text to the existing texts, you should do it in one of the appropriate .txt-files, named pedia.txt or something like this.

Third, anyone know if it's possible to create an air/transport unit. Like a big zeppelin, which can transport units. Is it possible?

Well, the best idea is to change the ocean into land in the graphics departement, and then change the transports into zeppeliners, but this has obvious disadvantages! http://forums.civfanatics.com/ubb/wink.gif Otherwise I know of no ways of making land or air units "carry" other units.

But do check out the docs! There might be some other ways to tweak the game mechanics, so as to achieve the effects you want.
