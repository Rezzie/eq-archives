Whenever the Domination Victory is enabled, everybody should be aware of it and act accordingly. If you want another victory type just don´t expand as much towards the limit.:p

But never EVER reload, this is one of the main rules of the GOTM. Reloading takes all the fun of learning from mistakes from the game and so it isn´t allowed under whatever gameplay circumstances (except crashes of course ;) ).

The new random seed event calculator in Civ3 was especially designed by Firaxis to prevent reloading.

I´d like to have had a conquest victory myself in GOTM, got a domination victory, too.:(

So NO reloading!
