You could try an IDE to USB adapter. :yeah:
If you have an external harddrive, the adapter cable is all you need.

If it´s an internal drive you need a so called external enclosure, mainly for the power supply.
For example here:
http://www.pcwebshopper.com/5213fir20com.html

You put the HDD into it and connect the device via USB to the laptop.

If you have an SCSI harddrive, you might not need anything at all (except power supply). Several notebooks have an external SCSI port.
