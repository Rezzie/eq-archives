We have 18 size 3+ cities and well over half of these can grow if we continue celebrating. However, food shortage will soon prevent them from continuing to grow so we should celebrate for a maximum of two more turns. This is option A2 - there the decision whether to celebrate one or two turns would be made by the president depending on the situation one turn from now.

When we stop celebrating luxuries are lowered to 20% (a lower value results in civil disorder).

There is also a lot of things we want to accomplish, science discoveries to be made, we need many new engineers, barracks and military units to start peacekeeping operations in Hell, libraries in the big cities, more banks, supermarkets when we discover refrigeration etc.

There are some important techs we will soon discover. From the science ministry's recommended tech path we still need the following (in this order):

Leadership, Electricity, Conscription, Tactics, Refrigeration

Refrigeration is extremely useful to make our cities grow and we have lots of cash so the Ministry of Domestic Affairs is recommending setting science=80% and taxes=0% until we have discovered all of these techs and probably steam engine as well. This enables us to make roughly one discovery per turn. We can afford to run a tax deficit for almost 20 turns if we do not buy anything, we have $1522 and an upcoming caravan delivery.

The different options result in the following amounts of $$ and discoveries:

B1: $1144/77, 6 turns
B2: $499/77, 2 turns
B3: $0/77, 1 turn

Once this is accomplished we'd collect cash by setting taxes to 70-80% for a few turns. This yields more than $1000 per turn.

Build queue polls will be posted tomorrow. It should be noted that the 'contents' of the build queues will not be affected by the results of this poll; only the speed with which they can be implemented.
