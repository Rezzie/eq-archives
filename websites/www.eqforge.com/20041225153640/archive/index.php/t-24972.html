The Problem
1. You cannot see the file details in MS Explorer.
2. You cannot see the extensions of your files in MS Explorer (like .zip, .exe, .txt, .dll, ...).

Description
As you use and manipulate computer files, the default settings of MS Explorer make it difficult to see file details, change file extensions.

Procedure
1. With Explorer, click on the pull down menu called "View", and select "Details". Here is what it looks like in Windoze 95 (others MS OS's are similar):

http://www.civfanatics.com/uploads/CFC_WinG_E-ViewDetails.gif

2. The above step will let you see all the juicy file details of your shortcuts and files.


3. Now, we just need to let Explorer show you the extension names (.EXE, .ZIP, .DOC, etc. etc. etc.). Go to the same pull down ("View"), and notice at the bottom there is an "Options".... just select it, and this (or something similar) will appear:

http://www.civfanatics.com/uploads/CFC_WinG_E-Options.gif

4. Notice the middle check box called "Hide MS DOS File Extensions...". If you remove the check from this box, you can then see the extension of a file.

5. These are all options that simply allow an individual to choose the way file information is displayed. Changing these options will not cause system damage, and you can always just reverse each option if you don't like it :)

Caveats
The screenshots are for Win9x, but should be close enough to other MS OS's for you to do the same thing.
