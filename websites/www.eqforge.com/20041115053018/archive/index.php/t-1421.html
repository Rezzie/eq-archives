Blitz Basic is a language for creating games.

- Compiled programs require no runtime dll.
- Compiles to machine code
- It is fast
- It uses directx
- Has various collision commands
- Has commands to play Wav,Mp3,Midi,XM/S3M/IT ea
- Supports BMP/JPG/PNG
- It's cheap
- 3D version is in the making
- Overal easy language

There are more basic languages for creating games but I think this one is the overal best.
