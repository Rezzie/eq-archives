Here is what I propose:

Noshuret:

Temple, Legion, Legion, settler

I would like this to be the last settler Noshuret produces for awhile, since its primary purpose is to build legions, which are significantly more expensive than warriors.

Gorina

Settler, Temple, Palace prebuild

I have removed the warrior from the queue because it will actually be a legion, which will take too long. See the Nar Shadaa queue for more information.

Nar Shadaa

warrior, warrior, worker, warrior

These two warriors should be sent south to garrison Gorina during its wonder prebuild to avoid unhappiness. If we decide to try to build a large military, it might be useful to keep this city from being connected to the iron, so that it can build warriors instead of legions.

Penguinadua

Granary, settler, settler
