The tax rate for the next several turns is to be decided in conjunction with a science plan (see this thread (http://forums.civfanatics.com/showthread.php?s=&threadid=56092)) and this plan also requires that we continue building caravans in most of our cities so this is a relatively short poll. It concerns our colonies, whether we should found cities more or less as we had decided (see maps in this thread (http://forums.civfanatics.com/showthread.php?s=&threadid=55342)). We should change C2's location slightly in response to better geographic knowledge. The poll also asks whether to rush buy a unit in ASAP in the new cities, especially those in Woodstock (I'll now start using these province names even though they are not official yet). This is strongly recommended because these cities are in hostile territory, we must be able to expel diplomats etc. The unit should be a horseman, crusader, phalanx or warrior, depending on $$ (preferably crusader).

Also the city names "Electrical" and "Superstorms" popped up as a joke in a discussion thread and I immediately liked them :D. So I'm also asking whether to add these names to our list of city names we will use (the previous city name poll can be found here (http://forums.civfanatics.com/showthread.php?s=&threadid=53373)).

Finally, to get rid of unhappiness in democracy we should do something about the elephant near Ipec, either move it to Ipec or build a fortress. Building a fortress looks to me like the better option since the current location of the elephant is very central.

Only vote for C1 or C2 if you vore for B1.
