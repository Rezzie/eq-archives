The download file structure is very complicated and not handled properly in the readme. My suggestions regarding this are:

1. The game.txt can be used from the scenario folder and doesn't need to be in the Main. 3. What are the xxx4 files?

5. There are no instructions regarding the FW scenario file and as far as I know, the describe.txt will not work for FW which will rely on the pedia.txt.

The last time I played an Ultima game was on an Apple IIe so it would be nice to have some background to what is going on and what you are supposed to do (kill every other race?). 2. I am very impressed with the sounds.

3. I am very impressed with the map and terrain.

4. I am very impressed with the units, mostly your own creations?

TEXT
^Lord British dies in battle.

You need lots more messages to move the game along and give some flavour to the game. When your events.txt is about 10-15k, that would make the game perfect.
