The biggest decisions now are IMO the following:

(1) At the moment Dou does not need to build wonders (we have built all the 'SSC wonders' we have techs for at the moment). This means it can build imporvements or caravans. I recommend caravans and establishing trade routes ASAP using domestic trading. We can send caravans abroad later. Since Dou already has lots of improvements and the relevant wonders there's a lot to be gained from trade routes.

(2) Should we strengthen out defenses by building several crusaders ? I think we need to, especially in the east where the Greeks are nearby. This also strengthens our chances of getting tribute once we found cities on other foreign continents. This delays Mike a little but our defenses are very weak now. Possible tribute would compensate.

(3) Improving land. I'm including a map showing the most important roads (R) and irrigation (W) that we need. I'm not including any roads that become possible with bridge building.

(4) Where to build Mike (and we need to start/finish it ASAP). Neu is probably the best option, it already has MPE and building Mike there means there will be fewer critical cities that need strong defense.

Hopefully it's obvious that you should only vote for one of the C's if you vote for A1.
