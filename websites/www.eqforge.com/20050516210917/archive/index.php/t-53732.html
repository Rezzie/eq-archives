Note: You can find a detailed description in the PBEM forum.

ok, on IRC you go to:

File-->Options

Select Connect (left side).
Click Add.

Type this in these fields:

Description: CFC Demogame
IRC Server: irc.irc-chat.net
Ports: 6667

hit add, then hit ok


Then, hit the connect button (in mIRC/DarkStyle Pro 2, it's a lightning bolt)

To join a channel, type in:

/join #demogame
or
/j demogame

We're also in #civfanatics when we're not at the turn chat.

_________________________

For those who don't have/can't use mIRC or DSP2:

You can access the java chat at the bottom of the forum. (an alternative is through the main site's page's chat link at the top). Click on the "Chat Room" link at the bottom of any thread or forum.

http://www.civfanatics.net/uploads4/JAVACHAT.jpg

Setup your user name, and what room you want to go in. Don't worry about the server - the default is irc.irc-chat.net.

http://www.civfanatics.net/uploads4/JC_SETUP.jpg

This is the chatroom. If you want to go to another room, simply repeat the first step, and open another window to go into a room like #civfanatics or #demogame_RPG

http://www.civfanatics.net/uploads4/JC_ROOM.jpg

Notice that Iris and Mafisio are bots. Iris was added by one of the IRC admins of that chat server. They're Civ3 fans, too. :D
