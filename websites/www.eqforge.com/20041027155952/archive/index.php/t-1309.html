Presumably what you are talking about is a modpack. There is no practical difference between a scenario and a mod pack for your purposes other than the fact that there is no starting map.

My first sugestion is to make a scenario instead of a mod pack first. Then you can use the scenario tools plus if you make a mistake, you will be able to fix it.

When you have finished, just copy the entire contents of the scenario file into the main folder and overwrite the original civ files.

There are a few other things you can fiddle with that you can't in a scenario, like the launch menu screen and the labels files. You can leave these to the last after you are satisfied with your handiwork.

p.s. There is a very recent thread on the finer points about the difference between a mod and a scenario which you can look up.

The date stuff is stored in the Games.txt if I am not mistaken but again the simplest way to do this is to change the parameters in the scenario and overwrite the original games.txt.

The second question refers to the tech tree which you can edit from the rules.txt. Again you can use the scenario tools or edit the rules.txt directly (prefer the former).
