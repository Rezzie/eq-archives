Hi engineer, many civ2 scenarios contain a file named events.txt. In this file you can define several actions that should happen during your scenario (comparable to the triggers in Starcraft or Age of Kings).

Therefore you can use the events macro language, simple commands that allow you things like, disabling diplomacy in a scenario, create units at a special turn and in special locations on the map, give techs to a certain civ and so on.

Simply open an existing events.txt of another scenario and look into it, then you´ll find out a lot more. If you have the FW or MGE version, you can simply use its events editor to create and edit the events.txt for your scenario. It has also a "Help" button, which leads you to detailed info about the single commands.

Many players prefer instead a simple text editor like notepad for editing the events.
