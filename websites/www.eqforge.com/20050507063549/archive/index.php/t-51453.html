I propose that for DG3, it should be required to name all files in the following way:


For Demogame III files

DG3_<game year>_<discriptor>.jpg
or
DG3_<discriptor>.jpg


For RPG files

RP3_<game year>_<discriptor>.jpg
or
RP3_<discriptor>.jpg


DG3 - This is to denote that all files are from Demogame 3. Why? We have 2 demogames already, and some demogame files are in the uploads4 directory. This is to prevent any duplicates, and accidently showing an image from DG2.

<game year> - This is so we can group the game year. If we were to look through the file directory for some reason, we could easily find them.

<discriptor> - This tells the file info, such as province_plan1, province_plan2, wm_sm.

It would look something like this:

DG3_AD0250_OperationZealousZulu.jpg
DG3_BC4000_cityscreen.jpg
DG3_BC4000_Province1.jpg
DG3_BC4000_start.jpg
DG3_BC4000_wm.jpg
RP3_VolcanoThread_Volcano.jpg
RP3_Despot_Palace.jpg
