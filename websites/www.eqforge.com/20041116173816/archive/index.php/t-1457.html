The objective system works in the way, that you can assign cities to be objective cities, which means players get points for conquering these. You can toggle wonders to count also as objectives, so that a certain city can count for more points, if it has wonders in it. You do it by right clicking on a city and pressing CTRL + C (or edit city via the Cheat menu), and then assign it as objective.

Then you set up the victory conditions, which basically is how many points a player needs to "win" the game in the given time.
