This poll is intended to prioritize the wonders we can build either now or not too far from now; please vote for 4 wonders from the list. I'm deliberately omitting Michelangelo's Chapel from the list since it had already been decided earlier on to build it.

Of these wonders we already have the techs to build GL and Oracle in addition to Mike but I recommend not building them; I included them for completeness only. The AI is building both of these (the Zulus Oracle and the Aztecs, French and Indians the GL).

Of these Isaac Newton's College must be built in our SSC if we build it; I recommend doing so. Shakespeare should either be built in the SSC or not at all. I strongly recommend aiming for it since it eliminates any unhappiness problems in the SSC once and for all.

IMO Bach and Leo would also be very nice to have; they can be built anywhere we please. Magellan might also be useful, however a few AI civs are not very far from us.
