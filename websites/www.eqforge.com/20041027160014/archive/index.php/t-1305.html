The problem lies in the type of gif file that is required. In the simplest terms:

1. Open any unit.gif file icon that come with your Civ package with paint and save it as units.bmp. Civ will recognise the bmp file more easily as there is only one kind of bmp format. Saving in gif format from most programs (with the exception of paintshop pro and a few others) will result in a crash as they are palette and format issues.

2. From now on, work exclusively in paint and save only in bmp format (256 colours).

3. This solution does not require you to download anything, learn anything or transfer anything from one program to another.
