Morten Blaabjerg
May 22, 2001, 08:31 PM
There are some excellent documents in the downloads section that can really give you a head start. Personally I would (depending on the idea) start with making a good map with the map editor -which could be interesting to play on. Then I would look out for alternate graphics to suit the style of the game, and start building the scenario via the Cheat Menu (and possibly the Events Menu).

-Also, the RULES.TXT file is the key to editing the scenario rules. The file can be changed with Notepad, as can all the other txt-files. The graphics in the Gif can be changed with an image processor like Paintshop Pro.

From there on, there's a whole world of opportunities! I really recommend you to check out the docs in the downloads section. There are many good articles, also about getting started.

Morten

Zorak_zoran, you can also find a chart of the existing tech tree somewhere -if not in the downloads section, then possibly on some other Civ2 site. -the most difficult part in my opinion is altering the tech tree -and make it work properly. -it's not at all necessary to make a scenario -but it does give you the possibility to tailor-adapt the game to your settings.

But the best article -and the longest, most comprehensive without doubt- is Leon Marricks "Adv Scenario Making" -check out also Kobayashis pioneer work with experimenting with different unit types -i found that very inspiring.

Peterval, you can find the downloads section
here :
http://www.civfanatics.com/civ2downloads.shtml

Events.txt are only used with the Fantastic Worlds or Multiplayer Gold Editions of Civ2 -or with all versions except the classic version of Civ2.

If you only have the classic version, you can download a patch to convert it to MGE, and be able to play almost all the scenarios available. -same link as above.
