Originally posted by Circee:
I use MGE the only way i found to use it is to place it in a folder "modpack", just next to the folder "scenarios", to start a game, save it and put it in...

That's the only way to do it (that I know <IMG SRC="http://forums.civfanatics.com/ubb/wink.gif" border=0>) When you load a saved game, Civ 2 (MGE) tries to load the graphics and rules from the same directory.....if it can't then it loads them from the original files.

Now that you know that.....you can see why you can't start MODS right from the start.
What.....you can't..... <IMG SRC="http://forums.civfanatics.com/ubb/wink.gif" border=0>
Ok then. When Civ 2 starts a new game it starts, and creates the autosave in the root directory. This means that it will always use the default rules and graphics, therefore forcing you to save it and then move it to the MOD directory.

It you always want to play with the MOD then you should do what JValdezToo suggested. Copy your Civ 2 directory and replace all the original files with the MOD files. Then when you start a new game it will automatically load the MODified rules/graphics.

Hope this helps. <IMG SRC="http://forums.civfanatics.com/ubb/smile.gif" border=0>
