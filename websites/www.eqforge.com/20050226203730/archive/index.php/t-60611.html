Africa Status Report 1752 AD

The infrastructure is overall much better here than in Hell, in particular we already have a rail network.

Bapedi: Temple, marketplace. Needs irrigation on the shielded grassland square SW of the city ASAP.
Zimbabwe: Engineer, harbor, temple, aqueduct, marketplace
Ulundi: Needs irrigation. Temple, marketplace, aqueduct. Engineers later.
Hlobane: Shouldn't build engineer until it is a size 4+ city. So build temple, marketplace. Maybe cavalry if we want military units.
Intombe: Can become big. Engineer, temple, aqueduct, marketplace.
Isandhlwana: Can become big. Engineer, temple, aqueduct. Railroad needed to Intombe ASAP. May want to move cavalry west if we decide to attack the Indians, otherwise it's useful here to attack the Aztecs. Rehome at least one cavalry (not the one from Dou !!) here.

The generic build queue looks like the Hell queue: Harbor, temple, marketplace, aqueduct but the list above only includes what's needed ASAP. It would be nice to focus on getting one big city here, probably Intombe. This means rush buying lots of stuff there.

One important question is if we should build more cavalry, especially if we decide to attack the Indians.

Input please !
