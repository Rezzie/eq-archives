GOTM Global Rankings Formula (courtesy of Taé Shala in an earlier thread)

(PlayerScore / AverageScore) * 25 = GameRanking

Best 3 of 5 GameRankings / 3 = GlobalRanking

GOTM 4 average: 3332
GOTM 3 average: 3039
GOTM 2 average: 3596
GOTM 1 average: 2405

DirtyClint's ranking:

6132/2405 * 25 = 63.7

15445/3596 * 25 = 107.4 (you should be on top till GOTM 8 at least because of this score :lol: )

5866/3039 * 25 = 48.3

(63.7 + 107.4 + 48.3) / 3 = 73.1


SirPleb's ranking:

6015/3332 * 25 = 45.1

13692/3596 * 25 = 95.2

7883/3039 * 25 = 68.8

(45.1 + 95.2 + 68.8) / 3 = 69.7

I guess I've been doing rounding a bit differently than Matrix does (off by about 2 for each ranking), but you get the point ;)
