Rate of fire means you get to fire more shells (which means more damages) and not more turn.

The only time a unit can shoot/attack more than once is when a unit has the blizt capability checked in the rules file. Currently only Panzer and Modern Armor has that ability in the original mod file. By blizt, it means the unit get to attack 3 times if it has 3 movement points (if it didn't die in the first few attempt :D )
