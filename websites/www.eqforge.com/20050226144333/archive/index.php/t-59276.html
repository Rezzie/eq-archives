There are two wonders we can build now, Eiffel Tower and Darwin's Voyage. Once we discover Industrialization (which we are going to delay a bit) Women's Suffrage becomes available.

We can easily build all wonders ahead of the AI if we want to. Eiffel Tower is useless (especially in this game since we must finish with a spotless reputation anyway) so we would only build it 'for fun'. Darwin is useful to prevent the Indians from getting extra techs and it is not expensive. We can build it and, as a consequence, immediately discover Atomic Theory and Fundy and end up researching Industrialization.

The purpose of this poll is to check which of these wonders to build. It's not about their prioritization; we are economically overwhelmingly powerful and can easily build all of them ahead of the AI.

So vote for none, 1, 2 or all of these wonders.
