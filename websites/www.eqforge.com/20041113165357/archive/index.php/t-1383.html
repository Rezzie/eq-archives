Some time ago Mercator made a very good Earth map. He made the map in 4 parts each about the max allowed Civ2 map size (then there were no ideas about Gigamaps yet).
As I remember he didn't finished the maps, everything was grassland, but it was in high quality.
You can use his Mapeditor to make these 4 maps to one Gigamap and then add terrain.
It is good in scenario making if you start from all-grassland map because then you can decide yourself which terrain to use for what. For example you can use some terrain types for special resources (oilfields etc).
