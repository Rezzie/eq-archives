If you copied the new files to the civ2 main directory then civ2 should use them. After all you have over-written the old version there so the program can't be using a file which no longer exists. The only thing I can think of is that you might have a cities.bmp file in the civ2 directory, as Bitmaps take precedence over Gifs. But then again I thought only FW+ used bmp files.
