There is a non Tweakui way. Longer....and harder.....but you know....

Go to Control Panel
Go to Network
Go to Primary Windows Logon and change this option to Windows Logon
Press OK
Press No to the restart the computer question.
Go to Passwords
Go to Change Windows Password
Write in old password (if there was one)
Then press OK. (leaving everything else blank.)
Then press OK.
Then say Yes to reboot.
IF you get a new logon box, just press OK with out typing anything.
Then you won't get it again. :)

The DELL approved way.....as we were generally not allowed to install TweakUI on customers systems. ;)
