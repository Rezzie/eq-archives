Best of the Best Announced!!!
Land Squid | 8/14/2000 - 10:25 PM

Who Are The Greatest Champions Serverwide???

Starting August 27th, the new events launch with the first of the “Best of the Best” series, that will pit the greatest of the servers against the greatest of the servers. During this week all servers will run the Warriors Championship. Rules for this event are simple, with no artifacts, no buffs, and no right clicking on any item (but wearing activatable items is acceptable, just not the use of those items). These events will not have any entry fee, but players must be of 55th level or greater to enter. The winner of the event will be given a title of “The Champion” which is the 51st level title of the class. All of “The Champions” will be copied to Test Server, so that on Sunday at Noon (PST) all of the champions will face each other. Those that place (4th, 3rd, and 2nd places) will receive their 59th level title, and the winner of the “Main Event” will be dubbed the 60th level title (the Warlord).

Security for these events will be absolute, with Lake Rathetear shut down on Test Server for the duration of the event. The only characters that will be allowed in the Arena on Test Server will be the Events Staff, the Champions of each server, and possibly some members of the press who are covering the events. All others who come into the arena via gating in or having logged out there will be summoned out of the arena and binded in a far away land by a member of the event staff as soon as the player is discovered. There will be no exceptions to this.

After the event, all characters copied to test will be deleted, and titles will be given to the characters on their respective “live” server’s characters. This is to insure that the Test Server does not receive added players or items. Security measures will be in place to make sure that no items from the “live” servers end up in the hands of the Test Server. I know that there were concerns with this with the original announcement of bringing the champions to the Test Server, but rest assured that actions will be taken against those responsible IF something happens, not the server itself.

www.everquest.com will also carry all information and standings of the events for this great event.

At the conclusion of the warrior’s competition, a random roll will determine the next class that will have the excitement of this style event. The titles awarded again will be 51st level title for the winners of the server competition, the 59th level title for the runners up, and the 60th level title for the grand champion of that specific class Norrath-wide.

There will be a week of normal events that all levels and classes can enjoy, then the week of the new classes competitions start. This is so that all levels, races, and classes get to have fun with the events. When it comes to the game-wide championships, those will be dedicated only to the 55th level or higher players. One week dedicated to all levels, then a week dedicated to the class specific events that will determine the best of each class in the game.

More information should be released early next week, including official rules and regulations of the Warriors Event. The Events Staff and myself look forward to bringing you these new events.


Lady Daegarmo “The Faithful”
Online Events Coordinator
Verant Interactive

New Changes on Test
Land Squid | 8/10/2000 - 8:56 AM

08-09-2000 06:41 PM

Hello all,

I thought I'd drop by and let you know of a new feature that we are trying out on Test. Last week it was suggested here on the boards that we implement a "Looking for Group" flag that players can set and view in order to help people get together quickly, allowing more time for adventuring. We've decided to try it out and it's on Test now.

The /lfg [on|off] command will set or unset your "LFG" flag. Players can then use all of the "/who" and "/who all" filters to find each others. For instance, "/who all rogue 45 50 lfg" will return all rogues between 45 and 50 who are looking for a group. "/who lfg" will return all players in the current zone who are looking for a group.

Your LFG flag will automatically unset when you enter a group. One final note, though anonymous and roleplaying characters can set their LFG flag, they will not show up in filtered /who lists. In other words, the first example above wouldn't return any anonymous rogues, however "/who all lfg" would return everyone in the game who is LFG (subject to the 20 person limit).

Assuming all goes well in testing, this change will most likely be in the next patch.

-Gordon

New Patch Message
Land Squid | 8/9/2000 - 9:51 AM


***Character Purge***
Time has come to do some cleanup on character data across the servers. On August 23rd (Pacific), we will be doing a system-wide cleanup of level 1 characters. All level 1 characters that have that have not been played within 30 days from the date of the purge will be deleted. As GMs will not be able to restore any characters that are deleted, please be certain to log in any level 1 characters that you wish to keep prior to the purge.

- The EverQuest Team


August 9th, 7:00am
------------------

*Guide Application*

The Guide Application is now back up at http://guideapp.everquest.com. If you are over 18 years of age, have a good customer-oriented attitude, enjoy helping your fellow player, and are interested in joining the Guide Program, we will be pleased to review your application.

*Kunark Users*

A new file, emeraldjungle_chr.s3d, is included as part of todays patch. This patch adds additional displayable NPCs to the Emerald Jungle. Please run the "Kunark Patch" by pressing the "Kunark Patch" button on the bottom of the patch program UI prior to entering EverQuest.

*Patch Day*

Greetings everyone! Today's patch contains some exciting new changes.

*General Enhancements*

- Text messages that wrap will be indented on subsequent lines. This means that if a person or NPC says something that is more than one line long, all subsequent lines will be indented a few spaces. This should make it a lot easier to pick out separate messages.
- The spells "United Resolve" and "Heroic Bond" had their levels transposed accidentally. Rather than putting them back as intended, United Resolve has been changed to Heroic Bond, which means you might have two copies of that spell in your spellbook. Feel free to move one to the back. Clerics get a new group spell at 58 to compensate.
- The Wurmslayer is now a one-handed slashing weapon.
- The /help command will now list all /-commands available to you.
- Rangers, Paladins, and SKs now have mana equivalent to that of a pure caster with the same stats. This will give those hybrids a good deal more mana, regardless of their stats.
- NORENT items, items that disappear when you log out, will now show a "NORENT" tag when inspecting the item.
- Most summoned items now have the word "summoned" in their name.

*Changes/Fixes*

- Small Studded Tunic recipe now works
- Scaler recipe now works
- High Elves can now use the Moss Covered Twig and Branch
- Iksar can now wear the Crescent headband.
- The Shintl Lowbrew quest will now work
- The Rusty Rapier now looks like a rapier.
- Elven Wine can now be brewed
- Some pelt-dropping animals were always dropping ruined pelts, rather than pelts of quality. This has been repaired.
- The Guild-invite toggle in display options will now work.
- The faction problems with the Charasis Key Quest have been resolved.
- Many pathing errors have been repaired.
- Repaired a bug that caused some anomalies after being charmed or feared.
- Repaired a bug that kept the Unswerving Hammer of Faith and Flaming Sword of Xuzl from disappearing.

*Dragons?*

Rumor has it that some new dragons have been spotted in Kunark. It should be noted however that the dragons can be engaged from much further away than is intended. Rather than hold off implementing these dragons for another week, we've decided to implement them now and fix the problem in the next patch. Enjoy


*New Command For Tracking*

New option for those with tracking: /trackplayers. The command will work for all players with the tracking skill. The command has two settings:

- ON: Default setting, same as it is now. Players will show up in the tracking window.
- OFF: Players will not show up in the tracking window.

*Client Customizations*

Upon starting EverQuest, a new file (EQCLIENT.INI) will be created in your EverQuest directory. This file contains customizable settings for:

- Colors for text messages in the in-game chat window
- Default screen viewing mode
- Default travel speed (run or walk)

The documentation will be included in the online manual in your EverQuest directory once the design is finalized over the next several weeks, but the working document is posted below. We felt that people would rather have access to the customizations right away while we're in the process of building them.

Modifying this file is optional.

*EQCLIENT.INI documentation*

The [TextColors] section contains entries for various text colors in the client. All colors are to be set with Red, Green, and Blue values, in decimal. Valid settings for each color range between 0 and 255. "Black" (0 red, 0 green, and 0 blue) will be white in overlay mode, and black in normal mode.

Here is a list of user colors to use as a guide when editing the file:

1 - Say
2 - Tell
3 - Group
4 - Guild
5 - OOC
6 - Auction
7 - Shout
8 - Emote
9 - Spells (meming, scribing, casting, etc.)
10 - You hit other
11 - Other hits you
12 - You miss other
13 - Other misses you
14 - Some broadcasts (duels)
15 - Skills (ups, non-combat use, etc.)
18 - Default text and stuff you type
20 - Merchant Offer Price
21 - Merchant Buy/Sell
22 - Your death message
23 - Others death message
24 - Other damage other
25 - Other miss other
26 - /who command

The [Defaults] section contains some default setting for the client. Here is a list of valid settings:

RunMode - TRUE for run, FALSE for walk.
ScreenMode - NORMAL for the default window view, FULL for the full view with overlays

- The EverQuest Team
