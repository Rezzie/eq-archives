EQ2 Template Users Guide

Remember that the wiki does not care about case when referring to a page or template name, so although the following shows the names properly cased the wiki does not care if you use proper case, all lower case, or even all upper case when calling a template!

  • EQ2 - {{eq2}} places the EQ2 gametag on the page. Should be used on the very bottom of ALL wiki pages that are in the EverQuest II game wikibase and are not either a DB page or already categorized by another template.

Zone Templates

  • EQ2 Island - Displays a right-side box with information regarding an island. For use with Kingdom of Sky islands.
  • EQ2 POI - Displays links for poi in the form: POIName in ZoneName - Location (loc) eq2map_link
  • EQ2 POIInfo - Displays the right-side info box for a POI. Use ONLY on POI pages!
  • EQ2MapSidebar - Displays a right-side info box with a link to EQ2MAP for the maps for this zone.
  • Raid - Displays a right-side info box on Zone, Quest, or stand-alone pages with all basic Raid info. If used on a quest qhere the database quest category is City_Task, the quest will appear on City_Task_Raids_Quest_Series

Quest Templates

  • BQuest - Book Quest
  • CityTask - City Writs
  • Claymore Quest Series - Box for bottom of page that displays the entire Claymore Quest Series
  • CQ - Concurrent Quests. Use this to tag steps in a quest series that share steps in common and should be done together
  • CQuest - Collection Quest
  • EQ2 Kunark Raids - Displays a box with the RoK Raid Progression information and links. Use this at the bottom of any Quest or Mob page involved in the Raid Progression.
  • LangReq - Displays a message that this quest requires a given language to start or complete.
  • LnLQuest - Displays basic format for Lore and Legend Quests
  • qm - Notes a quest name that contains a Question Mark(?)
  • Raid - Displays a right-side info box on Zone, Quest, or stand-alone pages with all basic Raid info. If used on a quest qhere the database quest category is City_Task, the quest will appear on City_Task_Raids_Quest_Series

Quest Series Templates

  • EQ2 Services - Displays a right-side box with lots of information about a location within a Quest Series. Should be used only on Quest Series pages.
  • QSInfo - Right-side info box with stats on a Quest Series, used only on Quest Series pages.
  • QSWheel - Message box that points to q Quest Series page, and hub, when the quest is part of the series page but not really a direct part of a sequence of quests.
  • series - Navigation Bar for a quest series. Provides links to the previous and next quests in a series, and to the Quest Series page.
  • Raid - Displays a right-side info box on Zone, Quest, or stand-alone pages with all basic Raid info. If used on a quest qhere the database quest category is City_Task, the quest will appear on City_Task_Raids_Quest_Series

Mob Templates

  • EQ2 Banker - Displays a box identifying this mob as a Banker. Called by EQ2 Mob Bottom.
  • EQ2 Broker - Displays a box identifying this mob as a Broker Called by EQ2 Mob Bottom.
  • EQ2 Mender - Displays a box identifying this mob as a Mender Called by EQ2 Mob Bottom.
  • EQ2 Vendor - Displays a box identifying this mob as a Vendor Called by EQ2 Mob Bottom.
  • EQ2 Trainer - Displays a box identifying this mob as a Trainer Called by EQ2 Mob Bottom.
  • EQ2 Hunter - Displays a box with information about Hunter/Slayer/Destroyer titles for this mob race. Use this on any mob page for the race.
  • EQ2 Kunark Raids - Displays a box with the RoK Raid Progression information and links. Use this at the bottom of any Quest or Mob page involved in the Raid Progression.
  • EQ2 Mob Race - Displays a box with the Race and Type of the mob.
  • EQ2 MobAA - Displays a box with the message that this mob rewards AA Exp the first time it is killed.
  • EQ2 MobPOI - Displays a box with the mobs zone and POI.
  • EQ2 MobRaceInfo - Use only on Mob Race category pages! Displays box to right with image, type, language, L&L link, etc. See: Category:EQ2 Mob Races
  • NPCReq - Displays an info box with requirements to talk to/trade with this NPC.
  • Pet - Displays a message that the current mob is a generated pet name (like Kobober), and tells the players how to change the name.
  • ph - Placeholder. Use this on ph and Named pages to cross-link th two.
  • QSMob - Displays a message that this mob is involved in a specific quest series.
  • Shared Spawn - Place this on two or more mobs pages to indicate that they share the same spawn points.
  • See Invis - displays a message that this mob can see players that are Invisible.

Item Templates

  • EQ2 Title - Creates am examine-window look alike for Titles.
  • EQ2 guideitem - Used on items that originate as Guide Quest rewards.
  • EQ2 BadgerItem - tags the item as being from one of the Burynai Quests, and therefore obsolete.

Auto Templates

The following templates are auto-included at the top or bottom of all db pages and contain NO information usable by editors, so just ignore them!

  • EQ2 Zone and EQ2 Zone Bottom
  • EQ2 Quest and EQ2 Quest Bottom
  • EQ2 Mob and EQ2 Mob Bottom
  • EQ2 Object and EQ2 Object Bottom
  • EQ2 Faction and EQ2 Faction Bottom
  • EQ2 Ability and EQ2 Ability Bottom
  • EQ2 Item and EQ2 Item Bottom
  • EQ2 Tradition and EQ2 Tradition Bottom - not wikified yet

Credit Templates

  • Athenaeum - This page refers to a book, which can be read at The Athenaeum
  • Credit - Generic credit template. Use only when a specific template for this resource does not exist!
  • EQ2i - Credit to EQ2i
  • EQ2TC - Credit to EQ2 Traders Corner
  • SOEForums - Credit to SOE Forums
  • ZAMCredit - used to point to a guide on our system that the current page originated as.
  • EQ2TestCredit - use this to extend a big THANK YOU to the users of the Test servers
  • Wikipedia - Credit to Wikipedia
  • TheBrasse - Credit to TheBrasse.com. Note:Brasse's work is copyrighted. Please link to her rather than copying over any images.

Event Templates

  • Brew Day - This page refers to something that happens during Brew Day (St. Patrick's Day), aka Brell's Day
  • Bristlebane Day - This page refers to something that happens during Bristlebane Day (April Fool's Day)
  • EQ2 Froglok - This page refers to something involved in the Froglok Quest Series
  • Erollisi Day - This page refers to something that happens during Erollisi Day (Valentines Day)
  • Frostfell - This page refers to something that happens during Frostfell (Christmas)
  • Nights of the Dead - This page refers to something that happens during Nights of the Dead (Halloween)
  • SoHWE - This page refers to something involved in the Shard of Hate World Event
  • LLWE - This page refers to something involved in the Living Legacy World Event
  • VSWE - This page refers to something involved in the Void Storms World Event

Miscellaneous Templates

  • EQ2 Archetype Tree - Displays the table with all the EQ2 Archetypes, Classes and Subclasses. USE SPARINGLY!!!
  • EQ2 Character Race - Displays a table to the right with basic Character Race information. Should be used ONLY on Character Race pages. See Category:EQ2 Character Races.
  • EQ2 ChRace Icon - Called by EQ2 Character Race. Displays the icon for the race and sex. Could be useful to users to display the race and sex of their toons!
  • EQ2 Class - Displays a table to the right with basic Subclass information. Should be used ONLY on Subclass pages. See Category:EQ2 Subclasses.
  • EQ2 Class Icon - Called by EQ2 Class. Displays the icon for the Subclass. Could be useful to users to display the subclass of their toons!
  • EQ2 Coin - Displays coin amounts with colors
  • EQ2 Epic - Use this on Quest pages (at top) to display the Epic Weapon box
  • EQ2 GodInfo - Displays a table to the right with basic Deity information. Use only on deity pages. See also: EQ2 Gods
  • EQ2 Gods - Place at the bottom of any page to display a table with links to all of the Gods.
  • EQ2 Language - Displays basic information about a Language. Should be used only on Language pages. See: Category:EQ2 Languages
  • EQ2 Loc - Displays location (x,z,y) and, if provided, a link to the EQ2MAP page for this location/POI.
  • EQ2 RaceImage - Called by EQ2 Mob and EQ2 Mob Race to display the default racial image for a mob. May have limited usefulness to users.
  • EQ2 Crafting - At this time, all this template does is add the page to Category:Crafting.
  • DevQuote - Use this for something that a Dev has said, which is therefore highly reliable.

Global Templates

A few global templates that you will use a lot are:

  • Stub - Marks the page as a Stub in need of some loving!
  • OoC - Out of Character. Use this for notes that reference real-world things that relate to the in-game information. This can be the title of a movie or song that a quest name is a reference to, or links to Wikipedia for notes on real-world mythology for things such as Dragons or Unicorns.
  • Trivia - You might use this to link an EQ2 page to the EQ1 lore it came from.
  • Typo - Use this to note something that is misspelled in game, to show that we KNOW it is misspelled but we have to keep it this way.
  • Wikify - put this on a page if you don't know how to fully wikify it, so that a more knowledgeable editor will come along and clean up your information.
  • MobTitle - does nothing more than display the title, bold, with < and > around it.
See: Global Template Users Guide for the complete list.
