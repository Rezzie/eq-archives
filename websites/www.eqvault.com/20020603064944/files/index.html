EverQuest Files

Below is our archive of EverQuest-related files. If you have a file you think should be added please email that file to aurum@vaultnetwork.com.

  • [6-17-00] EQ Numbers Source (190KB): Source for EQNumbers. Pick up the Readme file prior to using/editing. Author: Strom Stillwater

  • [6-04-00] EqLogSplitter v1.1 (250KB): This program was designed to separate an EverQuest log file into one log file per character. Author: Neebbo(neebbo@mindspring.com)

  • [5-21-00] Logger.zip (30KB): This program that searches your log for the levels of the people in a guild of your choice. Author: Luthair Paendrag (webmaster@paladins.gq.nu)

  • [5-21-00] EQMusic3.zip (300KB): Updated set of music MIDIs from the current retail version of EQ. Author: Adrian

  • [4-15-00] EQ Class v1.1 (267KB): The "fool-proof" way to determine which class is right for you! Author: Yappie

  • [8-26-99] Hell Level Experience Calculator (209KB): You choose number of monsters you had to kill at certain level (say 39), and program show you how many same monsters you need for other levels. You can see how hell levels work, you can plan if you need to change mob you are killing for EXP, and you can see difference in EXP for different class/race combos. Author: Nenad

  • [8-26-99] Feekle Log (417KB): Another nifty utility that will analyze any EverQuest log file. Author: Feekle

  • [8-26-99] EQParse082 (108KB): Utility program used with your EverQuest log file Author: Solos Chimamire

  • [8-11-99] C-EQNumbers (108KB): This program combines the best of both worlds and allows you to calculate your mana capabilities and find out how much damage you can do with your melee weapon. Good for any class! Author: Strom Stillwater - Povar Server

  • [7-29-99] C-EQ Mana Calculator (13KB): Another useful app that calculators your character's mana. This does not require VB6 in order to run. Author: Speed

  • [7-29-99] EQ Mana Calc v1.2a (373KB): A graphical mana calculator that allows you to find out how much mana your character has. Note: You will need to download the VB6 Runtime files in order for this program to run. Author: Valgoth SoulReaver

  • [7-24-99] EQMana Calculator (32KB): Program designed to help calculate your character's amount of mana. Note: You will need to download the VB6 Runtime files in order for this program to run. Author: Arnold Cota

  • [7-24-99] Mana calculating spreadsheet (27KB): Another useful spreadsheet that helps you calculate your character's mana capabilities Author: Ziskin

  • [6-27-99] EverQuest WinAmp Skin: A graphical skin for the popular WinAmp MP3 Player. Author: Jenovah

  • [2-5-99] EverQuest MP3 Music: CD quality audio files of the EverQuest game music. We recommend Winamp to listen to these files.

  • [2-1-99] EverQuest MIDI Music: The MIDI music, from the EverQuest phase 2 and 3(and possibly earlier) clients. Basically a MIDI version of EverQuest's music.

  • [1-25-99] EverQuest Splash Screen Wallpaper:This is a wallpaper made from the splash screen of the EverQuest phase 3.5 beta, in 1280x960 resolution.

  • [1-25-99] EverQuest Splash Screen Wallpaper:This is a wallpaper made from the splash screen of the EverQuest phase 3.5 beta, in 1024x768 resolution.

  • [1-25-99] EverQuest Splash Screen Wallpaper:This is a wallpaper made from the splash screen of the EverQuest phase 3.5 beta, in 800x600 resolution.

  • [1-24-99] EverQuest Splash Screen Wallpaper:This is a wallpaper made from the splash screen of the EverQuest phase 3.5 beta, in 640x480 resolution.

  • [1-24-99] Daily Screenshot Archive -- December 1998:This is the archive of all the Daily Screenshots and their associated html companions for December of 1998.

  • [10-8-98] Athynox's Guide to EverQuest: A guide to the EverQuest world as known outside of the closed beta. This document is in Microsoft Word 97 format. You can download the latest Microsoft Word viewer by clicking here.
