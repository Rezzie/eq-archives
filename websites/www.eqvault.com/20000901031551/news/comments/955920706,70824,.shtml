#7   Name: Dave ()   Date: 4/17 @ 2:03 AM PST
There is also a full list of / commands (and emotes) in the eqgame.exe file (including GM commands which don't work unless you are a GM).

Two (new?) usefull player commands which I haven't seen mentioned in any of the patch messages are:

/ftell message

This sends a message to each person on your friends list.

/who all GUILD

This shows you all non-anonymous guild members that are in the game. Note that the word GUILD must be in capital letters. It looks like you can also use NONGM to see everyone that isn't a GM.

In addition, there are 4 other commands which it looks like are non GM commands (if you issue them, you can use Shift-Up to scroll back and see them, something you can't do with GM commands).

/channel channel

The message you get with this one says it is used to toggle text channels on/off. Some apparently valid values for channel are ooc, auction, and shout, but they didn't seem to have any effect at the moment.

3 other commands that don't seem to do anything at the moment but look interesting are:

/wordlist

/microphone

/voice

Hmmm, voice communication (within groups?) maybe? Probably not, as it would eat up some bandwidth.

There is also a /resetdiscipline command, which is probably used to return you to a default /discipline.
