Hello,

Here is a Script to get a NPC spawn to give a quest out to only good or evil players.

1) Go into the OnSpawn script for the NPC and uncomment the line 'SetSpawnInCondition (NW_FLAG_PERCIEVE_EVENT);', then re-save under a different name and compile.

2) Make a new script for the OnUserDefined event like so:

#include "NW_I0_GENERIC"
void main()
{
int nEvent = GetUserDefinedEventNumber();
if (nEvent == 1002) // OnPerceive
{
object oPC = GetLastPerceived();
int nAlign = GetAlignmentGoodEvil(oPC);
if (nAlign == ALIGNMENT_GOOD)
{
AdjustReputation(oPC, OBJECT_SELF, -100);
DetermineCombatRound();
}
}
}

This will cause a NPC to attack any Good aligned PC. Set the NPC starting faction to +100 PC friendly, if an Good aligned PC comes close it drops the faction to Hostile, and attacks

To change it to attack any EVIL PC just change the:
if (nAlign == ALIGNMENT_GOOD) to if (nAlign == ALIGNMENT_EVIL)

And compile.

Useful for setting mob to hand out quests to certain alignments.

Hope that helps
Keyno
