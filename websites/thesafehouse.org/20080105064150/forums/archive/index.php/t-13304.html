Patje over at DriverHeaven.net has released DHMODTOOL 2.0 for ATI Catalyst drivers. It allows you to add mobility support to any set of Catalyst drivers. This is a great tool for laptop owners of Mobility vid cards, as you can use the latest Catalyst drivers for your Radeon mobillity cards.

Release notes:

Hi, This is a mass mail to all Modtool users who ever mailed me about a problem or just a friendly thanks.

The 2 year anniversary of DH called for something special.

- Changed the setup files, now the Catalyst files installs the way it was meant to be, including the control panel.
- Added support for the Mobility Radeon 9700.
- You now have the option to mod the files only for your card, which speeds up the modding.
- And with the new layout (Credit goes to Craig5320!) people who where experiencing *.ocx errors, those are gone as well.
- Possible resolution fixes for some cards. (Did not have the chance to test on all cards, send me some laptops and I will )
