If you want to mesh worlds together a great system to use is GURPS: Generic Universal roleplaying system. It's designed to allow for all settings and worlds to use the same rules. And unlike many other systems the rules actually make sense in relation to real life. www.sjgames.com/gurps/lite/
That link goes to a free distillation of the GURPS system and explains how the system works and has some basic information about it. From the information in the download at the link you can make a campaign, create characters, and run combat scenes. However, unlike most systems GURPS does not give any backround story, it simply gives generic rules and leaves the game worlds up to the DM, which I suppose is what you want.
