Left axe in a nutshell:

Left Axe skill decides:

Damage range of your left axe weapon
Total Damage done with 2 weapons, using the following formula- Total Damage = (normal damage/2)+25%+(left axe/2)%

Left axe is one skill shadowblades can ignore, because we can use 2h weaponry. Currently I use sword and shield and am not having any issues with it at level 39.

As far as letting main weapon atrophy and focusing on left axe, I haven't ever read about it raising critical strike damage, and all other weapon styles go off main hand as well, so it would have to do the same for those as well, for it to be a viable option. Dither Disgustipated
