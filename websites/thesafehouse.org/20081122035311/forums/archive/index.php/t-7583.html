@#%$. Ignore everyone that's posted so far except Greldek. The auto-attack/strafe/backstab/evade stuff doesn't work on Vallon.

First ask yourself this question: "Do I want to actually learn the rogue class? Or do I want to just run around owning people?" If you want to learn, don't twink (beyond banded, maybe an SBD or Obsidian shard). If you can't resist the temptation then start the rogue on a new server like Tallon so you can't twink.

You WILL NOT learn how to play the class if you run around at lvl 5 with a trak tooth, regen tunic, and BD jewelry. And you certainly won't learn how to pvp with a rogue with that stuff on. With a rogue, it's best to be an underdog. Personally I still love being the underdog at higher levels.

Most skills you'll learn on your own, but here are a couple to master:

Learn to joust. There are many variations, learn them.
Learn poison as soon as you can. This skill is very time-consuming, but it IS the difference between an average rogue and a good one on VZ.
Keep skills maxed (which is hard to do if you're twinked), especially backstab.
You wanna twink yourself? Buy alot of pumice instead.
