Just realized I hadn't seen this mentioned in our forums before.

The Cosmos UI for WoW is great. You can set autoloot, set yourself to create tradeskill items one after another for a set amount (rather than hitting the create button once per item), and time spell casts (great for things like Find Herbs or Find Minerals) and it has a shared quest system so you can see how far along other folks in your party are on their part of their quests.

Here's the thread about it from the official beta forums:
