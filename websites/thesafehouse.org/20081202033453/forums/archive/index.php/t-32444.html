The gorgalok swamp armor is pretty good, but if you don't want to spend that much time farming mobs and trying to get a core of ezigoth, go for crafted.

If you can, buy flawless cavalier's jagged armor. You can add little effects like health, or end regen, or melee avoidance to those pieces as well, but it will reduce the amount of +crit you get.

As for weapons, you can buy striking weapons from a T5 weaponsmith, but it will be expensive to buy all this stuff.

If you want to camp/quest for some stuff, have you done the helm quest in Rhaz Inkur? While working on that quest, you should be able to get the necessary faction to start the Arazmus weapon quest in Afrit as well. The longest part of the Arazmus weapon is collecting the coin bags necessary to be able to start it. (1 bag = 50 faction). If you are a good race and have not factioned there before, you will need 60 bags to be able to start the quest, and another 20 bags to finish it. The other parts really don't take too long though.

That will give you a nice dagger with particle effect, and you will probably get a Djarn's Longsword of Accuracy from Djarn while you work on the Legendary helm quest.
