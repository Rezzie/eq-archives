This is a guide for flashing a Linksys WRT54G/GS/GL wireless router with DD-WRT firmware (http://www.dd-wrt.com/). DD-WRT can be used on other routers as well but there are usually special instructions that you need to follow for those. You can find this information and much more on the Official DD-WRT Wiki (http://wrt-wiki.bsr-clan.de/index.php?title=DD-WRT_Docu_%28EN%29)

Here's the official DD-WRT Support Forums: http://www.dd-wrt.com/phpBB2/

You can view all the Supported Devices for DD-WRT here (http://wrt-wiki.bsr-clan.de/index.php?title=Installation#Supported_Devices). It supports several brands and models so make sure to check and see if your router is supported. Listed below, you'll see the Linksys router versions and how to determine what version you have. You just look at the first several characters of the serial number.Linksys WRT54G 1.0 CDF0xxx or CDF1xxx
Linksys WRT54G 1.1 CDF2xxx or CDF3xxx
Linksys WRT54G 2.0 CDF5xxx
Linksys WRT54G 2.2 CDF7xxx
Linksys WRT54G 3.0 CDF8xxx
Linksys WRT54G 3.1 CDF9xxx
Linksys WRT54G 4.0 CDFAxxx
Linksys WRT54G 5.0 (JTAG only with cfe update, see here. (http://wrt-wiki.bsr-clan.de/index.php?title=Flash_Your_Version_5_WRT54G))
Linksys WRT54GL 1.0 CL7Axxx
Linksys WRT54GL 1.1 CL7Bxxx
Linksys WRT54GS 1.0 CGN0xxx or CGN1xxx
Linksys WRT54GS 1.1 CGN2xxx
Linksys WRT54GS 2.0 CGN3xxx
Linksys WRT54GS 2.1 CGN4xxx
Linksys WRT54GS 3.0 CGN5xxx
Linksys WRT54GS 4.0 CGN6xxx
This guide is only going to focus on the most common routers out there - the Linksys WRT54G/GS v1.0 - v4.0 and the WRT54GL (same as WRT54G v4.0).

NOTE: Flashing firmware to a device is a very tricky thing. You MUST do it at a time when you're pretty sure there will be no interruptions to your power, computer or attention span! If there is a problem during the flashing process, you can sometimes "brick (http://wrt-wiki.bsr-clan.de/index.php?title=Bricked)" your router. Here is a link to recover a bricked router: Recover from a Bad Flash (http://wrt-wiki.bsr-clan.de/index.php?title=Recover_from_a_Bad_Flash) (Make sure you save it offline!)

OK now that we've gotten that out of the way, we can start off with the things you'll need:

A supported device I mentioned above. If it's a WRT54G v5.0, please stop right now and go read the special link (http://wrt-wiki.bsr-clan.de/index.php?title=Flash_Your_Version_5_WRT54G) for that instead and stop following this guide.

A wired connection between your computer and your router. This means you have to hook up your computer via ethernet cable to the router. Make sure you have the IP address for your computer set to a static IP (i.e. 192.168.1.2) and the gateway is set to your router's IP (default is 192.168.1.1) Example picture: TCP/IP Settings (http://schrade.best.vwh.net/articles/hardware/DD-WRT_Firmware/TCP-IP.png)

The Linksys Firmware updater from the Linksys site for your specific model of router. You can get it by going to the Linksys Support Downloads Page (http://www.linksys.com/servlet/Satellite?c=L_Content_C1&childpagename=US%2FLayout&cid=1115416835852&pagename=Linksys%2FCommon%2FVisitorWrapper). This is important because if you have any problems you'll want to use this to try and recover from them by reflashing your router to the Linksys Firmware. Also, make sure that you're running the very latest Linksys firmware for your router. Update your firmware first if it's not the latest.

The DD-WRT v23 SP1 firmware you're going to flash the router to. For now, there is one you need to download: dd-wrt.v23_mini_generic.bin (http://www.dd-wrt.com/dd-wrtv2/down.php?path=downloads/dd-wrt.v23%20SP1/mini/&download=dd-wrt.v23_mini_generic.bin) . If you want some other specialized features you can get one of these versions after you've flashed to the mini one above: Standard (http://www.dd-wrt.com/dd-wrtv2/down.php?path=downloads/dd-wrt.v23%20SP1/standard/&download=dd-wrt.v23_generic.bin), VOIP (http://www.dd-wrt.com/dd-wrtv2/down.php?path=downloads/dd-wrt.v23%20SP1/voip/&download=dd-wrt.v23_voip.bin) and VPN (http://www.dd-wrt.com/dd-wrtv2/down.php?path=downloads/dd-wrt.v23%20SP1/vpn/&download=dd-wrt.v23_vpn_generic.bin). I personally use the VPN version. Avoid the versions with the specific model of the firmware in the filename. (wrt54g/wrt54gs/wrt54gsv4) Those are for flashing via TFTP only! The "generic" one is the one for flashing via web interface.

Once you've established that you have a supported device and have downloaded all the firmware I listed above, we can start the flashing process.

The first thing you do is to make screenshots or write down the settings you have in your router for port forwarding and/or any other things you may have set up. I recommend using the Scrapbook Extension (http://amb.vis.ne.jp/mozilla/scrapbook/) for Firefox to do this. It'll save an exact copy of a page in your router's config interface with any settings in the fields and whatnot. It is also good for saving a local copy of the web page that tells you how to unbrick your router :) It is important that you have your computer physically plugged in during the flashing process. This means that you have to be plugged in via Ethernet cable to the router. Do NOT flash over a wireless connection!

And now, it's time to flash!

Reset your current firmware to factory defaults. This can be done by going to the appropriate tab in the router config interface and selecting factory defaults, or by holding the reset button on the back of the router in for 30 seconds. This will clear your NVRAM settings in the router and then load up the factory defaults. Remember that the Linksys firmware sets your login to be nothing and the password is "admin".

Once that is done and the router has completed rebooting, go to the firmware upgrade tab and browse for and select this firmware on your hard drive: dd-wrt.v23_mini_generic.bin (http://www.dd-wrt.com/dd-wrtv2/down.php?path=downloads/dd-wrt.v23%20SP1/mini/&download=dd-wrt.v23_mini_generic.bin)

Now, click on the upgrade button and wait for your router to finish flashing. It is VERY important that there are no interruptions at this stage. Wait up to 5 minutes during this process before you try to do anything. Watch the lights on the front of the router. You should be able to tell when the modem reboots after a flash because the power light will start blinking. Once the power light becomes stable then the router should have rebooted successfully. Check to see that the router came up successfully by going to the router config page: http://192.168.1.1/ You should see a System Information page of your router.

You should now be running the DD-WRT Mini version of the firmware. The login is now "root" and the password is now "admin".

Click on the Administration tab and then click on the Factory Defaults sub tab. Click on the Yes radio selector and then click on the Save Settings button. The router should now reboot and load up the factory default settings for DD-WRT. If you are having any problems at this point with accessing the router config page, try to do the 30 second reset. (Push in the reset button and hold it for 30 seconds)

This step is optional. If you want to install a DD-WRT firmware with more features like XBOX kaid, VOIP/SIP enhancements, OpenVPN client/server, etc. you can now install one of the other firmwares I listed above. It is ESSENTIAL that you first flash to the DD-WRT Mini firmware before upgrading to any of those other firmwares. The reason for this is because the default Linksys firmware does not support flashing firmware images larger than 2 megabytes. You have to be running DD-WRT to do this.


DD-WRT Feature List: Feature List (http://wrt-wiki.bsr-clan.de/index.php?title=What_is_%22DD-WRT%22%3F#Feature_List) @ DD-WRT Wiki

From v23 File Versions (http://wrt-wiki.bsr-clan.de/index.php?title=What_is_%22DD-WRT%22%3F#v23_File_Versions) @ DD-WRT Wiki
Micro - does not contain: chillispot, nocat, rflow, kaid, samba client, SNMP, IPv6, MMC/SD Card Support, SSH, PPTP/PPTP Client, UPnP. This file is under 2MB in size. While it is aimed at routers with less than 2MB of flash space (e.g., Linksys WRT54G version 5), any router should be able to run this version, including Linksys WRT54G versions before 5. Note that the Micro version is considered in beta, so it has a chance of instability.

Mini - does not contain: chillispot, nocat, rflow, kaid, samba client, SNMP, IPv6, MMC/SD Card Support.

Standard - all features but without VOIP

VOIP - all features (without kaid) + sipath for sip voice over ip routing

VPN - all features (without radvd, samba, and kaid) + OpenVPN

OK.. make note of the special information for flashing the Buffalos then!

Flashing the Buffalo WHR-G54S and WHR-HP-G54 (http://wrt-wiki.bsr-clan.de/index.php?title=Installation#Flashing_the_Buffalo_ WHR-G54S_and_WHR-HP-G54)

Make sure you have your computer set to 192.168.11.2 when you go to flash the router. The router's IP address will be 192.168.11.1. You won't be able to flash to DD-WRT using the default Buffalo firmware because they have it set up so you can only flash signed/encrypted firmware.

To get around that you have to use TFTP to flash it. TFTP is a builtin command with Windows XP and most Unixen. You'll see the commands you need to do in that link above.

So.. steps to do this:


Make sure you have your Computer's NIC set to 192.168.11.2, Full Duplex 100 MBit, the Buffalo router should have the IP address of 192.168.11.1 by default.

Unplug power to the router.

Open up two command prompts (Start -> Run -> cmd [enter]) and type this in one and hit enter:
ping -t 192.168.11.1

In the second command prompt cd to the directory where you have the TFTP Version of the firmware you want. I'm just going to point to the VPN one since that's what you mentioned. So type this:
tftp -i dd-wrt.v23_vpn_generic.bin (http://www.dd-wrt.com/dd-wrtv2/downloads/index.php?path=dd-wrt.v23%20SP1%2Fvpn%2F&download=dd-wrt.v23_vpn_generic.bin) (don't hit enter yet)

Now, plug the router back in. The second you see pings coming across in the ping command window, hit enter on the tftp command window. If you see a successful transfer, leave the router alone for about 2 minutes. I know this will seem like eternity but just do it for safety's sake. Once that's up, then unplug the power, wait 10 seconds and plug it back in. Your Router's IP address is now 192.168.1.1! Make sure you change it on your computer so that you have IP address 192.168.1.2 on the computer, 192.168.1.1 as your gateway.

To login to the router you just go to http://192.168.1.1/ and login name is root, password is admin. Once logged in go to the Administration - Factory Defaults tab and reset the router to factory defaults. This is important!


You should now be able to enjoy DD-WRT firmware on your Buffalo router. You will no longer have to upgrade the firmware via tftp with it, you can just upgrade it via the web interface. The default Buffalo firmware just doesn't allow for that.

Also, should you ever want to go back to the default Buffalo firmware, you can use this: http://brainslayer.braincontrol.org/backup_whr_hp_g54.bin Just make sure you use tftp to get it over there.

Disclaimer: I do not own a Buffalo router so I am only try to translate the instructions to make them a little easier to follow. Do these at your own risk!
