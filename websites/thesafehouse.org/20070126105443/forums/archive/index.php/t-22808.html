well, like the quest says: behind the Ravenhold manor.

NB:Had you been a rogue, you would have had visited Ravenhold for that broken roguish quest before :pulid

a) finding Ravenhold: ride from Hillsbrad->Arathi, on the pass entrance where 2 griffons court, follow the hill line rounding west-north toward river. If you keep close to mountains, there is a path begin behind (next to) a tree that leads up into mountain and Ravenhold (its first leg climbs IIRC direct north). It is easy to miss though but it is there - path ground is brownish and distinguishable from grass.

b) To find apothecary you have to pass behind the main building and then practically drop down a bit behind the plateau to be able to climb left up to apothecary undead.

HINT: not sure if it all is worth a violet suit or a dress for ya! :pulid
