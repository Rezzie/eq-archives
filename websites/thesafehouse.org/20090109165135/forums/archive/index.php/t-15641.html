For what it's worth, as a 22 ranger I still use Ambush all the time, despite the fact that it's grey. I do a ton more damage per hit with Shadowblade (requires stealth and postion, long reuse delay), but Ambush refreshes faster.

A typical fight group fight for me, assuming no heroic oportunities, starts like this:

Hunter's Instinct (agi buff)
Hunt (stealth)
target the tank
Ambush (stealthed attack for 75ish)
Bleed (damage over time)
Impale (80ish DD + reduces target's piercing defence for a short while)
Shrouded Strike + autoattack-off (positional attack for 90ish puts you in stealth)
Shadowblade (Positional stealthed attack for 250ish)
Sneak Attack (positional attack for 60ish)
Back Shot (positional ranged attack for 200ish)
Wounding Arrow (hits for about 30 and lowers targets defences)
Shrouded Strike + autoattack-off again
Ambush again

For soloing, Cheap Shot + autoattack-off + Stalk + Ambush is a lot easier than with Shadowblade. But I usually focus on HOs instead.

#!/usr/bin/perl

while (&lt;>) { #onward, green hornet!
$search_var = "YOU hit"; #unique to melee attacks
if ($pattern_match = grep (/$search_var/, $_)) { #ensure this line matches
split (/for/,$_); #grab the right side from for
$val=@_[1]; #pull that part out
split (/ /,$val); #grab just the number
$dmg_value = @_[1]; #and assign that
if ($dmg_value) { #if we have a number
$hash_dmg{$dmg_value} += 1; #make the dmg count array
$linecount++; #this is a valid line, so count it
$total_dmg=$total_dmg+$dmg_value; #add this dmg line to total
} #close if ($dmg
} #close if ($pattern_match
} #close while (&lt;>)
if ($total_dmg &amp;&amp; $linecount ) { #do we have results?
$average=$total_dmg / $linecount; #then calculate the average
print "Total accumulated damage: $total_dmg\n"; #print total dmg
print "Total lines parsed: $linecount\n"; #print line count
print "Average hit damage:"; #print avg dmg
printf " %2.2f\n", $average; #with formatting
foreach $keys ( sort {$a &lt;=> $b} keys %hash_dmg ) { #walk through hash
$pct = $hash_dmg{$keys} / $linecount * 100; #calc percent
print "Damage value $keys count is $hash_dmg{$keys}"; #print each key in the hash
printf " %2.2f%%\n", $pct; #with percent formatted right
} #close foreach $keys
} #close if ($total_dmg
exit; #and done

A bit late here, but here's the skivvy on Ambush based on what I've seen:

The difference between Ambush and Sneak Attack/Backstab/Shadow Blade is that it's not a positional attack. Thus, it's only really worth using from the front, or while waiting for other abilities to pop. As an Assassin, it becomes fairly useful every once in a while while killing agro: Evade, Stalk (also reduces agro), Ambush. It's fairly low agro, so Evade and Stalk more than cancel it out.
It becomes a bit more useful with Shrouded Strike because of its low refresh time, between Shadow Blade and Cut Throat timers. It's never exactly won a fight for me, but it comes in handy in a pinch.
