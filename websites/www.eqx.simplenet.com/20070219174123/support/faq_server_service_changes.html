Dedicated Server FAQs: Service Changes

1. How do I upgrade my bandwidth level?

Here are the bandwidth upgrade options:

  • 512Kbs / 160GB per month - $150.00 per month
    This option is great for high-traffic web sites or web sites offering large files such as multimedia or downloads.
  • 1024Kbs / 320GB per month - $300.00 per month
    This high-speed connection is the equivalent of two-thirds of a T-1 connection.

If you would like additional bandwidth for your server, please email with the following information:

  • IP address
  • Amount of bandwidth that you wish to add
  • Main password for the account
  • Last four digits of the credit card that supports the account
  • Authorization to modify the billing of the account

2. How do I add more RAM to my server?

If you would like to add more RAM, pricing is as follows:

  • Upgrade Memory from 64 MB to 128 MB = $25/month
  • Upgrade Memory from 128 MB to 256 MB = $50/month
  • Upgrade Memory from 256 MB to 512 MB = $100/month

To submit a request, please email with the following information:

  • Amount of RAM that you wish to add
  • Main password for the account
  • Last four digits of the credit card that supports the account
  • Authorization to modify the billing of the account

3. How do I get more IP addresses?

The dedicated server comes with just one static IP address, but on that one IP address you can 'virtually' host up to 200 web sites. You do not need to pay for additional IP addresses unless you want to give static IP addresses to your various customers or if you plan on having more than one site on your server that is using a SSL certificate (you are only allowed to have one SSL certificate per IP address).

To have additional IP addresses allocated to your server, there is an additional monthly fee of $25 for a block of eight. If you would like to have this service added, please email with the following information:

  • Last four digits of the credit card on file for the account
  • Main password for the account
  • Authorization to modify the billing for the account
