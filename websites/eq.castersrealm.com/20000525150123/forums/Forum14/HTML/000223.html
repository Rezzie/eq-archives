An example where you want higher Int than Cha (no Cha argument here) assuming your lowest cha is above 150...

Fighting things you cannot mez. Seafury Cyclopses for example. The only crowd control here is charm, root and mem blur. Charm works very well on them at 165 cha (my lowest setting). Here we need more Int for debuffs and buffing other members and the occasional nuke.

In Guk or Solb I generally lower my int and raise my cha for crowd control if it is needed, but these days there isn't much use for CC. So lately I have been running around with 165 cha, 189 Int and doing just fine, even on the planes.

Stats:
1. 165 cha, 189 Int
or
2. 195 Cha, 175 Int

Lately I have grown to like option 1 better. Can't wait for a Syrin Hair Hood (or better) as it would allow me to improve both stats.

As far as JC goes... and this is my opinion. Forget wisdom, and go all out intelligence. There is no proof whatsoever that wisdom for an int based caster does anything for success rates.
