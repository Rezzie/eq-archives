Strategy

Cast Screaming terror, then Darkness, Vampiric Curse, boil blood.  Do this while running to pet, which should be a ways east from pillar.  Sick pet and start casting Shock of Poison.

 

This is much safer if Squire Wimbley is being killed, otherwise you need to use Screaming Terror on him.  He may run up and help out but he is only around level 2o so no worries.  If things get too bad, just Harmshield and run to bridge, which isn't far away.
Faction Standing Lowers faction standing with Qeynos Guards, Temple of Life, and Karana Residents.
