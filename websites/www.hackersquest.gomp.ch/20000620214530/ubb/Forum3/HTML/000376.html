Decaying Skeleton (Newbie) 		posted 05-12-2000 11:31 PM            
Try this:

$ /usr/X11/bin/xset s off

With no parameters, it will display some other useful screensaver options.

$ /usr/X11/bin/xset s <# seconds>

That will change the timeout.

Hopefully that was the problem for most,
