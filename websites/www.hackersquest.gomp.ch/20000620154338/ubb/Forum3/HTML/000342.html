There isn't a link. You need to use CVS.

Thanks to Anon SEQ user, I was able to figure this out also from his posts. But here is the step by step instructions he posted:

make a dir wherever you want the source to be kept. It will make a showeq subdir by itself. At your console prompt, type the following:

1: export CVSROOT=":pserver:showeq@www.hackersquest.gomp.ch:/showeq"

2: cvs login
it will ask for password, type showeq and hit enter
it *should* give a prompt back. If it doesn't give you a prompt, the server is hung... hit ctrl-c and try again later.

3:
a) if you have never downloaded the src before:
cvs checkout showeq
It will download all the files. It will automatically create a new showeq dir in your current working dir so don't use one that has a showeq dir.
b) if you already have the src and want to update:
cvs update showeq
it will only download the new files. Do this in the parent dir of the showeq dir you made by cvs checkout above NOT in the showeq dir itself.

4: cvs logout
Please be sure to do this so the rest of us can login :)

5) cd showeq

6) bash ./autogen.sh
it will run and compile. Hopefully no errors.

7) su root
must be root to make install

8) make install

you should be done!
