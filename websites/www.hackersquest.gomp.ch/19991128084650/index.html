10.06.99 :: EQInside Update! (v0.2B)
EQInside Version 0.2 beta:

    • Skill required
    • Spell messages
    • Casting / Recover times
    • Spell Effects
    • Experience level required
    • Components required (both consuming and not consuming).
  • Saving selected spells information to the disk
  • Easy navigation. Select spells by:
    • Character class
    • Levels
    • Spell skill
    • Spell Effect
