Summarized Question: On a machine with a dual monitor setup, is it possible to play EQ and access the web via a browser or email client simultaneously.

Quick Answer: No.

Detailed Answer: No. Since EQ uses DirectX and API calls to isolate the GUI(Graphical User Interface) and the desktop from the memory areas in which EQ resides, you would not be able to launch or use applications on the secondary monitor. Verant uses this isolation to prevent (or at least, make more difficult) access to the memory areas that EQ uses by being able to ALT-TAB into the GUI and run another application. Are there ways around this? Yes. Are there ways around it without encountering coding issues? Not really, and it comes down to the point where it's more bother than it's worth.

So as soon as you switch the active application from EQ to whatever else you're trying to use, your EQ memory get corrupted and the game crashes.

Having never actually tried it, I can't say for certain what would be displayed on the secondary monitor, but I tend to believe that since EQ takes total control of the GUI and the Primary Display Device, it would just remain blank while all game output was directed to the primary monitor.
