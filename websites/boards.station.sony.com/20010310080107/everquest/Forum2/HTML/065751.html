It's easy to find out how much experience you need for each level. There are web sites with this, for this example, or you can calculate it from the forumulas, which have been well publicized. I need 18519380 XP to get through 45, for example.

The silly XP bar has some non-linearity in it, so you'll get some error, but you can get a reasonable estimate from it.

If you don't want to go to the trouble of finding out how much XP you need for a level, and going through logs to figure out how long you were fighting, so as to get an XP/minute estimate, you can just count how many kills it takes per bubble. If it takes you 15 kills for the first bubble in a level, and after the patch it takes 13-17 kills for the second bubble, then XP is probably the same (there's gonna be some variation like that because you'll be fighting mobs that range in levels).

Finally, you can get a pretty good idea of the level of many Kunark mobs by seeing how much damage they took. Sarnak Knights, for example, range from level 33 to 37, if I recall correctly. They have a different number of HP at each level, so if you melee them, or nuke them and finish with small nukes, you can add up the damage and get a good idea which level the mob was (before Gems, that was something to do while medding--scoll back and add up the damage). From that, you can calculate the XP.
