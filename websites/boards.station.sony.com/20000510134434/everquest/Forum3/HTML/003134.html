The test server is just like it sounds. It is the server that all the patches and new stuff (and nerfs) are tested on. Other than that, it is idetical to all the other servers. It even has player guilds.

In fact, a lot of people play their main character there, because there is no overcrowding. The only thing to watch out for is that Verant has NOT guaranteed that there will never be a character wipe on the test server.

I don't believe that has happened yet, and it PROBABLY won't. However, it happens quite frequently in other games, like UO.

Then again, I could be wrong. Maybe there have been character wipes. Anyone know for sure?

To get on it, you must find the program called EQTest.exe (or something similar), in the Everquest directory, and open it. It works just like the regular EQ app, but it only offers one server.

Another thing about the test server is that it has more down time than normal servers due to the large number of changes (testing things). There are also more bugs, and it has even changed between PvP and non-PvP before. Basically, everything is less reliable there. Still, the lack of crowding and being able to experience new things first may well be worth it to some people.

There's a separate executable (NOT Everquest.exe) in your Everquest directory. I don't recall the name offhand, but it's pretty obvious. Just sort by file type, and check the names of your executables.
One other thing to be aware of: there are different patch versions (obviously) between Test and Prduction servers. If you switch back and forth between the two, you will patch each time, as the zone, etc. files overwrite themselves.
