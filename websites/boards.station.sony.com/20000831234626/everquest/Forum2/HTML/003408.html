Hello all,

I thought I'd drop by and let you know of a new feature that we are trying out on Test. Last week it was suggested here on the boards that we implement a "Looking for Group" flag that players can set and view in order to help people get together quickly, allowing more time for adventuring. We've decided to try it out and it's on Test now.

The /lfg [on|off] command will set or unset your "LFG" flag. Players can then use all of the "/who" and "/who all" filters to find each others. For instance, "/who all rogue 45 50 lfg" will return all rogues between 45 and 50 who are looking for a group. "/who lfg" will return all players in the current zone who are looking for a group.

Your LFG flag will automatically unset when you enter a group. One final note, though anonymous and roleplaying characters can set their LFG flag, they will not show up in filtered /who lists. In other words, the first example above wouldn't return any anonymous rogues, however "/who all lfg" would return everyone in the game who is LFG (subject to the 20 person limit).

Assuming all goes well in testing, this change will most likely be in the next patch.

-Gordon
