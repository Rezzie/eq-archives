Hello folks, just wanted to respond to a few more questions that we received via this forum over the last few days.
1) Complete Heal bug. There is a fix for this in on Test, and has been for awhile. However, due to the smaller Test population, it is hard to definitively say it is fixed. I can say that we have been totally unable to reproduce the bug on Test, so that is a good sign. This code will be in the next major code patch.
2) Stacking issues with Shadow Sight and Dead Man Floating. This is known and currently being worked on.
3) Torrent of Fatigue. Yes this was broken last spell patch, it is being worked on.
4) Rogue poisons being interupted. No, this is not by design and should all be fixed on test. Those of you that play rogues on test please feel free to let us know what poisons are still being interupted. These fixes will go live in the next code patch.

All times are PT (US)
