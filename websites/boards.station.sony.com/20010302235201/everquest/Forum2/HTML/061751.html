NOT TRUE. Get your facts straight before you make things up. A bug is ANYTHING that makes a program work in ANY manner other than intended. Generally, the bug you mention (making the game unplayable) are called syntax errors, because they are a problem with the code itself, and will halt the program. Bugs that you think are not bugs are generally called symantic errors, because the code does not work in the manner intended. By definition, that makes anything that you think "works fine, just not the way [VI] want[s]" a bug, usually a symantic one.
