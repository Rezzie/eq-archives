Before reformating anything try this:

hit F8 at boot choose safe mode command prompt only.

at the c:\ prompt type:
cd windows
deltree temp
(hit yes and delete everything)
type
md temp
reboot and try the install again. I have had problems where old self exe's and setup rutines left junk behind that caused problems.
also delete any folders on the hard drive that start with a ~
Good luck.

The default temporary files directory is C:\Windows\Temp, in case you can't find it.

I believe your problem is software related though, probably corruption of the Registry/etc. Now, you CAN 'clean' install w/o formatting using the initial copy of the registry Win9x makes during installation but considering all the trouble you've been through, you might as well run a thorough diagnostic, fix anything you find then format/reinstall.

Naturally, copy all the sutff you want to keep, saved games , Favourites/Bookmarks, email/newsgroup folders. Most programs allow you to specify the location of stored data files, try using this as well as exporting stuff like email messages into text format.
