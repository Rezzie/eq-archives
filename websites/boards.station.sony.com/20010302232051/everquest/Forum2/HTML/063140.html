Just a quick note in response to

"As for you being able to take your items with you, not a chance.

When you are playing in a testing environment like the test server you are able to obtain many items from "bugs" and/or "tuning issues" as encounters are brought into alignment with the intended design.

To allow those items to be kept on a "live" server would be unfair to the players who had to achieve them via the normal route.

Essentially, it's like cheating."

This is a common misconception of Test Server. In actuality people play there exactly the same as in "live" servers. "Bugs" and "tuning issues" are not any more of a factor on test server than they are on live servers. Unfortunately, most bugs of this nature make it to live servers (think Kunark and Vellious item availability and how it changes soon after release to live servers). If a bug happens that people get items they shouldn't, there is a roll back or item wipe. If the same thing happens on a live server (and it has) they do nothing. Playing on test server is much more of a challenge than playing on a live server, you cannot just go buy anything you want as there are no farmers. Some rare spawns happen more often on test, some are almost never seen (depends upon if they pop on server up or after a week or so... test is rarely up for a week at a time). Test players deserve respect for playing on the hardest server of them all, not dismissal for being on a "cheat" server.
