Firstly, i would like to thank Verent for crediting we Gnome to make the SoV boat - i feel proud of it.

I love my Gnome, although we occasionally be teased because we are short, i am cute, lovely, and i know tinkering ! The skill which is so unique among all races ! I felt so excited when i recognized that i would possess this special skill if i choose to be a Gnome. And so, i become a Gnome necromancer.

Now, being a 54th Heretic, i felt rather disappointed to the tinkering skill. Not only number of items we can make is so few, nothing really useful we can make.
