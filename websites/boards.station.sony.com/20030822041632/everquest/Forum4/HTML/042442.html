To play on the test server, you have to run the executable file "TestEverQuest.EXE". To play on the normal servers, run "EverQuest.EXE". I'm not sure what you mean by:
quote:
how do I remove the test thingie?

/testbuff will only work on a character created on the Test server. You can only connect to the test server if you run the Testeverquest.exe executable. This will patch all the test files and allow you to connect to the Test server. If you are thinking of using /testbuff on a character you have already made on a regular server, I'm afraid you are out of luck.

Yeah, just copy the EQ directory.
