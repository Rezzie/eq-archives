Usually if/when I run into this sort of problem, I'll consult the folks over at the tech support forum.

The search feature can really come in handy there. Another good, but not so reliable, way to find out a solution to a problem is to try the Knowledge Base.

Contacting tech support doesn't net me the best results. All the representatives can do is direct you to try simple things like reboot, reset modems/routers, and just wait it out. With a constant crash bug, all they can do is reassure you that it's being looked into, but it's not like they can fix it right then and there.

As far as an ETA goes, I don't think anyone on this message board can supply you with one. You could try emailing tech support.

I frequently have this problem after a patch. Usually, it has to do with my firewall. I have to allow permission for a new contact from EQs server. Try exiting and see if you have a warning message from your firewall provider. If so check and see if the server attempting to access your PC is a new EQ server. If so, you need to let the firewall software know its approved for contact.
