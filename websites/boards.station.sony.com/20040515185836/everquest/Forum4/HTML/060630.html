If the expansion was not registered, the book wouldn't do ANYthing.

However, if the expansion IS registered, but the zone files are missing, the book will drop you in the current zone "safe spot".

Exit to desktop, and set your patcher to complete file check (assuming you have high speed access, or a few hours to burn).

Otherwise, take your PoP CD and install it, making sure the files go to the main EQ directory.

If you've registered the expansion you can click on the "Advanced" button on the patcher start up screen. Make sure the update boxes for all your expansions are checked. There's also a button that will let you do a complete file check so you can see if any files may have been overwritten or deleted.
Alternatively, you can do an in game check by running out to North Karana and taking the Nexus Spires and trying to use the PoK book in Nexus.
