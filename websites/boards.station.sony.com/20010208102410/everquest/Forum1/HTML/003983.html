You can use them as follows:

example: To trace to the Test Server

At your MSDOS prompt while connected to the internet)

C:\WINDOWS>tracert test.everquest.com

To get a hard copy :

C:\WINDOWS>tracert test.everquest.com > c:\windows\desktop\mytrace1.txt

This will create a hard copy of your connection to the Test Server and place it on your desktop for use when calling about Connection problems.
