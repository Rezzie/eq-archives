Hello,

To run a trace run simply get online and get into MS-DOS prompt. Type the command tracert chat.everquest.com look for any types of *'s or -'s or high MS's for example.

* * *
57ms - -
102ms 670ms 1000ms

When you run this trace don't worry about 'Destination Net Unreachable' thats our firewall, it says that for everyone. You can also download a trace program if you like from http://www.neotrace.com/ or http://www.visualroute.com/

If you do see any of the above then you are most likely having a problem with a certain router your ISP is connecting you through. Give it some time it might clear up on its own, if not you can try connecting with another access number with your ISP to try getting through another set of routers, or if you are on a cable modem try contacting your ISP and running a trace to chat.everquest.com and see if they could put you on a connection with minimal Packet Loss.
