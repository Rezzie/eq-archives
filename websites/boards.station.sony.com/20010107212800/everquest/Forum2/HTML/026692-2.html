>cookies are generally bound to your MAC address

Incorrect. Cookies are used for saving information regarding webpages you have visited. It has no bearing whatsoever on your MAC address (Hardware address) of your NIC. Cookies are also only sent back to the site that created them.

This is done via basic TCP/IP tracing which includes traceroute and nslookup. Not via any buzzwords like MAC Address or Cookies.

While it *IS* very possible to trace a packet back to a specific computer it is not done in any way that has been listed thus far.

Each packet that is sent across the internet has 2 IP addresses associated with it. The originating IP address and the destination IP address. The destination knows where that IP address came from and how to send it back. having that information is all you need to find where a certain packet came from.
