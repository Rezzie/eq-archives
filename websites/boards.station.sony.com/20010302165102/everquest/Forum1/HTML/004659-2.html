Aptiva - ESS audio driver for Windows 95/98 Release Date 2000-03-31

To receive automatic notification of driver updates, click 'Profile' above.
Applicable Countries
Worldwide
Description
This package installs an update to the Microsoft Windows 95/98 device driver for the ESS Solo-1 audio adapter that comes preinstalled in your IBM Aptiva computer. This update requires approximately 2 MB of space on your hard disk (drive C).

This update package is intended for IBM Aptiva machine types 2158-xx, 2163-xx, 2164-xx, 2170-xx, 2171-xx, 2172-xx, 2174-xx, 2165-xx, 2190-xx, that come with a preinstalled ESS Solo-1 PCI audio adapter.
Supported Machines
2158, 2163, 2164, 2170, 2171, 2165, 2172, 2174, 2190
File Details
Version 4.06.00.1126
Release Date 2000-03-31

File link File size File description
an1z00us.exe 1,881,722 ESS Solo-1 audio driver for Microsoft Windows 98.


an1z00us.txt 2,462 Installation instructions for ESS Solo-1 audio driver for Microsoft Windows 98. (English)


ao1z00us.exe 4,528,009 ESS Solo-1 wavetable device driver for Microsoft Windows 95/98.


ao1z00us.txt 2,227 Installation instructions for ESS Solo-1 wavetable device driver for Microsoft Windows 95/98.



Additional Information
Affected configurations
All Aptiva systems with ESS Solo-1 audio.

Description
These files will update the ESS audio driver on the system to version 4.06.00.1126. They will also update the software wavetable to maintain compatibility with this new driver level.

Note: Please print these instructions before you begin. You must download and install both of these updates.

Installing the audio drivers

Click the file link to download the file from the Web page.
Select Start -> Find -> Files and folders.
Type an1z00us.exe in the search box.
Click Find Now. This will locate the an1z00us.exe file you just downloaded.
Double-click the icon for an1z00us.exe. The default folder setting should remain C:\DRIVES\AN1Z00US.
Follow the on-screen prompts to install the device driver.
Shut down and restart your computer so that the changes take effect.

Installing the wavetable update

Click the file link to download the file from the Web page.
Select Start -> Find -> Files and folders.
Type ao1z00us.exe in the search box.
Click Find Now. This will locate the ao1z00us.exe file you just downloaded.
Double-click the icon for ao1z00us.exe. The default folder setting should remain C:\DRIVES\AO1Z00US.
Follow the on-screen prompts to install the wave table files.
Shut down and restart your computer so that the changes take effect.

Testing for successful installation

Click Start -> Settings -> Control Panel.
Double-click System.
Select the Device Manager tab.
Double-click on the Sound, video, and game controllers device.
Double-click on the ESS Solo-1 PCI Audio Drive device.
Select the Driver tab.
Click on the Driver File Details button.
Highlight the C:\WINDOWS\SYSTEM\ESSOLO.DRV.PCI line. The version should read 4.06.00.1126.
