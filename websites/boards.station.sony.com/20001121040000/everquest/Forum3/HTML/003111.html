Station Member 		posted 10-24-2000 06:09 AM           
In many programming languages you decide how much space you want to reserve for each value. The stats in EQ seem to be stored in an 8-bit variable.

A bit can be either 0 or 1. With 8 bits you can only have 256 different combinations/values (values 0 to 255).

If the EQ-stats were stored in a larger variable more memory and more bandwidth would be needed.
