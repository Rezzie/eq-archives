You can either turn it on in the .ini file or turn it on every time you log on... it dumps a log file in the Everquest directory which contains all the text messages you see in the game. The log is permanent until you delete it, and can be read in any sane text editor (e.g. NOT Wordpad, which just chokes on an 8MB log file).

It creates a textfile that includes the name of your character, so that you will have separate files for each character. There are many downloadable parsing programs out there that are designed to analyze various aspects of the game, such as melee combat.

Note that each line includes a timestamp, down to the second (but not the tenth of a second). This is helpful for damage analyses.

The command to activate it from within the game is /Log . If I remeber corectly, the syntax is /Log on to activate, and /Log off to deactivate.
