I'm sure many of us have a favorite character name.. a name that we pick for a character and, when we decide the class we picked for that character isn't quite what we wanted, have rerolled the character, but kept the same name.

This is all well and good, but after rerolling three or four times.. perhaps the class wasn't right, perhaps you picked the wrong head or messed up your stats.. Suddenly the next time you go through the character creation (often right after deciding to delete/reroll said character), the name is being rejected!

Nothing against the naming policy, nobody could have picked the name for themselves (unless through some INCREDIBLE coincidence), no reason at all for the name to be rejected, but there it is.. your favorite name gone.

I've heard all kinds of explanations for this, but really, it doesn't matter, the name's been put onto some kind of 'taken' list...

My question for anybody that would know (Absor, or any of the other names I see tossed about) is this: Do you guys have any kind of name purge planned for any time in the future? I'd love to have my favorite name back (as I'm sure several others out there would as well)
