Hello,

I know many users have encountered the issue of dropping to the desktop. This message is to those who exeperience this problem. If you actually get onto the server select screen and choose your server, but it drops to the desktop everytime then you'll need to do the following. Check the file labeled DBG.TXT in your Everquest folder, look for any type of error. The one you will mainly be looking for is "There was an error allocating memory." If you get that message then you'll need to check for a few things.
First off run Testeverquest.exe and patch those files, then try rerunning Everquest.exe. If that does not work try the following, make sure you have at least 500megs free on your C:\ and let windows manage your virtual memory.

Tech-Nimble
